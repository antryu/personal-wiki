# ArcKit Governance Toolkit

> ArcKit is an enterprise architecture governance and vendor procurement toolkit designed to transform architecture governance into a structured, AI-assisted workflow. It enables architects to enforce principles, manage risks, create requirements documents, and streamline vendor selection processes.  
> Last updated: 2026-04-18

## Overview
ArcKit is a comprehensive toolkit for enterprise architects, offering a systematic approach to architecture governance. It includes tools and workflows for establishing and enforcing architecture principles, analyzing stakeholder goals, managing risks, and creating detailed requirements. The toolkit also supports data modeling, technology research, strategic planning, and vendor management. ArcKit is designed to work with AI assistants like Claude Code, Gemini CLI, and GitHub Copilot, providing an integrated and automated experience.

### Key Features
- **Architecture Governance**: Establish and enforce architecture principles with structured workflows.
- **Stakeholder Analysis**: Understand and align stakeholder drivers and outcomes.
- **Risk Management**: Follow HM Treasury Orange Book guidelines for risk assessment.
- **Business Case Justification**: Apply HM Treasury Green Book SOBC for business case creation.
- **Requirements Documents**: Generate comprehensive and traceable requirement documentation.
- **Data Modeling**: Create ERD models, ensure GDPR compliance, and maintain data governance.
- **Technology Research**: Conduct build vs buy analysis using web search and Microsoft Learn MCP.
- **Vendor Management**: Streamline RFP and selection processes with integrated workflows.
- **Design Review**: Conduct high-level and low-level design reviews with formal frameworks.
- **ServiceNow Integration**: Support for ServiceNow service management design.
- **Traceability**: Maintain traceability of requirements and citations with inline markers.

### Installation and Usage
ArcKit is available through multiple platforms, including **Claude Code**, **Gemini CLI**, and **GitHub Copilot**. Each platform offers a tailored experience, with Claude Code providing the most complete integration, including all commands, agents, and automation hooks.

#### Installation Options
- **Claude Code**: Install the ArcKit plugin from the marketplace for a full-featured experience.
- **Gemini CLI**: Install the ArcKit extension with zero configuration.
- **GitHub Copilot**: Use the ArcKit CLI to scaffold prompt files for integration with VS Code.

## Key Points
- ArcKit transforms architecture governance into a structured, AI-assisted workflow.
- It supports a wide range of enterprise architecture activities, including risk management, requirements documentation, and vendor selection.
- The toolkit is designed for integration with AI assistants like Claude Code and Gemini CLI.
