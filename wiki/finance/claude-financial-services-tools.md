# Claude Financial Services Tools

> This repository provides reference agents, skills, and data connectors tailored for common financial services workflows such as investment banking, equity research, private equity, and wealth management. It offers a modular system that integrates with Claude Cowork and the Claude Managed Agents API, enabling flexible deployment options.
> Last updated: 2026-05-06

## Overview

The **Claude Financial Services** repository is a comprehensive toolkit designed to support a wide range of financial workflows. It includes named agents for full end-to-end workflows, along with vertical plugins for specific skills and data connectors. The repository supports two deployment methods: as a [Claude Cowork](https://claude.com/product/cowork) plugin or through the [Claude Managed Agents API](https://docs.claude.com/en/api/managed-agents). This ensures consistency across deployment options, with the same system prompt and skills available regardless of the approach.

The repository is organized around a set of named agents and vertical plugins, with each agent bundled as a self-contained plugin. These agents are designed for tasks like pitch deck creation, client meeting preparation, industry research, model building, and operations management. Vertical plugins offer modular access to core skills and data connectors within specific financial services verticals.

## Key Points

- Modular system for financial workflows including investment banking, equity research, and more
- Two deployment options: Claude Cowork plugin or Claude Managed Agent API
- Named agents for end-to-end tasks with self-contained skills and connectors
- Vertical plugins for modular access to skills by financial vertical
- Full compliance emphasis: outputs are for review and sign-off by qualified professionals
