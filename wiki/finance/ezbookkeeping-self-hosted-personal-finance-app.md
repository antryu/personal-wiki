# ezBookkeeping - Self-Hosted Personal Finance App

> A lightweight, self-hosted personal finance app with a user-friendly interface and powerful bookkeeping features. It helps users track expenses, analyze spending patterns, and manage finances with full control over data and deployment.  
> Last updated: 2026-05-10

## Overview

ezBookkeeping is a lightweight, self-hosted personal finance application that provides users with a user-friendly interface and powerful bookkeeping features. It allows users to record daily transactions, analyze historical data using built-in charts, and run custom queries for a better understanding of their financial habits. Designed to be resource-efficient and easy to deploy, it runs smoothly on devices like Raspberry Pi, NAS, and MicroServers.

With support for both mobile and desktop devices, and PWA capabilities, ezBookkeeping offers a seamless user experience across platforms. It also supports multiple languages, currencies, and time zones, making it a versatile tool for personal finance management. The application is open-source and available under the MIT license, and it supports a variety of databases and deployment methods including Docker.

## Key Points

- Self-hosted and open-source personal finance management tool
- Lightweight and runs efficiently on low-resource hardware
- Fully supports mobile and desktop use with PWA support
- AI-powered features like receipt image recognition
- Multi-language and multi-currency support
- Secure with 2FA, OIDC, and login rate limiting
- Extensive data import/export support (CSV, OFX, QFX, QIF, IIF, Camt, MT940, GnuCash, Firefly III, Beancount, and more)
