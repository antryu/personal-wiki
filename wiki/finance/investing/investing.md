# 투자 전략 & 자동매매 봇 프로젝트

> ARK Invest Big Ideas 2026 리포트 분석을 시작으로, 크립토 선물 자동매매 봇 개발(Ultra Trader)과 은(Silver) 시장 재진입 전략까지 확장된 투자 리서치 및 자동화 프로젝트 기록.
> Last updated: 2026-04-05

## Overview

2026년 3월 말, ARK Invest의 Big Ideas 2026 리포트(111페이지, 13개 빅 아이디어)를 분석하면서 투자 테마별 종목 선정과 포트폴리오 배분을 진행했다. 이 과정에서 기존 MES(S&P500 Micro E-mini) 자동매매 경험을 바탕으로 크립토 선물 자동매매 봇 개발로 확장되었고, 동시에 은(Silver) 시장 폭락 후 재진입 전략도 수립했다.

## Key Topics

### 테마별 투자 종목 선정 (2026-03-25)

자동매매 제외, 중장기 포지션 관점으로 종목 선정.

**코어 (60%) — 확신도 높음:**
- **AVGO** (20%) — AI 인프라 ASIC 설계, GPU 점유율 잠식 중. 현재가 $322, 고점 대비 -22%
- **BTC/IBIT** (20%) — 기관 자산 클래스 성숙. 현재 $70,600, ATH 대비 -44%. ARK Base $710K
- **COIN** (20%) — Base L2 + 기관 커스터디 + DeFi 인프라. 현재 $181, 고점 대비 -59%

**위성 (30%) — 중간 확신:**
- **TSLA** (12%) — 로보택시 + Optimus. $383, 변동성 감수 필요
- **SHOP** (10%) — AI 커머스 인프라. $116
- **ILMN** (8%) — DNA 시퀀싱 비용 $10 시대 수혜. $124

**스윙 (10%) — 고위험 고보상:**
- **NTLA** (10%) — in-vivo 유전자 편집. $13, 바닥권

**매수 타이밍 판단 (3/25 기준):**
- 지금 괜찮음: BTC $70K (-44%), COIN $181 (-59%), NTLA $13 (52주 저점 근접)
- 기다려도 됨: AVGO (실적 시즌 변동성 활용), TSLA ($350 아래 선호)

### 트레이딩 봇 바이럴 트윗 분석 (2026-03-25)

@zostaff 트윗 분석 — "Claude로 15분 만에 Polymarket 봇 3개, 하루 $2,503 수익" (조회수 177만).

**빨간 깃발:**
- $1 → $2,503/일은 수학적으로 불가능 (승률 60%짜리로)
- 프로필에 텔레그램 그룹 링크 — 전형적 "guru selling shovels" 패턴
- 계정 2024.10 생성, 하루 ~20개 트윗 = 콘텐츠 팜
- MACD/RSI/VWAP은 1970~80년대 지표, 기본 TA로 2026년에 알파 불가능

**GitHub 레포(sinmb79) 분석:** 원저자 아닌 다른 사람이 Binance용으로 구현. 백테스트 결과 — MACD 승률 51.4%, $1,000 → 1년 수익 $34.21 (3.4%). 교육용으로는 괜찮으나 실전 수익성 없음.

**참고할 점:** Rich 터미널 대시보드 UI, 멀티 전략 병렬 실행 구조 (전략별 독립 루프 + 공유 리스크 매니저).

### 크립토 선물 자동매매 봇 개발 (2026-03-25 ~ 03-27)

기존 MES 자동매매 경험을 크립토 선물로 확장. 3개 프로젝트를 순차적으로 개발:

**Phase 1 — 커스텀 봇 (`crypto-futures-bot/`)**
- 5개 전략 (BB, MACD, RSI+VWAP, CVD, 펀딩비)
- Binance + Bybit 동시 지원 (ccxt)
- 백테스트 + 그리드 서치 최적화
- 리스크 매니저 + 터미널 대시보드
- 위치: `~/Projects/crypto-futures-bot/`

**Phase 1 — Freqtrade 봇 (`freqtrade-bot/`)**
- 동일 5개 전략 + FreqAI (LightGBM ML 모델)
- Hyperopt 파라미터 최적화
- Docker 배포, 텔레그램 봇 내장
- 위치: `~/Projects/freqtrade-bot/`

**Phase 2 — Ultra Trader (`ultra-trader/`)**
- 오픈소스(Freqtrade, Jesse, Octobot, Hummingbot) + 상업용(3Commas, Cryptohopper, Stoic AI) 장점 조합
- 25,559줄 / 73개 파일
- 12개 전략: BB, MACD, 추세추종, RSI+VWAP, CVD, 그리드, DCA, 마켓메이킹, 펀딩비 차익, 베이시스, 앙상블, 레짐 스위칭
- AI 레이어: LightGBM ML + Ollama LLM 시장 분석 + 센티멘트
- 웹 대시보드 (FastAPI + Chart.js + WebSocket)
- 백테스트: 몬테카를로 + Walk-forward + Bayesian 최적화
- 위치: `~/Projects/ultra-trader/`

**킬러 차별점:** 로컬 AI (클라우드 의존 없음), 오픈소스 투명성, 멀티 거래소 (CEX + DEX), 웹 UI, ML 자동 재학습, $0/월.

**다음 단계:** 테스트넷 API 키 발급 (Binance testnet.binancefuture.com / Bybit testnet.bybit.com) → 백테스트 → 테스트넷 페이퍼 트레이딩 → 승자 실전 투입.

### 은(Silver) 시장 분석 & 재진입 전략 (2026-03-27)

**시장 현황 (3/27 기준):**
- 현재가: ~$72/oz (ATH $121에서 -50% 폭락)
- 금/은 비율: ~65:1 (역사적 평균 ~70)
- 6년 연속 공급 부족 (2021~2026), 2026 부족량 6,700만 oz
