# Open Source Survey Tool

> Formbricks is an open-source alternative to Qualtrics for creating surveys, feedback tools, and experience management systems. With over 12,000 stars on GitHub and a robust feature set, it enables teams to collect user insights via in-app, website, link, and email surveys while prioritizing privacy and customization.

### Mission
Formbricks aims to empower teams to create "irresistible experiences" by gathering user feedback at every stage of the customer journey. It combines a no-code survey builder with privacy-first data analysis tools, offering both cloud-hosted and self-hosted deployment options.

### Features
- **No-code survey editor** with drag-and-drop interface and multiple question types
- **Prebuilt templates** for common use cases (NPS, feedback, onboarding)
- **Targeted distribution** to specific user segments without code changes
- **Collaboration tools** for team workflows
- **Link-based surveys** for external sharing
- **Self-hosted** or **cloud-hosted** deployment flexibility
- **Privacy-first design** with on-premise data control

### Deployment Options
- **Cloud Version**: [formbricks.com](https://app.formbricks.com/auth/signup) (managed service)
- **Self-hosted**: Requires local setup with full codebase access

## Key Points
- Open-source alternative to commercial tools like Qualtrics and Typeform
- Combines survey creation with advanced experience management capabilities
- High community engagement (12k+ stars, active GitHub Accelerator participant)
- Dual deployment model (SaaS + self-hosted) for flexibility
- Focus on privacy and data sovereignty with AGPL licensing
