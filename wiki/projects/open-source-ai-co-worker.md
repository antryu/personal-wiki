# Open-Source AI Co-Worker

> An open-source AI coworker that builds a local knowledge graph from emails and meetings to automate workflows and generate actionable insights. Notable for its 11,330+ stars and local-first architecture with Markdown-based data storage.  
> Last updated: 2026-04-10

## Overview
- **Repository**: [rowboatlabs/rowboat](https://github.com/rowboatlabs/rowboat)  
- **Description**: A local-first AI coworker that transforms email, meeting notes, and documents into a searchable knowledge graph for task automation.  
- **Features**:  
  - Generates PDF roadmaps, meeting briefs, and voice notes using contextual knowledge  
  - Tracks people, companies, and topics via live notes  
  - Maintains an Obsidian-compatible Markdown vault for editable knowledge graphs  
  - Supports voice input/output and web search integration  
- **Technical**:  
  - **Stars**: 11,330 | **Forks**: 1,067  
  - **Language**: TypeScript | **License**: Apache 2.0  
  - **Topics**: `ai-agents`, `generative-ai`, `multiagent`, `productivity`  
  - **Open Issues**: 77  

## Key Points
- **Knowledge Graph**: Automatically builds and updates a semantic network of work-related context from connected data sources.  
- **Task Automation**: Drafts emails, prepares meeting summaries, and generates artifacts like PDFs using contextual memory.  
- **Local-First Security**: Processes data privately on the user's machine with optional cloud integrations for Google services.  
- **Extensibility**: Supports voice APIs (Deepgram, ElevenLabs), web search (Exa), and external tools via API keys.
