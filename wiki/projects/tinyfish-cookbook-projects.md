## Overview

**TinyFish Cookbook** is a repository containing 8+ real-world automation projects built with TinyFish's web agent API. The projects demonstrate applications for anime streaming, sports betting, code analysis, competitor pricing, and more. The repository highlights TinyFish's SOTA performance (90% on Mind2Web benchmark, outperforming Gemini by 21 points) and enterprise-grade infrastructure used by companies like Google and DoorDash.
