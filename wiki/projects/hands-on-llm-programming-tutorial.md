# Hands-on LLM Programming Tutorial

> 《动手学大模型Dive into LLMs》系列编程实践教程，由上海交通大学课程讲义拓展而来，提供大模型相关的入门编程参考。教程完全免费，旨在帮助学生快速入门大模型，更好地开展课程设计或学术研究。
> Last updated: 2026-04-15

## Overview

《动手学大模型》系列编程实践教程由上海交通大学《自然语言处理前沿技术》（NIS8021）和《人工智能安全技术》（NIS3353）课程讲义拓展而来，由张倬胜老师指导。教程提供大模型相关的编程实践内容，涵盖微调与部署、提示学习与思维链、知识编辑、数学推理、模型水印、越狱攻击、大模型隐写等多个主题。教程以 Jupyter Notebook 为主要实现方式，适合初学者快速入门大模型开发。

教程内容包括：

- **微调与部署**：指导如何对预训练模型进行微调，并将其部署为可用的 Demo。
- **提示学习与思维链**：介绍大模型的 API 调用与推理方法。
- **知识编辑**：探讨语言模型中特定知识的编辑方法和验证手段。
- **数学推理**：讲解如何让大模型进行数学推理，快速蒸馏一个迷你模型。
- **模型水印**：在语言模型生成的文本中嵌入不可见的水印。
- **越狱攻击**：学习如何通过攻击测试大模型的安全性。
- **大模型隐写**：研究如何在大模型的输出中嵌入秘密信息。

## Key Points

- 教程内容覆盖大模型的多个核心领域，包括微调、提示工程、知识编辑、数学推理、模型水印、安全攻击等。
- 教程完全免费，适合初学者和研究人员快速上手大模型开发。
- 教程由上海交通大学课程拓展而来，具有较强的学术基础和教学价值。
- 教程支持中文社区，拥有广泛的社区支持和活跃的开发更新。
