# VidBee Downloader

> A modern open-source video downloader supporting 1000+ websites worldwide, built with Electron and powered by yt-dlp. Notable for its intuitive UI, RSS auto-download automation, and global platform compatibility.  
> Last updated: 2026-04-19

## Overview

**VidBee** is an open-source video and audio downloader that enables users to extract content from over 1000 platforms including YouTube, TikTok, Instagram, and Twitter. Built with **Electron** and powered by **yt-dlp**, it offers a clean graphical interface with advanced features like background RSS feed subscriptions for automated downloads.

### Features
- **Global Video Support**: Downloads from 1000+ websites using yt-dlp.  
- **Intuitive UI**: Real-time progress tracking, pause/resume/retry controls, and visual download queue management.  
- **RSS Automation**: Subscribes to creator feeds and auto-downloads new content in the background.  
- **Cross-Platform**: Available for Windows, macOS, and Linux via Electron.
