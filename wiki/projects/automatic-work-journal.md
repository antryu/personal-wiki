# Automatic Work Journal

> Dayflow is an open-source, local-first macOS app that automatically generates a private timeline of your daily activities by analyzing your screen with AI. It distinguishes itself through context-aware tracking, privacy-first design, and extensibility.  
> Last updated: 2026-04-16

## Overview

**Dayflow** is a native macOS application (built with SwiftUI) that creates a detailed, AI-powered timeline of user activities by analyzing screen content. It operates locally, ensuring data privacy, and supports multiple AI providers (e.g., Gemini, Ollama, ChatGPT). The app is lightweight (25MB) and efficient, using ~100MB RAM and <1% CPU.  

### Key Features
- **Context-aware tracking**: Differentiates between tasks (e.g., "researching on YouTube" vs. "watching cat videos").  
- **Privacy-first**: Data remains on the user's Mac; AI provider selection is user-controlled.  
- **Open-source**: Fully transparent codebase with MIT License.  
- **Automation**: Generates daily journals, highlights distractions, and exports timelines as Markdown.  
- **Battery efficiency**: Designed for minimal resource usage.
