# Spooty - Self-hosted Spotify Downloader

> Spooty is a self-hosted Spotify downloader that allows users to download tracks, playlists, or albums via a Spotify URL. It retrieves music metadata from Spotify and then finds and downloads the relevant tracks from YouTube. This makes it a versatile and privacy-friendly alternative for those who want to manage their Spotify content without relying on third-party services.
> Last updated: 2026-05-16

## Overview
Spooty is an open-source, self-hosted application built with NestJS and Angular. It is designed to download music from Spotify, but instead of downloading directly from Spotify, it leverages YouTube for the actual file downloads. This approach can be useful for users who want access to Spotify content but prefer a more flexible and self-controlled setup.

Key features include:
- Subscription to new releases from a playlist or artist.
- Support for downloading individual tracks, playlists, or albums.
- Easy installation via Docker or from source.
- Use of environment variables for configuration.
- Requires a Spotify application with proper credentials for API access.

## Key Points
- Spooty is a self-hosted solution, giving users full control over their data and downloads.
- It retrieves metadata from Spotify and downloads actual content from YouTube.
