# Paperless-ngx DMS

> A community-supported document management system designed to scan, index, and archive documents. Paperless-ngx is the successor to the original Paperless and Paperless-ng projects, offering a powerful, open-source solution for managing physical and digital documents in a searchable online archive.
> Last updated: 2026-04-19

## Overview

**Paperless-ngx** is a document management system (DMS) that allows users to digitize, index, and organize physical documents, making them easily searchable and accessible online. It is built as a successor to the original [Paperless](https://github.com/the-paperless-project/paperless) and [Paperless-ng](https://github.com/jonaswinkler/paperless-ng) projects, with enhanced features and community support.

The project is open-source and licensed under the **GNU General Public License v3.0**. It is written in **Python** and uses **Django** as the backend framework, with **Angular** for the frontend. The system supports Optical Character Recognition (OCR) through **OCRmyPDF**, and integrates with tools like **Tesseract**, **FFmpeg**, and **Poppler** for document processing.

The project is actively maintained, with a large and growing community. It is widely used for personal document management, small to medium-sized office needs, and is even supported by a demo instance hosted by [DigitalOcean](https://m.do.co/c/8d70b916d462).

### Key Features

- **OCR Support**: Uses OCRmyPDF and Tesseract for text extraction from scanned documents.
- **Searchable Archive**: Full-text search across all documents.
- **Tagging & Categorization**: Users can tag and categorize documents for easier navigation.
- **Cloud & On-Premises**: Can be deployed on-premises or in the cloud using Docker.
- **Multi-Language Support**: Available in multiple languages via Crowdin.
- **Community-Driven**: Maintained by a team of contributors and open to new community members.

### Getting Started

The easiest way to deploy Paperless-ngx is through **Docker Compose**. The project provides Docker Compose files in the `docker/compose` directory for quick setup.

A demo instance is available at [https://demo.paperless-ngx.com](https://demo.paperless-ngx.com) for users to try before deployment. The login credentials for the demo are `demo` / `demo`.

For full documentation, visit [https://docs.paperless-ngx.com](https://docs.paperless-ngx.com).
