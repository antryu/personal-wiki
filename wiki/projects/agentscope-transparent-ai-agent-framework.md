# AgentScope: Transparent AI Agent Framework

> AgentScope is a production-ready, easy-to-use AI agent framework designed for building and running transparent, understandable, and trustworthy agents. It provides essential abstractions that adapt to rising model capabilities and includes built-in support for fine-tuning.  
> Last updated: 2026-04-11

## Overview

AgentScope is a framework for building and managing AI agents that emphasize transparency and trustworthiness. It supports increasingly agentic large language models (LLMs) by leveraging their reasoning and tool-use capabilities without imposing rigid prompts or orchestrations. The framework is built for developers who want to create, manage, and deploy AI agents in diverse environments, including local setups, cloud deployments, and Kubernetes clusters.

The framework is backed by a growing community and offers a rich ecosystem of tools, memory modules, and observability features. It supports both single-agent and multi-agent systems through built-in Message Control Protocol (MCP) and Agent-to-Agent (A2A) communication. AgentScope also includes tools for human-in-the-loop steering, planning, evaluation, and model fine-tuning.

AgentScope is open-source and released under the Apache License 2.0. It is actively maintained and has a strong roadmap for future development, including upcoming features like AgentScope 2.0, real-time voice agents, and memory compression.

## Key Points

- **Transparent AI**: Designed to build agents that are visible, understandable, and trustworthy.
- **Flexible and Extensible**: Offers a wide range of integrations and supports custom tool development.
- **Production-ready**: Suitable for deployment in local, cloud, or Kubernetes environments.
- **Active Development**: Regular updates, community meetings, and a clear roadmap for future features.
