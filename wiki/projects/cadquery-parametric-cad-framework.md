# CadQuery Parametric CAD Framework

> CadQuery is a Python-based parametric CAD scripting framework that allows users to create complex 3D models using the powerful Open CASCADE Technology (OCCT) kernel. It offers a more intuitive and expressive way to generate CAD models compared to similar tools like OpenSCAD, making it a notable choice for engineers and developers.  
> Last updated: 2026-04-18

## Overview

CadQuery is a Python module designed for building parametric 3D CAD models using a script-based approach. It provides a high-level API that enables developers and designers to generate complex geometries using a clean, readable syntax. Unlike many traditional CAD tools, CadQuery does not require a graphical user interface, making it ideal for integration into automated workflows, scientific computing, and engineering simulations.

The project is built on top of the Open CASCADE Technology (OCCT) kernel, which allows for advanced modeling capabilities such as NURBS surfaces, fillets, and complex Boolean operations. CadQuery is often compared to OpenSCAD, but it benefits from Python's rich ecosystem of libraries and tools, as well as the robustness of OCCT.

CadQuery supports a wide range of output formats, including STEP, DXF, STL, AMF, and 3MF, ensuring compatibility with both industrial and open-source CAD tools. It also offers tools for visualization, such as the CQ-Editor and integration with JupyterLab.

The project is actively maintained and has a growing community, with extensive documentation and a variety of resources for new and experienced users.

## Key Points

- CadQuery uses Python as its scripting language, allowing users to leverage Python's extensive libraries and development tools.
- It is built on the powerful Open CASCADE Technology (OCCT) kernel, which supports advanced modeling features like NURBS, surface sewing, and complex Boolean operations.
- CadQuery supports high-quality, lossless formats such as STEP and DXF, in addition to common 3D formats like STL and AMF.
- The framework is designed for use in non-visual environments, making it ideal for server-side or automated modeling tasks.
- It provides an intuitive API that allows users to build parametric models with minimal code.
