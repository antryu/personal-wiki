# Multi-Platform AI Assistant Agent

> CowAgent is a lightweight, multi-platform AI assistant framework that supports autonomous task planning, multi-model integration (OpenAI, Claude, Gemini, Qwen, etc.), and seamless integration with WeChat, Feishu, DingTalk, and other platforms. It enables users to build personalized AI assistants or enterprise digital employees with skills, memory systems, and multimodal capabilities.

### Features
- **Autonomous Task Planning**: Understands complex tasks, executes step-by-step plans, and leverages tools to achieve goals.
- **Long-Term Memory System**: Stores persistent memories (core, daily, and distilled) with keyword/vector search capabilities.
- **Personal Knowledge Base**: Structures knowledge into graphs, supports dialogue-driven management and visual browsing.
- **Skill Ecosystem**: Installs and runs skills via [Skill Hub](https://skills.cowagent.ai/) or GitHub, enabling custom workflows.
- **Tool Integration**: Built-in tools for file operations, terminal execution, browser automation, and scheduled tasks.
- **CLI & Web Console**: Manages processes, installs skills, and adjusts configurations via terminal or web interface.
- **Multimodal Support**: Processes text, images, audio, and files for input/output.
- **Multi-Model Compatibility**: Supports OpenAI, Claude, Gemini, DeepSeek, Qwen, Kimi, and more.
- **Multi-Channel Access**: Integrates with WeChat, Feishu, DingTalk, QQ, WeCom, and web platforms.
