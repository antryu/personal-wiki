# Gin-Vue-Admin: AI-Powered Full-Stack Development Platform

> Gin-Vue-Admin is a full-stack development platform combining Vite, Vue3, and Gin, offering an AI-assisted development experience. It is designed for enterprise-level applications with features like JWT authentication, dynamic routing, component visibility control, and more.
> Last updated: 2026-05-08

## Overview

Gin-Vue-Admin is a powerful full-stack development platform that integrates modern web development technologies and AI-powered tools. It is built using **Vite**, **Vue3**, and **Gin** with support for **TypeScript** and **JavaScript**. The platform offers AI-assisted code generation, a modular architecture, and a rich set of features for enterprise-level application development. It includes functionalities such as **JWT authentication**, **role-based access control**, **dynamic routing**, **upload/download support**, **pagination**, **code and form generators**, and **configurable import/export tools**.

The project has been actively developed since **2019-09-01**, and the latest update was on **2026-05-08**. It has gained a significant following with **24,660 stars** and **7,073 forks** on GitHub and is licensed under the **Apache License 2.0**.

It is compatible with the **claw ecosystem** and supports **AI-assisted development** with features like **GvaClaw**, allowing developers to generate full-stack code quickly and efficiently.

## Key Points

- Integrates Vite, Vue3, and Gin for front-end and back-end development.
- Provides AI-assisted code generation for rapid development.
- Includes JWT authentication, role-based access control, and dynamic routing.
- Offers full-stack code generation tools (CRUD, forms, and more).
- Compatible with both TypeScript and JavaScript.
- Fully documented with video tutorials for setup and deployment.
- Supports enterprise-level application features such as pagination, upload/download, and i18n.
- Requires a basic understanding of Golang and Vue for usage.
- Licensed under Apache 2.0 with optional paid support and commercial licensing.
