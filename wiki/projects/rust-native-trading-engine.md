# Rust-native Trading Engine

> Production-grade Rust-native trading engine with deterministic event-driven architecture, combining Rust's performance and safety with Python's flexibility for strategy development and system orchestration.  
> Last updated: 2026-04-12

## Overview  
The **nautilus_trader** project provides a high-performance, open-source trading engine for multi-asset, multi-venue systems. It features a deterministic event-driven architecture, enabling research, simulation, and live execution within a unified framework. Rust powers the core engine for speed and safety, while Python serves as the control plane for strategy logic and configuration.
