# Twenty: Open-Source CRM Alternative

> Twenty is an open-source CRM platform designed as a modern, community-driven alternative to proprietary systems like Salesforce. With over 43,930 stars and active development since 2022, it emphasizes affordability, customizable workflows, and integration of modern UX patterns.  
> Last updated: 2026-04-13

## Overview
**Repository**: [twentyhq/twenty](https://github.com/twentyhq/twenty)  
**Description**: A modular CRM system with features for data customization, automation, and multi-view layouts (kanban, table, etc.). Built using TypeScript, React, and NestJS, it supports self-hosting via Docker and offers extensive documentation for developers.  

**Key Highlights**:  
- **Open-source** with community-driven development and plugin ecosystem goals.  
- **Self-hosting capabilities** via Docker Compose or local setup.  
- **Modern UX** inspired by tools like Notion and Airtable.  
- **Customizable data models**, permissions, and automation workflows.  

**Repository Metrics**:  
- **Stars**: 43,930 | **Forks**: 5,934  
- **Language**: TypeScript | **License**: Custom  
- **Topics**: CRM, PostgreSQL, React, GraphQL, open-source, and more  
- **Open Issues**: 79  

**Installation**:  
- [Self-hosting guide](https://docs.twenty.com/developers/self-host/capabilities/docker-compose)  
- [Local development setup](https://docs.twenty.com/developers/contribute/capabilities/local-setup)  

**Core Features**:  
1. Personalized layouts (filters, sorting, kanban/table views).  
2. Customizable objects/fields and role-based permissions.  
3. Workflow automation via triggers/actions.  
4. Integration with emails, calendars, and file management.

## Key Points
- **Cost-effective alternative**: Challenges proprietary CRM pricing models by offering open-source flexibility.  
- **Community-driven innovation**: Over 5,900 forks and active contributor base.  
- **Modern architecture**: Built with TypeScript, React, and NestJS for scalability.
