# Osaurus: Native macOS AI Agent Framework

> Osaurus is a native macOS framework for AI agents, offering persistent memory, autonomous execution, and cryptographic identity. It allows users to run AI models locally or connect to cloud providers while maintaining control over their data and context. Built in Swift, it is fully offline, open-source, and MIT licensed.  
> Last updated: 2026-04-10

## Overview
Osaurus is a native macOS framework designed to serve as a harness for AI agents, enabling users to run and manage AI models directly on their machines. It provides a local, secure, and customizable environment for AI agents to operate, with support for persistent memory, autonomous execution, and cryptographic identity. Osaurus allows the use of any model—local or cloud-based—while keeping the agent's context and data on the user's machine. The framework is built in Swift, optimized for Apple Silicon, and fully offline by default.

Key features include:
- **Agent Creation**: Users can create agents with custom prompts, memory, and themes for specific tasks such as research, coding, or file organization.
- **Model Agnosticism**: Supports models from OpenAI, Anthropic, Ollama, Apple Foundation Models, and local models via MCP Server.
- **Offline and Secure**: All operations can be performed offline, ensuring user data stays private and local unless explicitly shared.
- **Tool Integration**: Tools and skills are automatically selected based on the task at hand through RAG search, eliminating the need for manual configuration.

Installation is straightforward via Homebrew or direct download, and the framework includes a user interface, server mode, and CLI tools. Osaurus is actively maintained, with a vibrant community on Discord and Twitter, and an open plugin registry for extending functionality.

## Key Points
- Built natively in Swift for macOS and Apple Silicon, with no Electron dependencies.
- Fully offline by default, with the option to connect to cloud providers when needed.
- Supports a wide range of models including OpenAI, Anthropic, Ollama, and Apple Foundation Models.
- Agents can be customized with prompts, memory, and themes for specific tasks.
- Tools and skills are automatically selected via RAG search, enhancing agent capabilities without manual configuration.
- Open source and MIT licensed, with a strong community and active development.
