# GitButler

> GitButler is a modern Git-based version control system that enhances the traditional Git workflow with stacked and parallel branches, intuitive commit management, and an undo timeline. It's notable for its AI-powered design and its ability to simplify complex version control tasks while working as a drop-in replacement in any Git repository.
> Last updated: 2026-04-11

## Overview
GitButler is a powerful new Git-based version control system built from the ground up to be simple, powerful, and flexible. It supports both a GUI and CLI interface, making it suitable for developers who prefer either or both. The project is built using Rust, Tauri, and Svelte, offering a modern and efficient user experience.

The software enhances Git workflows by introducing features such as stacked and parallel branches, which allow users to manage multiple lines of development simultaneously. It also simplifies commit management through drag-and-drop functionality and intuitive CLI commands. Additionally, GitButler provides an undo timeline to track and revert changes easily.

GitButler is designed for use in any existing Git repository and aims to be a more user-friendly and powerful alternative to the traditional Git interface. It's particularly well-suited for developers working with AI agents or in complex collaborative environments.

## Key Points
- Offers stacked and parallel branches for more efficient version control.
- Provides an intuitive GUI and CLI for managing Git workflows.
- Includes an undo timeline for tracking and reverting changes.
- Built with modern technologies like Rust, Tauri, and Svelte.
- Works as a drop-in replacement for Git in any existing repository.
