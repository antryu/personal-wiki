# Awesome LLM Applications

> A repository containing 100+ ready-to-run AI Agent and RAG applications that you can clone, customize, and deploy. It offers a comprehensive and practical resource for developers working on LLM projects.
> Last updated: 2026-04-17

## Overview

**Awesome LLM Apps** is a curated collection of AI applications built around modern LLM technologies, including AI Agents, Multi-agent Teams, RAG, Voice AI, and more. The repository provides fully functional templates that are hand-built and tested, making it easy for developers to quickly start and deploy their own LLM-based applications.

The project was created to eliminate the need to rebuild common components like RAG pipelines or agent loops from scratch. Each template is a self-contained application with full source code and is designed to run with minimal setup.

Key features include:

- **Provider-agnostic support** for multiple LLM providers like Claude, Gemini, OpenAI, xAI, Qwen, and Llama.
- **Comprehensive coverage** of the modern AI stack, including Agent Skills, Fine-tuning, and Voice AI.
- **Easy to use** with a quick start process that requires just a few commands.
- **Highly customizable** templates that can be adapted for various use cases and deployed in production.

The repository is actively maintained, with regular updates and a strong community presence reflected in its 106,133 stars and 15,540 forks.

## Key Points

- Provides over 100 ready-to-run AI applications and templates for LLM-based projects.
- Templates are hand-built and tested, not just curated from external sources.
- Designed to be fully functional and easy to deploy with minimal setup.
- Supports a wide range of LLM providers and includes features like RAG, Multi-agent systems, and Voice AI.
