# T3 Code: Coding Agents GUI

> A minimal web GUI for coding agents like Codex and Claude, notable for its early-stage development and growing community adoption.  
> Last updated: 2026-04-17

## Overview
**T3 Code** is a lightweight graphical interface for interacting with AI coding agents, currently supporting **Codex** and **Claude** (with more providers planned). The project emphasizes simplicity and extensibility but is explicitly labeled as very early-stage, with active development and a vibrant open-source ecosystem (9.4k stars, 1.8k forks).

### Features
- **Supported Providers**: Codex (via OpenAI) and Claude (Anthropic)
- **Installation Options**:  
  - Run instantly with `npx t3`  
  - Desktop apps for Windows (winget), macOS (Homebrew), and Arch Linux (AUR)  
- **Authentication Requirements**:  
  - Codex: Install [Codex CLI](https://github.com/openai/codex) and run `codex login`  
  - Claude: Install Claude Code and run `claude auth login`

### Project Status
- **Development Stage**: Early (bugs expected, contributions not yet accepted)  
- **Dependencies**: Built with TypeScript under MIT License  
- **Community**: Active Discord support server ([discord.gg/jn4EGJjrvv](https://discord.gg/jn4EGJjrvv))

### Development Notes
- **Local Setup**: Requires `mise` for tool management and `bun` for dependencies  
- **Contribution Policy**: Contributions paused until further notice; see [CONTRIBUTING.md](./CONTRIBUTING.md) for guidelines  
- **Observability Docs**: [docs/observability.md](./docs/observability.md) for debugging/troubleshooting

## Key Points
