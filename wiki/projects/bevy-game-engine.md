# Bevy Game Engine

> A refreshingly simple data-driven game engine built in Rust. Bevy is notable for its modular design, performance, and strong community, making it a powerful tool for modern game development.
> Last updated: 2026-05-04

## Overview
Bevy is a game engine built in Rust that emphasizes simplicity, performance, and data-driven architecture. It is designed to be flexible for both beginners and advanced users, with a modular system that allows developers to use only what they need. The engine is in active development and uses the Entity-Component-System (ECS) paradigm for organizing app logic.

The engine is still in early development, with breaking changes being introduced regularly through a "train release schedule" — roughly every 3 months. This rapid development means the API can change significantly, but migration guides are available to help users adapt.

The project is maintained under the Apache License 2.0 and is hosted on GitHub. It has a vibrant community, with platforms such as Discord, Reddit, and GitHub Discussions providing active support and collaboration.

## Key Points
- Bevy is a data-driven game engine built in Rust, emphasizing performance and modularity.
- It uses the Entity-Component-System (ECS) architecture for organizing app logic.
- The engine is open-source and free to use under the Apache License 2.0.
- Bevy has a strong community and learning resources, including a Discord server and a quick start guide.
- Regular breaking API changes are introduced, and users are expected to adapt to these changes.
