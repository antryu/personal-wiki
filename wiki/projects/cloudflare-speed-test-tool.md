# Cloudflare Speed Test Tool

> Cloudflare Speed Test Tool 是一个用于测试 Cloudflare CDN 延迟和速度的工具，可以帮助用户找到最优的 IP 地址。该工具不仅支持 Cloudflare CDN，还适用于其他 CDN 或多 IP 解析的网站。由于其简单易用且功能强大，该项目获得了广泛的关注和使用。
> Last updated: 2026-04-13

## Overview
Cloudflare Speed Test Tool 由 GitHub 用户 XIU2 开发，旨在帮助用户从 Cloudflare 的大量 IP 段中快速找到最适合自己的 IP。该项目使用 Go 语言编写，遵循 GNU General Public License v3.0 协议。用户可以通过测试延迟和速度，选择最优的 IP 地址，从而提升访问速度和稳定性。

该项目的 README 提到，Cloudflare 为不同地区的用户分配的 IP 可能存在延迟高、丢包多、速度慢等问题，尤其是对中国内地用户而言。因此，该项目提供了一种自动化方式，从众多 IP 中筛选出最优解。此外，该项目还支持对其他 CDN 或多 IP 解析的网站进行延迟测试，但下载测速地址需要用户自行提供。

在使用方面，用户可以下载编译好的可执行文件，或者通过命令行工具（如 scoop 或 wget）安装。项目提供了详细的使用说明和安装指南，适用于 Windows 和 Linux 系统。用户也可以通过 GitHub Releases 页面获取最新的版本。

## Key Points
- 支持测试 Cloudflare CDN 的延迟和速度，获取最优 IP。
- 使用 Go 语言编写，易于部署和运行。
- 提供多种安装方式，包括编译文件和命令行工具。
- 支持其他 CDN 和多 IP 解析的网站测试。
