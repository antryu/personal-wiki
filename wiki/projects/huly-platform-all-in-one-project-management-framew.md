# Huly Platform - All-in-One Project Management Framework

> Huly is an all-in-one platform that combines project management, CRM, HRM, chat, and more into a single framework, offering a powerful alternative to tools like Jira, Notion, and Slack. It is designed for both enterprise use and self-hosting.
> Last updated: 2026-05-10

## Overview

The **Huly Platform** is a robust framework for building and deploying business applications, such as CRM, project management, and team collaboration tools. It offers a modular architecture that supports rapid application development and deployment, making it suitable for both internal use and as a foundation for third-party products. The platform is actively maintained and has a growing community, with over 25k stars on GitHub.

This repository serves as the core of the Huly framework and includes essential components like Chat, Project Management, CRM, HRM, and Applicant Tracking System (ATS) modules. Developers and companies like Huly and TraceX are building products on top of this platform. It supports self-hosting via Docker, offering easy deployment and scalability.

## Key Points

- All-in-one framework combining project management, CRM, HRM, and team collaboration tools.
- Modular architecture for flexible application development and integration.
- Open-source and self-hostable with Docker.
- Supports production and development versions with clear release strategies.
- High community engagement with over 25,845 stars and 1,898 forks.
