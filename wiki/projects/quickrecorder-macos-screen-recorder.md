# QuickRecorder macOS Screen Recorder

> A lightweight, high-performance screen recorder for macOS based on ScreenCapture Kit, offering features like HEVC with Alpha video format, driver-free audio loopback, and presenter overlay support. Notable for its open-source AGPLv3 license and active development community.  
> Last updated: 2026-04-16

## Overview
QuickRecorder is a versatile macOS screen recording tool designed for developers and content creators. It leverages Apple's ScreenCapture Kit to enable recording of screens, windows, applications, and mobile devices. Key features include driver-free audio loopback, mouse highlighting, screen magnifier, and support for macOS 14's "Presenter Overlay" for real-time camera integration. The tool outputs HEVC with Alpha format (compatible with iMovie and FCPX) and allows independent audio track separation for system sound and microphone input.  

**System Requirements**: macOS 12.3 or later  
**Installation**:  
- Download the latest release [here](https://github.com/lihaoyun6/QuickRecorder/releases/latest)  
- Or install via Homebrew:  
  ```bash
  brew install lihaoyun6/tap/quickrecorder
  ```

## Key Points
- Supports recording screens, windows, applications, and mobile devices  
- Driver-free audio loopback and mouse highlighting/magnifier  
- Full support for macOS 14's Presenter Overlay (real-time camera overlay)  
- Outputs HEVC with Alpha video format (alpha channel for transparency)  
- Independent audio track separation for system sound and microphone input  
- Open-source AGPLv3 license (non-sandboxed, no App Store plans)  
- Inspired by and contributes to projects like [Azayaka](https://github.com/Mnpn/Azayaka) and utilizes libraries like [SwiftLAME](https://github.com/hidden-spectrum/SwiftLAME)
