# Open Source DocuSign Alternative

> Documenso is an open-source, self-hosted document-signing platform that provides a transparent and trustworthy alternative to DocuSign. With over 12,600 stars and a GNU AGPLv3 license, it empowers users to review and control the infrastructure behind digital signatures.  
> Last updated: 2026-04-15

## Overview
- **Repository**: [documenso/documenso](https://github.com/documenso/documenso)  
- **Purpose**: A fully open-source solution for digital document signing, emphasizing self-hosting and trust transparency.  
- **License**: GNU Affero General Public License v3.0 (AGPLv3).  
- **Tech Stack**: Built with TypeScript, Next.js, Prisma, and PostgreSQL.  
- **Community**: 12.6k stars, 2.5k forks, 178 open issues, and active Discord/roadmap engagement.  
- **Features**:  
  - PDF document signing and management.  
  - Compliance with PAdES (PDF Advanced Electronic Signatures) standards.  
  - Integration with NextAuth for authentication.  
  - Self-hosted deployment options via Dev Containers.  

## Key Points
- **Open Trust Infrastructure**: Designed to eliminate reliance on proprietary tools by enabling full code transparency and self-hosting.  
- **Active Development**: Regular updates (last commit 2026-04-15) and community-driven roadmap.  
- **Recognition**: Featured on Product Hunt for innovation in open-source e-signature tools.
