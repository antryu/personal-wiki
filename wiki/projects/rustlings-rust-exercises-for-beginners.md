# Rustlings: Rust Exercises for Beginners

> Rustlings is a project that provides small exercises to help newcomers get familiar with writing and reading Rust code. It is widely used as a learning companion alongside the official Rust book.  
> Last updated: 2026-05-04

## Overview
Rustlings is a collection of beginner-friendly exercises designed to help users practice and reinforce their understanding of the Rust programming language. The exercises are structured to be solved in order, covering core concepts like ownership, borrowing, types, and more. The project is maintained by the Rust community and is recommended for learners who are working through the official Rust book or seeking hands-on practice.

The repository is actively maintained, with a large number of stars and forks, and it supports multiple platforms. Each exercise is self-contained and provides immediate feedback, making it an effective tool for learning Rust in a structured and interactive way.

## Key Points
- Designed for beginners learning Rust, with small and incremental exercises.
- Serves as a practical supplement to the official Rust book.
- Includes interactive exercises with instant feedback.
- Open source and MIT licensed.
- Maintained by the Rust community.
