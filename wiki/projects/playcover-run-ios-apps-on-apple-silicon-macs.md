# PlayCover: Run iOS Apps on Apple Silicon Macs

> PlayCover is a community fork that allows iOS apps and games to run on Apple Silicon Macs with mouse, keyboard, and controller support. It gained popularity for enabling Genshin Impact and other apps on M-series chips, offering a native-like experience with customizable touch controls.

### Project Description  
PlayCover enables iOS apps and games to run on Apple Silicon Macs (M1/M-series) by wrapping applications in an iPad emulator. It supports mouse, keyboard, and controller inputs with customizable touch controls, surpassing alternatives like Sideloadly in functionality. Originally designed for Genshin Impact, it now supports a wide range of apps, though compatibility varies.  

### Key Features
- **Native Performance**: Apps run natively on Apple Silicon with minimal overhead.  
- **Custom Touch Controls**: Map keyboard inputs, WASD, camera movement, and individual keys (similar to Bluestacks).  
- **Community-Driven**: Actively maintained with 11k+ stars and 900+ forks.  
- **Localization**: Multilingual support via Weblate.  

### Getting Started  
1. **Prerequisites**: Requires an Apple Silicon Mac (M1/M2/...).  
2. **Download**:  
   - [Stable releases](https://github.com/PlayCover/PlayCover/releases)

     ```bash
     brew install --cask PlayCover/playcover/playcover-community
     ```  
3. **Documentation**: [PlayBook Guide](https://playcover.github.io/PlayBook)
