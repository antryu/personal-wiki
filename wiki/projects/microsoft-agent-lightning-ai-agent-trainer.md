# Microsoft Agent Lightning: AI Agent Trainer

> Microsoft Agent Lightning is an open-source framework that enables users to train AI agents using reinforcement learning and other optimization techniques with minimal code changes. It supports integration with various agent frameworks and offers scalable training for single or multi-agent systems.

### Core Features
Agent Lightning simplifies training AI agents by:
- Enabling **zero-code-change optimization** (with minor exceptions) for agents.  
- Supporting integration with **any agent framework** (e.g., LangChain, OpenAI Agent SDK, AutoGen, CrewAI, Microsoft Agent Framework) or plain Python code.  
- Allowing **selective optimization** of individual agents in multi-agent systems.  
- Incorporating advanced algorithms like **Reinforcement Learning (RL)**, Automatic Prompt Optimization, and Supervised Fine-tuning.
