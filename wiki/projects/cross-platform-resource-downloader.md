# Cross-Platform Resource Downloader

> A cross-platform resource downloader supporting WeChat Video, Douyin, Kuaishou, Xiaohongshu, m3u8 streams, and more. Built with Go and Wails for Windows/macOS/Linux.  
> Last updated: 2026-04-16

## Overview
The **res-downloader** project (https://github.com/putyy/res-downloader) is a versatile tool for capturing and downloading online resources from major platforms. Key features include:

- **Multi-platform support**: Works on Windows/macOS/Linux
- **Resource types**: Videos, audio, images, m3u8 streams, live streams
- **Supported platforms**: WeChat Video, Douyin, Kuaishou, Xiaohongshu, Kugou Music, QQ Music
- **Proxy-based architecture**: Uses system proxy (127.0.0.1:8899) for traffic interception

Repository statistics:

The tool combines Go for backend logic with Wails for native UI, offering a simplified alternative to Fiddler/Charles for resource capture. Documentation is available at [res.putyy.com](https://res.putyy.com/).

## Key Points
- Proxy-based resource interception with automatic filtering
- Native desktop application with minimal system requirements
- Support for encrypted video decryption (WeChat Video)
- Multiple download methods integration (Motrix, Neat Download Manager)
- Community-driven development with active issue tracking
