# FineTune macOS Audio Controller

> FineTune is a free, open-source macOS menu bar app offering per-app volume control, multi-device audio routing, 10-band EQ, and headphone correction. It serves as a feature-rich alternative to commercial audio utilities like SoundSource.  
> Last updated: 2026-04-13

## Overview
FineTune enables granular audio management on macOS through a menu bar interface. Key capabilities include:
- **Per-app volume control** with adjustable boost levels (2x–4x)
- **Multi-device output routing** and device priority management
- **10-band equalizer** with 20 presets and custom profiles
- **Headphone correction** via AutoEQ and ParametricEQ.txt imports
- **Loudness compensation** using ISO 226:2023 standards

The app supports macOS 15+ and uses Swift/SwiftUI for its interface. It requires "Screen & System Audio Recording" permissions for operation. Installation options include Homebrew (`brew install --cask finetune`) and direct downloads from GitHub.

## Key Points
- Open-source under GPLv3 with 5,618 stars and 180 forks
- Features 88 open issues for community contributions
- Includes automatic device reconnection and routing persistence
- Offers input device monitoring and alert volume controls
- Implements real-time level management for consistent loudness
