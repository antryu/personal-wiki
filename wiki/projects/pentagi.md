# PentAGI

> PentAGI is a fully autonomous AI agents system designed for performing complex penetration testing tasks. It stands out with its secure sandboxed environment, integration of 20+ professional security tools, and advanced AI-driven task planning, making it a notable tool in the field of automated security testing.  
> Last updated: 2026-04-10

## Overview
PentAGI leverages cutting-edge AI technologies to automate penetration testing for information security professionals, researchers, and enthusiasts. The system operates in a sandboxed Docker environment, ensuring secure and isolated execution. Key features include:
- **Autonomous AI agents** that plan and execute penetration steps with optional monitoring.
- **Integrated tools** like nmap, Metasploit, and sqlmap for comprehensive security testing.
- A **knowledge graph** powered by Neo4j for semantic relationship tracking.
- **Web intelligence** via embedded browsers and APIs (e.g., Tavily, Perplexity) for real-time information gathering.
- **REST/GraphQL APIs** with bearer token authentication for automation and integration.

The project includes a modern web UI, persistent PostgreSQL storage with `pgvector`, and detailed reporting capabilities. Community engagement is supported through Discord and Telegram channels, with a dedicated video overview [available here](https://youtu.be/R70x5Ddzs1o).

## Key Points
- Fully autonomous AI-powered agents with intelligent task planning for penetration testing.  
- Integration of 20+ security tools and sandboxed Docker environment for secure operations.
