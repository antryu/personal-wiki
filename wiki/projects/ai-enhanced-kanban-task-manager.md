# AI-Enhanced Kanban Task Manager

> A productivity tool that integrates AI coding agents with Kanban workflows to streamline software development. Combines task planning, agent execution, and code review in a unified interface.  
> Last updated: 2026-04-11

## Overview
**Vibe Kanban** is a task management system designed to maximize efficiency with AI coding agents like Claude Code, Gemini CLI, and Codex. It enables developers to:  
- Plan tasks using Kanban boards with drag-and-drop issue management  
- Execute coding agents in isolated workspaces with terminals and dev servers  
- Review code diffs and provide inline feedback directly in the UI  
- Switch between 10+ AI agents and preview applications with built-in browser tools  
- Generate pull requests with AI-assisted descriptions and merge workflows  

The platform supports self-hosting via Docker and integrates with GitHub for collaborative code reviews. With 24,840 stars and 2,489 forks, it has become a popular tool for teams leveraging AI in software development.

## Key Points
- **AI Agent Integration**: Supports 10+ coding agents including Claude Code, Gemini CLI, and GitHub Copilot  
- **Kanban Workflow**: Task planning, prioritization, and team collaboration via visual boards  
- **Unified Development Environment**: Combines code execution, terminal access, and live previews in workspaces  
- **Code Review Tools**: Inline commenting, diff analysis, and GitHub PR integration  
- **Self-Hosting Option**: Open-source Docker deployment for private instances
