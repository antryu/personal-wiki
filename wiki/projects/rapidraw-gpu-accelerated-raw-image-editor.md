# RapidRAW: GPU-Accelerated RAW Image Editor

> RapidRAW is a modern, non-destructive RAW image editor leveraging GPU acceleration for high-performance editing. Built with Rust, React, and Tauri, it offers a lightweight, cross-platform alternative to Adobe Lightroom®.  
> Last updated: 2026-04-13

## Overview
RapidRAW provides a fast, intuitive workflow for photographers seeking powerful yet simple tools for color grading and masking. Developed as an open-source project under the GNU Affero General Public License v3.0, it prioritizes speed, a clean UI, and GPU-driven rendering for real-time adjustments. The application is under 20MB and supports Windows, macOS, and Linux.  

Key technologies include:
- **Rust** for core performance-critical logic
- **React** for the UI
- **wgpu** for GPU acceleration via WebGPU
- **Tauri** for cross-platform desktop deployment  
The project has gained traction with 6,382 stars and 235 forks since its 2025 launch, though it remains in active development with 201 open issues.

## Key Points
- GPU-accelerated non-destructive editing for RAW images  
- Lightweight cross-platform application (<20MB)  
- Modern tech stack: Rust, React, wgpu, Tauri  
- Open-source under AGPL v3.0
