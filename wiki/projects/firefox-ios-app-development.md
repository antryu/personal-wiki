# Firefox iOS App Development

> The official repository for Firefox and Focus iOS apps developed by Mozilla, featuring a monorepo structure with Swift-based codebases and robust community engagement. Last updated: 2026-04-14

## Overview
The **mozilla-mobile/firefox-ios** repository hosts the source code for Firefox and Focus iOS applications, developed using Swift and maintained under the Mozilla Public License 2.0. Key details include:
- **Repository Metrics**: 12,879 stars, 3,176 forks, and 1,916 open issues as of April 2026.
- **Development Tools**: Requires Xcode 26.3 and Swift 6.2 for building, targeting iOS 15.0+.
- **Monorepo Structure**: Contains both Firefox (`Client.xcodeproj`) and Focus (`Blockzilla.xcodeproj`) projects.
- **Setup Workflow**: Recommends the `fxios` CLI tool for automated installation, supporting forks and dependency management.
- **Community Involvement**: Encourages contributions via pull requests, issue reports, and Element chat (`#fx-ios`).

## Key Points
- **Monolithic Architecture**: Combines Firefox and Focus iOS codebases in a single repository for streamlined development.
- **Swift & Xcode Dependency**: Built with Swift 6.2 and requires Xcode 26.3 for compilation.
- **Open Source Collaboration**: Actively maintained with clear contributing guidelines and community-driven issue tracking.
- **Cross-Platform Focus**: Emphasizes privacy features in Focus app alongside core Firefox functionalities.
