# Evilginx2 Phishing Framework

> A standalone man-in-the-middle attack framework for phishing login credentials and session cookies, enabling bypass of 2-factor authentication. Notable for its ease of use and advanced evasion capabilities.  
> Last updated: 2026-04-18

Evilginx2 is a successor to the 2017 Evilginx tool, rewritten in Go as a standalone application. It integrates HTTP and DNS servers for seamless phishing attacks, bypassing 2FA by capturing both login credentials and session cookies. The framework includes features like phishing detection evasion, bot traffic prevention, and support for wildcard TLS certificates.  

A commercial version, **Evilginx Pro**, offers advanced capabilities such as automated server deployment, JavaScript/HTML obfuscation, and integration with external DNS providers. The project also includes an official training course, *Evilginx Mastery*, for red team engagements.  

## Key Points
- **MITM Functionality**: Phishes credentials and session cookies, bypassing 2FA protections.  
- **Standalone Design**: Written in Go, eliminating dependencies with built-in HTTP/DNS servers.  
- **Phishing Evasion**: Evades detection by Chrome's Enhanced Browser Protection and similar systems.  
- **Pro Features**: Includes botguard, Evilpuppet (Google phishing), and SQLite database support.  
- **Training & Integration**: Offers a dedicated course and Gophish compatibility for phishing campaigns.
