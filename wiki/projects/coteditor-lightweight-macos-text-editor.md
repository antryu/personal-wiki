# CotEditor: Lightweight macOS Text Editor

> CotEditor is a macOS-native plain text editor emphasizing simplicity, precision, and seamless integration with Apple's ecosystem. With over 7,800 stars and 484 forks, it balances accessibility for beginners with advanced features for developers.  
> Last updated: 2026-04-18

## Overview

**CotEditor** is a Swift-based text editor designed for macOS, prioritizing intuitive user experience and adherence to Apple's design principles. Launched in 2013, it supports 17 languages and leverages Cocoa's document-based architecture for robust performance. The project emphasizes macOS-native behavior, minimalism, and accurate handling of diverse text formats.

### Design Philosophy
CotEditor adheres to five core principles:  
1. **macOS-native behavior** (system UI integration, no non-standard features).  
2. **Universal accessibility** (simple for beginners, precise for experts).  
3. **Minimalism** (avoids unnecessary complexity).  
4. **Text format accuracy** (supports legacy and niche encodings).  
5. **Global inclusivity** (17 language localizations, macOS accessibility features).
