# Mullvad VPN Client App

> The Mullvad VPN client app for desktop and mobile, built with Rust and offering advanced privacy features like quantum-resistant tunnels and WireGuard multihop. Notable for its open-source design under the GNU GPL v3.0 license and cross-platform support.  
> Last updated: 2026-04-13

- **Platform Support**: Windows 10/11, macOS (latest 3 versions), Linux (Ubuntu, Fedora, Debian), Android 8+, iOS 17+  
- **Key Components**:  
  - **Desktop**: GUI, CLI, and system daemon (`mullvad-daemon`)  
  - **Mobile**: Android app with shared backend, iOS standalone implementation  
  - **Features**: WireGuard, quantum-resistant tunnels, DAITA protocol, split tunneling, obfuscation, and content blocking  

## Key Points
- Supports WireGuard with advanced configurations (multihop, TCP, Shadowsocks, QUIC, and obfuscation).  
- Cross-platform compatibility with official support for Windows, macOS, Linux, Android, and iOS.  
- Open-source and audited, ensuring transparency and security.  
- Offers cryptographic verification for downloads via Mullvad's Open Source page.  
- Includes optional local network access and custom DNS settings for enhanced privacy.
