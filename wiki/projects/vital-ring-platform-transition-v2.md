## PART 1. 플랫폼 재정의 — Read + Write

### 시대적 맥락 (Huberman)
- COVID 이후 "건강은 개인 책임" 시대 도래
- **웨어러블은 Read만 — Write는 다음 프론티어**
- 5년 내 미국인 50%가 GLP-1 복용 (국내도 시차 있게 따라옴)
- 105~120세 유전적 상한, **건강한 100세가 현실적 목표**

### 새 포지셔닝
```
v1: "생체 데이터를 자산화하는 헬스 데이터 플랫폼" (Read만)
     ↓
v2: "몸을 읽고, 쓰는 개인 건강 OS"
    - Read: HRV·수면·코르티솔·혈당 (2026~2028)
    - Light Write: Vital Cash 행동 변화 / peptide 큐레이션 (2027~)
    - Full Write: 수면 마스크·체온 패치·neurostim (2028~)
```

### "의료 가부장주의 붕괴" 비전
- 환자가 자기 데이터·결정권을 소유
- PIPA 준수하면서 사용자 주권 강화 = 플랫폼 해자
- 의료기관은 자문·검증 파트너 (결정은 사용자)

## PART 3. Vital Cash 2.0 — Personalized Peptide Cocktail

### Earning (v1 유지)
숙면·아침 준비도·HRV·가족 안부·팬덤·약 복용 (동일)

### Burning (대폭 확장)

#### 🆕 ③ Personalized Peptide Cocktail 큐레이션
**"당신의 코르티솔이 2σ 상승했습니다. 이 조합을 추천합니다"**

| 생체 상태 | AI 추천 Peptide Cocktail | 가격대 |
|---------|------------------------|-------|
| HRV 저하 3일 연속 (스트레스 누적) | BPC-157 + L-테아닌 | ₩99K/월 |
| 수면 점수 65 미만 2주 연속 | CJC-1295 + 마그네슘 L-트레오네이트 | ₩149K/월 |
| 코르티솔 일일 상승 (번아웃) | Ipamorelin + 아슈와간다 | ₩129K/월 |
| 회복 탄력성 저하 (과운동) | TB-500 + 크레아틴 | ₩119K/월 |
| 대사율 저하 (GLP-1 보완) | Tesofensine + L-카르니틴 | ₩179K/월 |

**왜 구글·아마존이 못하는가?**
- 구글: "관심사" 기반 광고 → 상관없는 사람에게도 노출
- 바이탈링: **실제 생체 상태** 기반 → 진짜 필요한 사람에게만
- 차병원 원격 처방 연계 → 법적 안전성 + 의료 신뢰도

**수익 구조 재편** (v1 → v2)
- v1: 광고 25% + 커머스 12% = **37%**
- **v2: 광고 25% + 커머스(cocktail) 25%+ = 50%+** (광고-커머스 통합)

## PART 5. 디바이스 로드맵 (Read-to-Write)

### Phase 1 (~2027) — Read 고도화
- 바이탈링 Gen 2 (혈압 추정·혈당 트렌드·**🆕 코르티솔**)
- 시니어 실리콘 소재
- 아티스트 에디션 정례화 (GD, 임영웅)
- **국내 최초 HRV + 코르티솔 통합 웨어러블**

### Phase 2 (2027~2028) — Light Write
- 바이탈 패치 (연속 혈당·체온)
- 바이탈 워치 (낙상 감지·SOS)
- 🆕 **바이탈 마스크** (빛 차단 + 능동 자극으로 수면 유도)
- 🆕 **바이탈 체온 패치** (손/발 체온조절 → 심부 체온 조작)
- 바이탈 밴드 라이트 ₩29,000 초저가

### Phase 3 (2028~2030) — Full Write
- 오픈 API — 삼성·애플워치 연동, API 사용료 수취
- 🆕 **비침습적 Neurostim** (눈·귀·얼굴 얕은 신경)
  - 집중력 유도
  - 수면 진입 유도
  - 각성 상태 제어

## PART 8. 결론 — 3가지 핵심 전환

### v1 결론: "플랫폼 vs 하드웨어"
### v2 결론: **"Read 시대에서 Write 시대로, 가부장주의에서 사용자 주권으로"**

**세 가지를 기억하십시오**

1. **하드웨어 → 플랫폼 → OS**
   - v1: 반지를 파는 회사 → 데이터 구독 플랫폼
   - v2: 데이터 구독 플랫폼 → **몸을 읽고 쓰는 개인 건강 OS**

2. **Read → Write**
   - 오우라·삼성·가민은 Read에 갇혀 있음
   - 바이탈링이 Write로 먼저 움직이면 5년 내 승자

3. **의료 가부장주의 → 사용자 주권**
   - 의사가 결정 → 사용자가 데이터·결정권 소유
   - 의료기관은 자문·검증 파트너로 재정의
   - PIPA 준수 + 사용자 주권 = 글로벌 확장 가능한 해자
