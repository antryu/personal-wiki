# Cross-Platform Download Manager

> A fast, modern download manager supporting HTTP, BitTorrent, Magnet, and ED2K protocols. Notable for its cross-platform compatibility (Windows, macOS, Linux, Android, iOS) and extensibility via APIs and custom extensions.

### Features
- **Multi-Protocol Support**: HTTP/HTTPS, BitTorrent, Magnet, and ED2K
- **Cross-Platform**: Windows (EXE/Portable), macOS (DMG), Linux (DEB/SNAP/AppImage/Flathub), Android, iOS
- **Customization**: Extend functionality via [APIs](https://gopeed.com/docs/dev-api) or [extensions](https://gopeed.com/docs/dev-extension)
- **High-Speed Downloads**: Parallel segment downloading and resume capabilities
- **Open Source**: Actively maintained under the GPL v3.0 license
