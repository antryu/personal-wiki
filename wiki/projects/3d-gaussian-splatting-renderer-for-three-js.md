# 3D Gaussian Splatting Renderer for THREE.js

> An advanced 3D Gaussian Splatting renderer for THREE.js, enabling developers to render and manipulate Gaussian splat data with performance and flexibility. Notable for its integration with standard THREE.js workflows, support for multiple file formats, and real-time GPU-driven editing.  
> Last updated: 2026-04-16

## Overview
This project, named **Spark**, is an open-source 3D Gaussian Splatting renderer built for [THREE.js](https://threejs.org/). It allows developers to seamlessly integrate Gaussian splat objects into existing 3D scenes, enabling real-time rendering, animation, and manipulation. Spark is designed to work across a wide range of devices and supports multiple popular Gaussian splat file formats, making it a versatile tool for developers and researchers working with 3D data.

Spark offers features such as dynamic splat editing, multi-viewpoint rendering, and a shader graph system for GPU-based modifications. The project is actively maintained and features detailed documentation, examples, and a CDN-based setup for quick integration into web projects.

## Key Points
- Integrates with the THREE.js rendering pipeline to support both splat and mesh-based objects
- Supports a wide range of Gaussian splat file formats including `.PLY`, `.SPZ`, `.SPLAT`, `.KSPLAT`, and `.SOG`
- Optimized for performance on low-powered devices and supports real-time GPU-based editing
- Offers dynamic features such as animation, color editing, and displacement of splat data
- Includes a shader graph system for advanced GPU-driven manipulation
- Provides a CDN and NPM installation options for quick setup
