# Agent-Native AI Tutor System

> DeepTutor is an agent-native personalized learning assistant that leverages multi-agent systems and large language models to deliver adaptive, interactive education. With 15,392 stars and 2,040 forks, it introduces innovative features like persistent autonomous tutors (TutorBot) and a CLI interface for cross-platform learning.

### Key Features
- **Agent-Native Architecture**: Two-layer plugin model (Tools + Capabilities) for modular, scalable tutoring.  
- **TutorBot**: Persistent autonomous AI tutors with memory retention and multi-channel support.  
- **CLI Interface**: Cross-platform command-line tool for seamless user interaction.  
- **Guided Learning**: Structured problem-solving with KaTeX support and progress tracking.  
- **Multi-Agent Collaboration**: Co-Writer and MinerU integrations for enhanced educational workflows.
