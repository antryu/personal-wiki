# SuperSplat Editor

> A browser-based open-source tool for inspecting, editing, and publishing 3D Gaussian Splats, notable for its accessibility and active community engagement.  
> Last updated: 2026-04-14

## Overview  
The **SuperSplat Editor** is a free, web-based tool designed for working with 3D Gaussian Splatting (3DGS) data. Built using TypeScript, WebGPU, and web technologies, it eliminates the need for downloads or installations, enabling users to optimize, visualize, and publish splat data directly in their browser.  

### Features  
- **Live Editing**: Modify Gaussian splats in real-time via an intuitive UI.  
- **Cross-Platform**: Runs in modern browsers (Chrome, Safari, etc.) without native dependencies.  
- **Localization Support**: Multi-language interface with community-driven translations.  
- **Active Development**: Maintains frequent updates (last updated: 2026-04-14) and a large contributor base.  

### Development Workflow  
1. **Local Setup**: Requires Node.js 18+. Clone the repo, install dependencies with `npm install`, and launch the dev server via `npm run develop`.  
2. **Hot Reloading**: Automatic rebuilds on code changes; refresh the browser to view updates.  
3. **Localization**: Add new languages by creating `.json` files in `static/locales` and updating `src/ui/localization.ts`.
