# Open-Source AI Coworker

> An open-source AI coworker that transforms work into a knowledge graph and acts on it, enabling tasks like generating meeting briefs, tracking projects, and automating workflows. Notable for its local-first design, Markdown-based knowledge graph, and integration with tools like Google Workspace and Exa search.  
> Last updated: 2026-04-10

## Overview

**Rowboat** is a local-first AI coworker that connects to email, meetings, and documents to build a persistent knowledge graph. It uses this context to automate tasks like drafting emails, preparing meeting briefs, and generating PDF roadmaps. The system operates entirely on the user's machine, prioritizing privacy and flexibility.

### Key Features
- **Knowledge Graph**: Stores and updates context in an Obsidian-compatible Markdown vault with backlinks.
- **Task Automation**: Generates artifacts (emails, briefs, slides) using contextual data.
- **Voice Integration**: Records voice memos and converts them into structured notes.
- **Extensibility**: Supports external tools via API keys (Google, Deepgram, ElevenLabs, Exa, Composio).
