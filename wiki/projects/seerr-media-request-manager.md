# Seerr Media Request Manager

> Open-source media request and discovery manager for Jellyfin, Plex, and Emby. Simplifies media library management through integrations with Sonarr, Radarr, and customizable request systems.

### Features
- Full integration with **Jellyfin**, **Plex**, and **Emby** (authentication, user management, library scans)
- Supports **PostgreSQL** and **SQLite** databases
- Manages **Movies**, **TV Shows**, and **Mixed Libraries**
- Integrates with **Sonarr** and **Radarr** for automated media management
- Customizable request system for individual seasons/movies
- Granular permissions and notification agents
- Mobile-friendly UI and watchlist/blocklist functionality
- Migrates from **Overseerr**/**Jellyseerr** with dedicated guides
