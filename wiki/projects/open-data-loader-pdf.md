### Core Features
- **AI-Ready Data Extraction**:  
  Converts PDFs (digital, scanned, tagged) into Markdown, JSON (with bounding boxes), and HTML. Supports hybrid AI mode for complex layouts (e.g., multi-column scientific papers) and deterministic local mode for consistent results.
- **Accessibility Automation**:  
  First open-source tool to auto-generate Tagged PDFs and PDF/UA (enterprise). Uses layout analysis and auto-tagging to meet global accessibility regulations.
- **Benchmarks**:  
  - **Overall extraction accuracy**: 0.907 (hybrid mode)  
  - **Table extraction accuracy**: 0.928  
  - **Speed**: 0.015s/page in local mode  
  - **OCR**: 80+ languages for scanned documents at 300 DPI+

### SDK & Integration
- **Languages**: Python, Node.js, Java  
- **LangChain Integration**: Pre-built connectors for RAG pipelines  
- **Requirements**: Java 11+  

### Pricing & Availability
- **Open-Source Core**: Free for data extraction, layout analysis, and auto-tagging to Tagged PDF
