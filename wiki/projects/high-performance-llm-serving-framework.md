## Key Points
- Achieves **25x inference performance** on NVIDIA GB300 NVL72.  
- Supports **multimodal models**, including video/image generation via SGLang Diffusion.  
- Native **TPU compatibility** with the SGLang-Jax backend.  
- Active community with **25,855 stars** and **2823 open issues**.  
- Day-0 support for cutting-edge models like **DeepSeek-V3.2**, **MiMo-V2-Flash**, and **Nemotron 3 Nano**.  
- Awards include the **a16z Open Source AI Grant**.
