# Navidrome Music Server

> Navidrome is an open-source, web-based music server and streamer that allows you to access and stream your personal music collection from any device. It's often compared to a personal Spotify and is notable for its low resource usage, support for multiple users, and compatibility with a wide range of audio formats.  
> Last updated: 2026-04-14

## Overview
Navidrome is an open-source alternative to Subsonic, Subsonic-compatible services, and commercial music streaming platforms. It enables users to self-host a music server that can be accessed from any browser or mobile device. Navidrome supports virtually any audio format and excels at managing large music collections, including compilations and multi-disc sets. It also allows multiple users, each with their own playlists, favorites, and play counts.

The project is actively maintained and has a strong community presence, with resources like Docker support, translations, and themes contributed by users. Navidrome is built in Go, making it lightweight and efficient. It is especially popular among Raspberry Pi users and those who prefer self-hosted solutions for media streaming.

## Key Points
- Open-source, self-hosted music server and streamer
- Supports virtually any audio format and large music collections
- Multi-user support with individual user settings
- Lightweight and efficient, suitable for low-resource devices like Raspberry Pi
- Docker and cross-platform support for macOS, Linux, and Windows
- Community-driven with active development and a strong user base
