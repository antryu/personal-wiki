# Free V2ray Configs

> This repository provides free, frequently updated V2ray configuration files for secure and anonymous internet access. Notable for its 15-minute update intervals and multilingual support, it serves users in regions with internet restrictions.  
> Last updated: 2026-04-19

## Overview
- **Repository**: [barry-far/V2ray-Config](https://github.com/barry-far/V2ray-Config)  
- **Description**: A collection of free V2ray configuration files updated every 15 minutes. Offers multilingual support (Persian, Chinese, Russian, Turkish) and multiple subscription links for client compatibility.  
- **Key Metrics**:  
  - Stars: 1,499  
  - Forks: 226  
  - Open Issues: 10  
  - License: MIT  
  - Last Commit: 2026-04-19  

### Usage Instructions
1. Open the `configs.txt` file in the repository's root directory.  
2. Copy a subscription link (e.g., [All_Configs_Sub.txt](https://raw.githubusercontent.com/barry-far/V2ray-config/main/All_Configs_Sub.txt)) or use direct links below.  
3. Import the link into your V2ray client (e.g., V2rayN, V2rayNG).
