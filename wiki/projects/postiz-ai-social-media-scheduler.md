# Postiz: AI Social Media Scheduler

> Postiz is an open-source AI-powered social media scheduling tool offering an alternative to platforms like Buffer and Hypefury. With 28.6k+ stars and support for platforms like Instagram, X, and TikTok, it combines automation, multi-platform management, and extensible APIs for developers.  
> Last updated: 2026-04-15

## Overview
- **Repository**: [gitroomhq/postiz-app](https://github.com/gitroomhq/postiz-app)  
- **License**: GNU Affero General Public License v3.0  
- **Tech Stack**: TypeScript, Next.js, Redis  
- **Activity**:  
  - **Stars**: 28,670 | **Forks**: 5,103  
  - **Last Updated**: 2026-04-15 | **Open Issues**: 235  
- **Features**:  
  - AI-driven content scheduling and analytics  
  - Supports 14+ platforms (Instagram, YouTube, X, TikTok, etc.)  
  - Public API, Node.js SDK, and automation integrations (N8N, Make.com)  
  - CLI agent for OpenClaw and custom workflows  
- **Usage**:  
  - [Register](https://platform.postiz.com) for the web app  
  - [Docs](https://docs.postiz.com) and [YouTube tutorials](https://youtube.com/@postizofficial) for guidance  

## Key Points
- Open-source alternative to paid social media tools with AGPLv3 licensing  
- High community engagement (28k+ stars, active development as of 2026)  
- AI-enhanced scheduling with cross-platform support for modern networks (e.g., Bluesky, Mastodon)  
- Extensible via APIs, SDKs, and custom nodes for automation platforms
