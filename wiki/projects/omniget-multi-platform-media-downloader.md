# OmniGet: Multi-Platform Media Downloader

> OmniGet is a free and open-source desktop application for downloading videos, courses, and files from over 1000+ platforms including YouTube, Instagram, TikTok, and Chinese services like Bilibili and Douyin. It leverages yt-dlp for broad compatibility and supports torrent downloads and P2P file sharing, making it a versatile media management tool.  
> Last updated: 2026-04-14

## Overview  
OmniGet is a cross-platform desktop app built with **Rust**, **Svelte**, and **Tauri**. It simplifies media downloads by accepting URLs from platforms like YouTube, Instagram, TikTok, and over 1000+ others via [yt-dlp](https://github.com/yt-dlp/yt-dlp). Key features include:  
- **Multi-platform support**: Downloads videos, courses, torrents, and files from services like YouTube, Instagram, Udemy, Bilibili, and Chinese platforms (e.g., Douyin, 小红书).  
- **P2P file sharing**: Transfers files between devices using a 4-word share code, bypassing network restrictions.  
- **Automatic dependency management**: Bundles yt-dlp and FFmpeg, ensuring compatibility and updates without user intervention.  
- **User-friendly interface**: Offers 11 color themes (e.g., Catppuccin, Dracula), 8 language options, and a mascot named Loop for interactive feedback.  

The app supports 1,642 GitHub stars and 96 forks, with a **GNU GPL v3.0** license. It is actively maintained, with the latest update on 2026-04-14.  

## Key Points  
- Downloads videos, courses, and files from **1000+ platforms** via yt-dlp.  
- Built-in torrent client for magnet links and `.torrent` files.  
- Cross-device file sharing over P2P networks.  
- Automatic updates for yt-dlp and FFmpeg dependencies.  
- Open-source with 5 open issues for community contributions.
