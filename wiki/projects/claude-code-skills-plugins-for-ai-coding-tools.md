# Claude Code Skills & Plugins for AI Coding Tools

> This repository provides over 235 production-ready skills and plugins for AI coding tools such as Claude Code, Codex, Gemini CLI, and more. It offers modular instruction packages, Python tools, and reference documentation across domains like engineering, marketing, product, compliance, and C-level advisory. The project is notable for its comprehensiveness and adaptability across multiple platforms.
> Last updated: 2026-04-14

## Overview

The **Claude Code Skills & Plugins** repository is an open-source library that provides a vast collection of modular skills and plugins to enhance AI coding agents. These skills are designed to extend the capabilities of tools like Claude Code, OpenAI Codex, Gemini CLI, Cursor, and others, covering a wide range of domains including engineering, marketing, compliance, and business strategy.

The project includes 235+ skills and 305+ Python-based CLI tools that are portable and require no external dependencies. Skills are structured into **SKILL.md** files, providing clear instructions, workflows, and decision frameworks. The repository supports multiple AI coding platforms and offers a flexible system to activate, combine, and orchestrate skills across different agents and personas.

## Key Points

- Over 235 production-ready skills and plugins for 12 AI coding tools.
- Modular, reusable, and platform-agnostic skills with structured documentation.
- Works with tools like Claude Code, Codex, Gemini CLI, Cursor, and more.
- Includes 305+ Python CLI tools that run anywhere Python is available.
- Offers personas, agents, and orchestration capabilities for complex workflows.
