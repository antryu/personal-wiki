Open WebUI is an extensible, self-hosted AI platform designed for offline operation. It supports LLM runners like **Ollama** and **OpenAI-compatible APIs**, with a built-in RAG inference engine. The platform emphasizes **ease of deployment** (Docker/Kubernetes), **customization** (plugins, themes), and **security** (granular permissions). Key features include voice/video calling, markdown/LaTeX support, and a model builder for Ollama integration. Enterprise users gain access to SLA support, custom branding, and LTS versions.

## Key Points
- Extensible architecture with plugin system and native Python function calling.  
- Multi-API compatibility: Ollama, OpenAI, LMStudio, GroqCloud, Mistral, and OpenRouter.  
- Enterprise-grade features: SLA support, custom theming, and LTS versions.  
- Offline-first design with PWA mobile support and persistent artifact storage.
