# Project AIRI: AI Waifu Companion

> A self-hosted AI companion system inspired by Neuro-sama, enabling users to create and interact with customizable "AI waifu" virtual characters. Features real-time voice chat, game integration (Minecraft/Factorio), and cross-platform support (Windows/macOS/Web).

### Features
- **Self-hosted ownership**: Users control their AI companions without cloud dependency  
- **Real-time interaction**: Voice chat, text-based conversations, and reactive animations  
- **Game integration**: Play Minecraft/Factorio with AI characters via API hooks  
- **Cross-platform support**: Windows, macOS, and web-based deployments  
- **Customization**: VRM/Live2D model support, multilingual documentation, and themeable UI
