# All-in-One AI Assistant Tool

> A cross-platform desktop application that unifies Claude Code, Codex, OpenCode, OpenClaw, and Gemini CLI into a single interface, notable for its high adoption (42.4k stars) and modular design for AI workflow automation.  
> Last updated: 2026-04-10

## Overview
- **Repository**: [farion1231/cc-switch](https://github.com/farion1231/cc-switch)  
- **Description**: Manages multiple AI coding assistants (Claude Code, Codex, Gemini CLI, OpenCode, OpenClaw) via a unified desktop GUI, supporting Windows, macOS, and Linux.  
- **Tech Stack**:  
  - **Language**: Rust (backend) + TypeScript (frontend)  
  - **Framework**: [Tauri 2](https://tauri.app/) for cross-platform desktop app development  
  - **License**: MIT License  
- **Metrics**:  
  - 42.4k stars, 2.6k forks, 490 open issues  
  - Created: 2025-08-04 | Last Update: 2026-04-10  
- **Features**:  
  - Provider management for 5+ AI models  
  - WSL (Windows Subsystem for Linux) integration  
  - Skill/shortcut management for workflows  
  - Multilingual support (English, Chinese, Japanese)  

## Key Points
- Integrates **Claude Code**, **Codex**, **Gemini CLI**, **OpenCode**, and **OpenClaw** into one interface.  
- Built with **Tauri 2** for lightweight, performant desktop apps across Windows, macOS, and Linux.  
- High community adoption (42.4k stars) and active development (490 open issues as of 2026-04-10).  
- Supports advanced workflows via **skill/shortcut management** and **WSL integration**.
