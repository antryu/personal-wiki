### Features
- Runs locally on your computer for secure, offline code generation
- Integrates with VS Code, Cursor, and Windsurf via IDE extensions
- Supports ChatGPT Plus, Pro, Business, Edu, and Enterprise plans
- Cross-platform binaries available for macOS (ARM/x86), Linux (ARM/x86)
- Open source with active community contributions

## Key Points
- **Local execution**: Processes code entirely on your machine without cloud dependency
- **Multiple installation options**: Supports npm, Homebrew, and direct binary downloads
- **ChatGPT plan integration**: Leverages user subscription tiers for enhanced features
- **Extensible architecture**: Designed for IDE and desktop app integrations
