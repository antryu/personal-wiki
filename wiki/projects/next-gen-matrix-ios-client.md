# Next-Gen Matrix iOS Client

> A modern Matrix communication app for iOS built with SwiftUI and the Matrix Rust SDK, offering cross-platform compatibility and active development.  
> Last updated: 2026-04-11

## Overview  
The **Next-Gen Matrix iOS Client** (Element X iOS) is a redesigned Matrix client developed by Element, leveraging SwiftUI for native iOS 18+ experiences and the [Matrix Rust SDK](https://github.com/matrix-org/matrix-rust-sdk) for cross-platform code sharing with Android. It replaces the older Element Classic app, providing improved architecture and integration with modern SDKs.  

Key features include:  
- **Modern tech stack**: Built with SwiftUI and Rust SDK via Swift Package Manager.  
- **Cross-platform alignment**: Shares core logic with Element X Android using the same Rust SDK.  
- **Active development**: Hosted on GitHub with 798 stars, 292 forks, and 445 open issues (as of 2026-04-11).  
- **Community & support**: Maintains a dedicated Matrix room (`#element-x-ios:matrix.org`) and detailed contribution guides.  

The project uses a dual licensing model (GNU AGPLv3 or paid commercial licenses) and is maintained by Element Creations Ltd.  

## Key Points  
- Built with SwiftUI for iOS 18+ and integrates the Matrix Rust SDK via FFI.  
- Actively developed as the recommended replacement for Element Classic.  
- Emphasizes cross-platform code reuse with Android via shared Rust SDK.  
- Hosted under GNU AGPLv3 with optional commercial licensing for enterprises.
