# Project Lyra

> Project Lyra is an open-source initiative by NVIDIA's Spatial Intelligence Lab, focused on developing generative 3D world models capable of creating explorable, long-horizon 3D scenes from single images or videos. It introduces Lyra 1.0 and Lyra 2.0, which advance 3D scene generation through diffusion models and self-distillation techniques.

### Key Features
- **Lyra 1.0** (2025-09-23): Feed-forward 3D/4D scene generation from a single image/video using video diffusion model self-distillation.
- **Lyra 2.0** (2026-04-15): Explorable generative 3D worlds with long-horizon, 3D-consistent generation.
- Open-source implementations and pre-trained models available via [HuggingFace](https://huggingface.co/nvidia/Lyra) and [GitHub](https://github.com/nv-tlabs/lyra).
