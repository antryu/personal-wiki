# AI-Powered Penetration Testing System

> A fully autonomous AI agents system for complex penetration testing tasks, leveraging sandboxed environments, integrated security tools, and knowledge graphs. Notable for its MIT-licensed open-source design and 14.7k+ GitHub stars.  
> Last updated: 2026-04-10

## Overview  
The **PentAGI** repository (https://github.com/vxcontrol/pentagi) is an open-source AI-powered penetration testing platform written in Go. It enables autonomous security testing through AI agents, sandboxed Docker environments, and a suite of 20+ integrated tools (e.g., nmap, metasploit). Key features include:  
- **Autonomous operation**: AI agents execute penetration testing steps with optional monitoring and task planning.  
- **Secure architecture**: All operations run in isolated Docker containers for safety.  
- **Advanced integrations**: Knowledge graph (Neo4j via Graphiti), web scraping, and APIs (Tavily, Perplexity, etc.).  
- **Tooling**: Built-in professional security tools and smart container management.  
- **Scalability**: REST/GraphQL APIs, PostgreSQL/pgvector storage, and Grafana/Prometheus monitoring.  

The project, created in 2025, has 14.7k stars and 1.9k forks, with active community engagement via Discord and Telegram.  

## Key Points  
- **Autonomous AI agents** automate penetration testing workflows with intelligent task planning and execution monitoring.  
- **Sandboxed security** ensures operations are isolated via Docker, minimizing risk to host systems.
