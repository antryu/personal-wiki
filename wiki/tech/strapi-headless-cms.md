# Strapi Headless CMS

> Strapi is the leading open-source headless CMS built entirely in JavaScript/TypeScript. It offers a developer-first approach with full customization, self-hosting or cloud deployment options, and support for modern web frameworks.  
> Last updated: 2026-04-16

## Overview  
**Strapi** is a flexible, open-source headless CMS designed for developers. It provides tools to manage content dynamically while supporting REST and GraphQL APIs, multi-database systems, and extensible admin panels.

### Key Features  
- **Deployment Options**: Self-hosted or cloud (Strapi Cloud).  
- **Admin Panel**: Customizable and extensible interface for content management.  
- **Database Support**: PostgreSQL, MySQL, MariaDB, SQLite.  
- **Extensibility**: APIs, routes, and plugins can be tailored to specific needs.  
- **Performance**: Built on Node.js and TypeScript for speed and scalability.  
- **Front-End Agnostic**: Compatible with React, Vue, Angular, mobile apps, and IoT.  
- **Security**: Default protections for CORS, XSS, and more.  
- **CLI Tools**: Streamlined project and API scaffolding.  

### Getting Started  
1. **Install via CLI**:  
   ```bash  
   npx create-strapi@latest my-project  
   ```  
   This command initializes a project with authentication, permissions, and core features.  

2. **System Requirements**:  
   | OS              | Recommended | Minimum    |  
   |-----------------|-------------|------------|  
   | Ubuntu          | 24.04       | LTS        |  
   | Debian          | 11          | LTS        |  
   | RHEL            | 9           | LTS        |  
   | macOS           | 14          | 12         |  
   | Windows Desktop | 11          | 10         |  
   | Windows Server  | No Support  | No Support |  

Full documentation is available at [Strapi Docs](https://docs.strapi.io/).
