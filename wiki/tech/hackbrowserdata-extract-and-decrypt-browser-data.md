# HackBrowserData: Extract and Decrypt Browser Data

> HackBrowserData is a powerful command-line tool for decrypting and exporting browser data, supporting multiple data types and operating systems including macOS, Windows, and Linux. It is widely used in security research and notable for its cross-browser and cross-platform compatibility.
> Last updated: 2026-05-10

## Overview

HackBrowserData is a Go-based command-line utility that allows users to extract and decrypt sensitive data from popular web browsers. It supports a wide range of data categories such as passwords, cookies, bookmarks, history, credit card information, and more from browsers including Chrome, Firefox, Safari, and others. The tool is actively maintained and has a large community, with over 13,952 stars on GitHub. It is developed under the MIT license and is designed to be used for ethical purposes in security research.

The repository provides detailed README documentation, including supported data types, browser compatibility, and installation instructions. The project is available for multiple operating systems and requires minimal setup for use. However, on macOS, certain browsers may require user authentication or specific permissions like Full Disk Access for successful data extraction.

HackBrowserData is developed by moonD4rk and is continuously updated to support new browser versions and operating system updates. It includes automated workflows for linting, building, testing, and releasing, ensuring the reliability and maintainability of the codebase.

## Key Points

- Extract and decrypt browser data including passwords, cookies, bookmarks, and history from popular browsers like Chrome, Firefox, and Safari.
- Supports multiple operating systems including Windows, macOS, and Linux.
- Developed in Go (Golang) and licensed under the MIT License.
- Widely used in security research and penetration testing.
- Requires specific permissions like Full Disk Access on macOS for some browsers.
