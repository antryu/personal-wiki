# Whisky: Modern Wine Wrapper for macOS

> Whisky is a modern Wine wrapper for macOS built with SwiftUI, designed to simplify the installation and management of Windows applications and games on Apple Silicon Macs. It integrates seamlessly with macOS and leverages Apple's Game Porting Toolkit to enhance performance. Last updated: 2026-05-16

## Overview

Whisky provides a user-friendly graphical interface for Wine, allowing users to create and manage "bottles"—isolated environments for running Windows apps and games—with minimal technical knowledge. Built using SwiftUI, it offers native macOS integration, intuitive UI elements, and one-click bottle management.

The tool is based on CrossOver 22.1.1 and supports Apple's Game Porting Toolkit (GPT) for better performance on M-series chips. However, it is no longer actively maintained, so users should be aware that apps or games may break without prior notice.

Key features include:
- One-click bottle creation and management
- Debugging and profiling tools
- Integration with Crowdin for multilingual support
- Installation via Homebrew
- Specialized game support via a dedicated wiki

## Key Points

- Built with SwiftUI for native macOS integration  
- Requires Apple Silicon (M-series) and macOS Sonoma 14.0 or later  
- Based on CrossOver 22.1.1 and Apple's Game Porting Toolkit  
- One-click setup and management of Windows apps and games  
- No longer actively maintained — apps may break at any time
