# Assistant UI Library

> A TypeScript/React library for building production-grade AI chat experiences with customizable primitives and broad model integration. Notable for its enterprise-ready features, fast deployment, and support for frameworks like Vercel AI SDK and LangGraph.  
> Last updated: 2026-04-18

## Overview
The **assistant-ui** library provides a toolkit for developers to create AI chat interfaces in React applications. It abstracts complex features like streaming, auto-scrolling, and accessibility while offering composable primitives inspired by shadcn/ui and cmdk. The library supports integration with major AI providers (OpenAI, Anthropic, Google Gemini, etc.) and frameworks like Vercel AI SDK, LangGraph, and Mastra. It includes production-ready components for message rendering, input handling, and tool call visualization, with optional enterprise features like chat history and analytics via Assistant Cloud.

Key capabilities:
- Pre-built streaming and real-time updates
- Customizable UI components (message list, input, thread)
- Broad model provider compatibility with extensible API
- Keyboard shortcuts, accessibility, and TypeScript support
- Optional managed chat persistence via Assistant Cloud

## Key Points
- **Fast deployment**: Ready-to-use primitives with built-in streaming, attachments, and markdown support
- **Customizable architecture**: Composable components instead of monolithic widgets
- **Enterprise integration**: Optional cloud-based analytics and chat history
- **Framework flexibility**: Works with Vercel AI SDK, LangGraph, Mastra, or custom backends
