# Custom CLI Status Line

> A beautiful and highly customizable status line for the Claude Code CLI, featuring powerline support, themes, and a wide range of widgets for displaying git, model, token usage, and more. It's notable for its flexibility and integration with modern development workflows.
> Last updated: 2026-04-17

## Overview

The **ccstatusline** project offers a customizable status line for the Claude Code CLI. It allows developers to display relevant information such as model details, git branch status, token usage, and more in a clean and organized manner in their terminal.

The project is built with TypeScript and follows the MIT License. It includes support for Powerline themes, a rich set of widgets, and localization options. The latest version, v2.2.8, includes features like a Git PR widget, smarter widget picker search, and minimalist mode for a cleaner UI.

## Key Points

- **Highly customizable**: Users can configure widgets, themes, and layout to suit their workflow.
- **Powerline and theme support**: Enables seamless integration with modern terminal themes and layouts.
