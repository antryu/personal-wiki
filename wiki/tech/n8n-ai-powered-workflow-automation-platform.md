# n8n - AI-Powered Workflow Automation Platform

> n8n is a fair-code workflow automation platform that combines visual interface building with custom code, offering native AI capabilities and 400+ integrations. It enables technical teams to build secure, scalable automations with full control over data and deployments.  
> Last updated: 2026-04-10

## Overview

n8n is a **low-code/no-code workflow automation platform** designed for technical teams, offering flexibility between visual workflows and custom code (JavaScript/Python). It features **native AI integration** (e.g., LangChain-based agent workflows), **self-hosting or cloud deployment**, and a **fair-code license** that balances open-source accessibility with sustainable development. With 400+ pre-built integrations and 900+ ready-to-use templates, it supports enterprise-grade use cases including SSO, air-gapped deployments, and advanced permissions.  

The platform emphasizes **developer control** through npm package support, extensible architecture, and self-hosted options. It is actively maintained (183k+ GitHub stars, 56k+ forks) and backed by a large community and enterprise support.  

### Key Features
- **Hybrid Automation**: Visual workflow builder + code execution (JavaScript/Python).  
- **AI-Native**: Build custom AI agent workflows using LangChain and private models/data.  
- **Deployment Flexibility**: Self-hosted (Docker, Node.js) or managed cloud (n8n.cloud).  
- **Enterprise-Ready**: Role-based access, SSO, audit logs, and air-gapped compatibility.  
- **Extensibility**: Add custom integrations (nodes) or leverage 400+ existing ones (Slack, Google Sheets, etc.).  

### Quick Start
Run locally via **Node.js** or **Docker**:
```bash
# Node.js
npx n8n

# Docker
docker run -p 5678:5678 docker.n8n.io/n8nio/n8n
```
Access the editor at [http://localhost:5678](http://localhost:5678).
