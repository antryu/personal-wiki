# CCX - AI API Proxy

> CCX is a high-performance AI API proxy and protocol translation gateway for Claude, OpenAI Chat, OpenAI Images, Codex Responses, and Gemini. It provides a unified entrypoint, built-in web administration, channel orchestration, failover, multi-key management, and model routing.
> Last updated: 2026-05-11

## Overview
The **CCX (Claude / Codex / Gemini API Proxy)** is a powerful AI API proxy and gateway built in Go. It allows developers to route and manage requests to different AI services—such as Claude, OpenAI, and Gemini—through a single, unified interface. With features like real-time monitoring, load balancing, and a built-in web administration console, CCX simplifies the integration and management of AI API backends.

The project has been actively maintained since January 2026 and has attracted 450 stars and 49 forks. It is distributed under the MIT License and supports flexible deployment options, including binary downloads, Docker, and source builds.

## Key Points
- Offers a unified API entrypoint for multiple AI backends including Claude, OpenAI, Codex, and Gemini.
- Includes a web-based administration dashboard for real-time monitoring and management of API channels.
- Supports advanced features like health checks, failover, and smart scheduling.
- Provides a single-port deployment with embedded frontend assets for easy setup.
- Enables flexible API key management, model routing, and per-channel configurations.
- Offers Docker and binary deployment methods for easy integration into production environments.
