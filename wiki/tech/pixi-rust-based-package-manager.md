# Pixi - Rust-based Package Manager

> Pixi is a powerful, cross-platform package manager for Linux, macOS, and Windows written in Rust. It is built on top of the Conda ecosystem and aims to provide a simple and fast experience similar to Cargo or npm. Last updated: 2026-05-08

## Overview

Pixi is a next-generation package manager and workflow tool that simplifies multi-language development workflows using the Conda ecosystem. It supports multiple programming languages, including Python, C++, and R, and offers a Cargo-like command-line interface for managing dependencies. Developed by [prefix.dev](https://prefix.dev), Pixi is fully open source under the BSD 3-Clause License and built using the [Rust](https://www.rust-lang.org/) programming language.

Key features include:
- Cross-platform support for Linux, macOS, and Windows.
- Support for multiple languages through Conda packages.
- A Cargo-like CLI for intuitive command usage.
- A lock file system to ensure consistent environments.
- Compatibility with Conda while improving speed and usability.
- A focus on simplicity and performance for developers and system administrators.

## Key Points
- Pixi is built on top of the Conda ecosystem, enhancing its functionality and usability.
- It supports multiple programming languages and is compatible with all major operating systems.
- Pixi is written in Rust, leveraging its performance and safety features for a more robust and fast experience.
