# Database DevSecOps Platform

> Bytebase is an open-source database DevSecOps platform designed for Developer, Security, DBA, and Platform Engineering teams. It provides GitOps-style database-as-code workflows, security, compliance tools, and automated schema management, making it the GitHub/GitLab for database operations. It is notable for being the only database CI/CD project included in the CNCF Landscape and the Platform Engineering tool list.
> Last updated: 2026-05-13

## Overview

The **Database DevSecOps Platform** is a solution that addresses the challenges of managing database schema changes in a secure, scalable, and collaborative manner. The platform is built by Bytebase, an open-source and CNCF-listed project. It supports GitOps-style database-as-code workflows, enabling teams to manage database schemas with the same rigor and automation as application code.

The platform offers features such as schema migration, SQL review, data masking, and fine-grained access control. It is designed for teams looking to integrate database changes seamlessly into their CI/CD pipelines while ensuring compliance and security. The platform supports a wide range of databases, including PostgreSQL, MySQL, MongoDB, Snowflake, and more.

The tool includes a web-based console, CLI capabilities, and integrations with GitHub, GitLab, and Terraform, allowing teams to automate and track changes with full visibility and control.

## Key Points

- Provides **GitOps-style database-as-code workflows** for schema management and version control.
- Includes **comprehensive security and compliance features** such as data masking, RBAC, and audit logging.
- Offers **multi-database support** for major databases like PostgreSQL, MySQL, MongoDB, and Snowflake.
- Integrates with GitHub, GitLab, and Terraform to enable automation and infrastructure-as-code practices.
