# Ultralytics YOLO

> Ultralytics YOLO is a cutting-edge implementation of the YOLO (You Only Look Once) object detection architecture, offering fast, accurate, and easy-to-use models for a wide range of computer vision tasks. It supports object detection, tracking, segmentation, classification, and pose estimation. The project is notable for its performance, flexibility, and active development by Ultralytics.
> Last updated: 2026-04-14

## Overview

Ultralytics YOLO is an open-source repository for the YOLO object detection framework, developed and maintained by Ultralytics. The framework provides a unified solution for multiple computer vision tasks, including object detection, instance segmentation, image classification, pose estimation, and tracking. It is built using PyTorch and is designed to be user-friendly, efficient, and highly customizable.

The project offers a variety of YOLO versions, including YOLOv8, YOLO11, and YOLO26, each optimized for specific use cases. The codebase includes extensive documentation, tutorials, and tools for model training, inference, and deployment.

Key features of the project include:
- High performance and accuracy across multiple vision tasks.
- Easy-to-use command-line and Python APIs.
- Integration with popular platforms like Colab, Kaggle, and Gradient for model training.
- Support for multiple languages in documentation, including Chinese, Korean, Japanese, and more.
- A vibrant community for support and collaboration via Discord, Reddit, and forums.

The repository is actively maintained, with the last update on April 14, 2026, and it has a strong presence on GitHub with over 55,000 stars and 10,000 forks.

## Key Points
