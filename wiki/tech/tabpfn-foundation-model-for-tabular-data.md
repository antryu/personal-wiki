# TabPFN: Foundation Model for Tabular Data

> TabPFN is a foundation model for tabular data, designed to enable fast and efficient machine learning with minimal user intervention. It leverages a pre-trained architecture optimized for both classification and regression tasks, making it a powerful tool for data scientists and machine learning practitioners. The project has gained significant traction in the community and continues to be actively maintained.
> Last updated: 2026-05-05

## Overview

TabPFN is a cutting-edge foundation model developed by PriorLabs for handling tabular data. It provides a robust and user-friendly API for performing classification and regression tasks. The model is trained on synthetic data and is designed to be used out of the box, eliminating the need for extensive data preprocessing. TabPFN is optimized to run efficiently on GPU hardware, enabling high performance for both small and large datasets.

Key features of the TabPFN include:
- **Pretrained models**: Comes with default models such as TabPFN-2.6, with the ability to select other versions.
- **GPU acceleration**: Designed for optimal performance on GPU hardware.
- **Minimal setup**: Requires no complex preprocessing or hyperparameter tuning.
- **Interactive tutorials**: Offers a Colab notebook to guide users through installation and usage.
- **Scalability**: Capable of handling datasets with up to 100,000 samples and 2000 features.

The repository includes tools for installation via pip or source, example notebooks, and clear usage instructions to help users get started quickly.

## Key Points
- TabPFN is a foundation model for tabular data optimized for classification and regression.
- It requires minimal data preprocessing and is trained on synthetic data.
- The model is available via pip and supports GPU acceleration.
- Users can choose different model versions such as TabPFN-2.5 or TabPFN-2.6.
- The project is actively maintained and features community engagement through a Discord server.
