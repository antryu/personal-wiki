# CodeGraph: Pre-indexed Code Knowledge Graph for Claude Code

> CodeGraph is a pre-indexed code knowledge graph for Claude Code, designed to reduce the number of tool calls and improve exploration speed while operating entirely locally.  
> Last updated: 2026-05-08

## Overview
CodeGraph is a tool that enhances Claude Code's ability to explore codebases by providing pre-indexed knowledge graphs. It offers significant performance improvements, reducing the number of tool calls and file reads while speeding up the overall exploration process. Developed in TypeScript and distributed under the MIT License, CodeGraph is fully local and supports Windows, macOS, and Linux. It is especially useful in large codebases where Claude Code's Explore agents typically rely on tool calls for scanning files.

## Key Points
- **Reduces tool calls by up to 94%**, enabling faster and more efficient code exploration.
- **Speeds up exploration by up to 82%**, particularly noticeable in large projects.
- **100% local execution** eliminates the need for external tools or network requests.
- **Works with multiple languages**, including TypeScript, Python, Rust, Swift, and C++.
- **Supports popular codebases** such as VS Code, Excalidraw, Claude Code, Alamofire, and the Swift Compiler.
