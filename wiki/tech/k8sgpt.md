# k8sgpt

> k8sgpt is a powerful tool that gives Kubernetes clusters diagnostic and triaging capabilities by translating complex cluster data into simple, actionable insights using AI. It is notable for integrating advanced AI backends and offering SRE-level analysis in an accessible format.
> Last updated: 2026-05-16

## Overview
`k8sgpt` is a command-line tool designed to analyze and diagnose Kubernetes clusters using AI. It codifies SRE (Site Reliability Engineering) experience into its analyzers, helping users extract and interpret the most relevant information from their clusters in simple English. With support for multiple AI backends, including OpenAI, Azure, Cohere, Amazon Bedrock, and Google Gemini, it is a versatile solution for Kubernetes management.

The project is actively maintained, has a large open-source community, and offers extensive documentation. It has been featured on platforms like Product Hunt and HelloGitHub, and is backed by a strong license (Apache License 2.0) and code quality practices.

The tool provides various installation methods, including Homebrew, RPM, and DEB packages, and includes a range of features such as analyzers, custom integrations, and the Model Context Protocol (MCP). It is designed to be lightweight and efficient while delivering powerful insights.

## Key Points
- Provides SRE-level analysis of Kubernetes clusters in simple, human-readable language.
- Integrates with multiple AI backends (OpenAI, Azure, Cohere, Amazon Bedrock, Google Gemini, and local models).
- Offers out-of-the-box tools for cluster diagnostics and triaging.
- Supports multiple installation methods for Linux, macOS, and RPM/DEB-based systems.
- Maintains high-quality code and community practices with an Apache License 2.0.
- Features a Model Context Protocol (MCP) for better AI model interaction and data handling.
