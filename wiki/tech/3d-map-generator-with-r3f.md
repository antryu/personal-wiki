# 3D Map Generator with R3F

> This repository provides a 3D city map generator using React-Three-Fiber (R3F), allowing for the creation of real-world 3D maps that include buildings and roads. It is notable for its integration with OpenStreetMap data and its export capabilities to GLB files.
> Last updated: 2026-05-11

## Overview

The project, named **map3d**, is a 3D building mapping service built with [React-Three-Fiber](https://github.com/pmndrs/react-three-fiber). It enables users to generate 3D city maps and export them as GLB files for use in various applications like **digital twins**, **drone surveying**, and **GPS marker implementations**. The maps are based on OpenStreetMap data, allowing for real-world representations of urban landscapes.

The repository has been actively maintained, updated on May 11, 2026, with a total of 717 stars and 104 forks. It is open-source under the MIT License, and it has 5 open issues for community contributions.

A roadmap outlines the project's development goals, including features such as **building texture**, **height customization**, and **heightmaps**, some of which are already implemented.

## Key Points
- Built with React-Three-Fiber (R3F) and TypeScript for 3D visualization.
- Uses OpenStreetMap data for real-world accuracy (with known limitations in height values).
- Provides GLB export capability for 3D models of cities and buildings.
- Features include digital twin support, drone surveying, and GPS marker integration.
- Active development with a roadmap outlining future features.
- Open-source and MIT-licensed for easy adoption and customization.
