# Datadog iOS SDK

> The `dd-sdk-ios` repository provides the official Datadog SDK for iOS development, enabling real-time application performance monitoring, error tracking, and observability in Swift and Objective-C projects. It is notable for its integration with Datadog's comprehensive monitoring platform, widely used for troubleshooting and optimizing mobile applications.  
> Last updated: 2026-04-11

## Overview
- **Repository**: [DataDog/dd-sdk-ios](https://github.com/DataDog/dd-sdk-ios)  
- **Description**: Official SDK for integrating Datadog monitoring into iOS apps.  
- **Key Features**:  
  - Real-time performance metrics collection.  
  - Crash and error tracking.  
  - Support for both Swift and Objective-C.  
- **Popularity**: 279 stars (no recent star changes today).  
- **Primary Language**: Swift.  

## Key Points
- Enables developers to monitor iOS app performance and user interactions via Datadog's platform.  
- Designed for seamless integration with minimal code changes.
