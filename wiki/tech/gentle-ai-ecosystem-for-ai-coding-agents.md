# Gentle-AI Ecosystem for AI Coding Agents

> Gentle-AI is an ecosystem configurator for AI coding agents, enhancing them with persistent memory, Spec-Driven Development (SDD) workflows, curated coding skills, AI provider switching, and more. It transforms basic AI coding tools into powerful, context-aware agents that teach and adapt.  
> Last updated: 2026-05-05

## Overview

Gentle-AI is not just another AI agent installer — it is a framework that enriches and extends the functionality of 12 popular AI coding agents, such as **Claude Code**, **Cursor**, and **OpenCode**. It enables developers to create smarter, more capable AI workflows by integrating persistent memory, task delegation models, and flexible model routing per SDD phase.

At its core, Gentle-AI offers a unified configuration system for AI agents, making it easy to manage workflows and skills across platforms. The system includes support for multiple AI providers, a teaching-oriented persona, and secure, permission-based execution models.

Gentle-AI supports a wide array of agents and provides a consistent, configurable interface for deploying and managing AI-powered coding assistants. The project also includes tools for skill registration, SDD initialization, and project context detection, ensuring that the AI agents adapt to the specific needs of each project.

The project is written in Go, licensed under the MIT license, and is actively maintained with features added in regular releases.

## Key Points

- Integrates 12 popular AI coding agents into a single, customizable ecosystem.
- Provides persistent memory and SDD workflows for enhanced agent behavior.
- Supports per-phase model switching and multi-mode overlay configurations.
- Offers tools for skill registration and project context setup.
- Includes a teaching-oriented persona and secure execution environment.
- Replaces and improves upon the deprecated [Agent Teams Lite](https://github.com/Gentleman-Programming/agent-teams-lite).
