# Maxun: No-Code Web Scraping & API Platform

> Maxun is an open-source no-code platform that transforms websites into structured APIs through web scraping, crawling, search, and AI-powered data extraction. It enables users to automate data workflows without coding, making it a notable tool for developers and non-technical users.  
> Last updated: 2026-04-17

## Overview  
### Key Features  
- **No-code automation**: Extract data, scrape pages, and crawl websites using intuitive interfaces.  
- **AI-powered extraction**: Leverage natural language prompts and LLMs to define data extraction rules.  
- **Scalable workflows**: Supports simple tasks (e.g., property listings) and complex automation (e.g., website crawling).  
- **Self-hosted & open-source**: Licensed under GNU AGPLv3, with full transparency and customization.  

### Ecosystem  
Maxun offers specialized "robots" for different data tasks:  
1. **Extract**:  
   - **Recorder Mode**: Record browser interactions to create reusable extraction workflows.  
   - **AI Mode**: Use natural language to define extraction goals (e.g., "Extract movie titles and ratings").  
2. **Scrape**: Convert webpages to clean Markdown/HTML and capture screenshots for AI/document workflows.  
3. **Crawl**: Traverse entire websites with customizable scope and discovery rules.  
4. **Search**: Automate web searches with time-based filters (e.g., "recent news articles").  
5. **SDK/CLI**: Developer tools for automation, scheduling, and integration.  

### How It Works  
Maxun robots emulate user behavior to collect structured data. For example:  
- **Recorder Mode Demo**: Extract Airbnb property listings by recording browsing actions.  
- **AI Mode Demo**: Extract IMDb movie data using natural language prompts.  

## Key Points  
- Open-source no-code platform for web scraping, crawling, and API generation.  
- Combines browser automation (Playwright) with AI extraction for flexible workflows.  
- High community engagement (15.4k stars, 1.2k forks on GitHub).
