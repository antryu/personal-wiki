# BunkerWeb - Open-source Web Application Firewall

> BunkerWeb is a next-generation, open-source Web Application Firewall (WAF) designed to provide robust security for web services by default. It integrates seamlessly into various environments, including Linux, Docker, Kubernetes, and more, and features an intuitive web UI for configuration.  
> Last updated: 2026-05-05

## Overview

BunkerWeb is a full-featured web server and WAF based on NGINX under the hood, designed to protect web services and enforce security by default. It acts as a reverse proxy and supports integration into a wide range of environments, including Linux, Docker, Docker Swarm, and Kubernetes. BunkerWeb offers a plugin system for extending its capabilities and includes a web-based user interface (UI) to simplify configuration for those who prefer not to use the command line.

BunkerWeb is cloud-native and open-source, developed under the GNU Affero General Public License v3.0. It provides a variety of security features out of the box, such as antibot capabilities, DNS-based blacklists (DNSBL), and security hardening, and is actively maintained with a strong community and CI/CD workflows. It also includes tools for threat mapping, status tracking, and community discussions to support collaboration and feedback.

## Key Points
- BunkerWeb is an open-source WAF that offers security by default with a plugin-based architecture.
- It integrates easily into Linux, Docker, Kubernetes, and other cloud-native environments.
- A user-friendly web UI is available for those who prefer a graphical interface over the CLI.
