# AI PPT Generator

> AI generates natively editable PPTX files from documents, PDFs, or URLs using real PowerPoint shapes and text boxes instead of images. This tool bridges domain expertise and product engineering by enabling zero-design-skill presentation automation.  
> Last updated: 2026-04-12

## Overview
The **hugohe3/ppt-master** repository provides an AI-powered solution for converting documents into fully editable PowerPoint presentations. Key features include compatibility with Claude Code, VS Code Copilot, and other AI editors, along with 10+ output formats (e.g., PPT 16:9, social media cards, marketing posters). The project, created by finance professional Hugo He (CPA · CPV · Consulting Engineer), addresses the frustration of non-editable AI-generated slides by producing native DrawingML objects.
