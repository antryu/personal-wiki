# Claude Code Plugins Directory

> This repository serves as an official directory of high-quality Claude Code plugins managed by Anthropic. It provides a platform for developers to contribute both internal and external plugins, offering users access to enhanced functionality and tools directly within the Claude Code environment. Last updated: 2026-05-08

## Overview

The **Claude Code Plugins Directory** is a curated collection of plugins developed and maintained by Anthropic and third-party contributors. It features a structured layout with separate directories for internal and external plugins, each following a consistent format to ensure ease of use and integration. The plugins can be installed directly within the Claude Code IDE using either the plugin installer command or the Discover interface. The project emphasizes user responsibility, noting that Anthropic does not verify the functionality or reliability of all plugin contents. Developers can contribute new plugins via a submission form, provided they meet quality and security standards.

## Key Points

- Hosts both **internal plugins** (developed by Anthropic) and **external plugins** (from partners and the community).
- Plugins are installed directly in Claude Code using commands or the Discover feature.
- Each plugin follows a **standard structure** that includes metadata, configuration, and documentation files.
- Contributors must meet **quality and security standards** for inclusion in the directory.
- Users are advised to **review plugin documentation and trustworthiness** before installation or use.
