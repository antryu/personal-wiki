# VibeProxy AI Coding Tool

> A native macOS menu bar app that enables users to leverage their Claude Code and ChatGPT subscriptions with AI coding tools without requiring API keys. Notable for its seamless integration and growing popularity among developers.  
> Last updated: 2026-04-11

## Overview  
VibeProxy simplifies the use of AI coding tools by acting as a bridge between user subscriptions (Claude Code, ChatGPT) and development workflows. Built natively for macOS, it operates directly from the menu bar, offering quick access to AI-powered coding features.
