# scrcpy: Android Screen Mirroring and Control Tool

> scrcpy is a tool that allows you to mirror and control an Android device's screen from your computer. It is lightweight, high-performance, and does not require root access or installation on the device.  
> Last updated: 2026-05-14

## Overview

**scrcpy** is a free and open-source application that enables users to display and control an Android device connected via USB or TCP/IP. It supports major operating systems (Linux, Windows, macOS) and is designed for performance with low latency and high frame rates. It allows full control of the device using a computer's keyboard and mouse and supports features like audio forwarding, screen recording, camera mirroring, and more.

The application is notable for being non-intrusive — no app or configuration is left on the Android device after use — and for its focus on user benefits like no ads or account requirements. It is also highly configurable and customizable.

Key features include:

- **Lightweight and native interface**, displaying only the Android screen.
- **Performance**: up to 120fps.
- **Low latency**: 35-70ms.
- **Low startup time**: ~1 second.
- **Cross-platform support**: works on Linux, Windows, and macOS.
- **No device installation required**: nothing is left on the device after mirroring.
- **Open Source**: under the Apache License 2.0.

## Key Points

- Supports Android API level 21 (Lollipop) and above.
- No root access required for most operations.
- Allows screen mirroring and control using a computer’s input devices.
- Supports advanced features like screen recording, camera mirroring, and gamepad support.
- Compatible with TCP/IP (wireless) connections.
