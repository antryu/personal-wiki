# AI-First Design Tool for React

> An open-source AI-first design tool for React that enables designers to visually build, style, and edit their React apps in real-time. It offers a prompt-to-build experience with an open-source alternative to tools like Figma and Webflow. Last updated: 2026-04-19

## Overview

**AI-First Design Tool for React**, known as **Onlook**, is a visual-first code editor that allows users to craft websites, prototypes, and designs using AI in Next.js and TailwindCSS. It provides a Figma-like interface for real-time editing and offers features such as brand asset management, page navigation, and component detection. Onlook is still under active development and welcomes contributors to help shape its future.

The project is open-source and licensed under the Apache License 2.0. It is built with TypeScript and is hosted on GitHub, where it has gained 25,466 stars and 1,955 forks since its creation in June 2024. The tool is notable for its integration of AI into the design-to-code workflow, making it a powerful option for developers and designers alike.

### Key Features

- **Visual Editor**: Edit your app directly in the browser DOM with a Figma-like UI.
- **Real-time Preview**: Instantly see changes as you make them.
- **AI Integration**: Use AI to generate and refine code based on prompts.
- **Next.js + TailwindCSS**: Build modern React apps with these popular frameworks.
- **Open Source**: Contribute to the project and customize it to fit your needs.
- **Deployment**: Easily generate sharable links and connect custom domains.
- **Multi-language Support**: Available in multiple languages including Chinese, Spanish, German, French, Portuguese, Russian, Japanese, and Korean.

## Key Points

- Onlook is an open-source, AI-first design tool for React that enables real-time visual editing.
- It provides a prompt-to-build experience with AI integration for generating and refining code.
- The project is still in active development and welcomes contributors to help with feature implementation and issue resolution.
- Onlook supports real-time code editing, component detection, and deployment with custom domains.
- It is an alternative to tools like Figma, Webflow, and other design-to-code platforms.
