# Jellyfin Media System Backend

> Jellyfin is a free software media system that allows users to manage and stream media from a server to various devices. It serves as an open-source alternative to proprietary platforms like Emby and Plex.  
> Last updated: 2026-05-04

## Overview

Jellyfin is a cross-platform media system that enables users to host, manage, and stream media files across devices. The project was forked from Emby's 3.5.2 release and was rewritten to support the .NET platform, allowing full cross-platform compatibility. It provides a centralized media server with a rich API and a variety of client applications for different devices.

The project is actively maintained and offers a wide range of features, including media library management, remote access, transcoding, and support for plugins and extensions. It is designed to be community-driven and open, with no restrictions or premium tiers.

Key features include:
- A modular, API-based architecture.
- Support for a variety of media formats and container types.
- Integration with third-party services like subtitles, metadata, and artwork providers.
- A vibrant community contributing to development, translations, and documentation.

For installation and usage, Jellyfin can be downloaded from its [official downloads page](https://jellyfin.org/downloads), and detailed installation instructions are available in their [documentation](https://jellyfin.org/docs/general/installation/).

## Key Points

- Jellyfin is a free and open-source media system designed as an alternative to commercial media servers like Emby and Plex.
- It is fully cross-platform, supporting Windows, macOS, Linux, Docker, and more.
- The project is community-driven, with no premium features or licensing fees.
- The codebase is written in C# and uses .NET for cross-platform capabilities.
- It offers a robust API for developers and integrators to create custom clients and plugins.
