# Build a Large Language Model from Scratch

> This repository provides a step-by-step implementation of a ChatGPT-like large language model (LLM) in PyTorch, from scratch. It is the official companion to the book "Build a Large Language Model (From Scratch)" and offers practical insights into developing and training LLMs.
> Last updated: 2026-05-11

## Overview
The `rasbt/LLMs-from-scratch` repository serves as the official code repository for the book *Build a Large Language Model (From Scratch)*. It includes practical Jupyter Notebook implementations that guide users in developing, pretraining, and fine-tuning a GPT-like LLM from the ground up. The approach mirrors the techniques used in large-scale foundational models like ChatGPT, enabling users to build a functional model for educational purposes.

The repository includes code to load and fine-tune larger pre-trained models, as well as instructions for setting up the environment. It has been updated regularly and is supported across Linux, Windows, and macOS platforms.

## Key Points
- Full step-by-step implementation of a GPT-like LLM in PyTorch.
- Accompanies the Manning book "Build a Large Language Model (From Scratch)".
- Supports educational training of functional models for learning and experimentation.
- Includes tools and code for fine-tuning pre-trained models.
- Cross-platform support with automated testing for Linux, Windows, and macOS.
- Comprehensive troubleshooting and setup guides are included.
