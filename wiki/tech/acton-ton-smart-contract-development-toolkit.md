# Acton - TON Smart Contract Development Toolkit

> Acton is an all-in-one TON smart contract development toolkit written in Rust. It offers a comprehensive CLI for the full contract lifecycle, including project scaffolding, build, testing, scripting, wallet and network operations, verification, linting, formatting, debugging, and low-level VM tooling.  
> Last updated: 2026-05-13

## Overview

Acton is a powerful toolchain designed for TON (Telegram Open Network) smart contract development. Built in Rust, it provides a unified command-line interface (CLI) that covers the entire smart contract lifecycle—from creation to deployment and verification.

Acton's unique features include support for Tolk (TON's contract language), built-in wrappers, testing utilities, and scripting, making it ideal for dApp development. It also includes a fast test runner with advanced testing capabilities such as fork mode, gas snapshots, coverage, mutation, and fuzzing, along with a browser test UI for inspecting failed tests, traces, logs, and coverage.

The project is actively maintained and supports macOS (ARM64, x86_64) and Linux (x86_64, ARM64). It can be installed via a script or manually for different platforms and is also available as a Docker image.

## Key Points
- Offers a full lifecycle smart contract development experience in a single CLI.
- Built with Rust for performance and speed.
- Supports Tolk-first workflows and provides built-in testing and debugging tools.
- Includes project templates and auto-generated TypeScript wrappers for dApp development.
- Provides a fast and feature-rich test runner with advanced capabilities like fuzzing and mutation testing.
- Available for macOS and Linux platforms with Docker support.
