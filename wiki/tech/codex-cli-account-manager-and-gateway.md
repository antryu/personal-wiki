# Codex CLI Account Manager and Gateway

> CodexManager 是一个 Codex CLI 账号管理与切换工具，提供本地网关转发功能。它支持统一管理 Codex 账号、使用量和平台 Key，并具备本地网关能力。该项目因其灵活性和对多平台的支持而受到开发者关注。
> Last updated: 2026-04-16

## Overview
CodexManager 是一个开源项目，旨在为 Codex CLI 提供账号管理与本地网关功能。该项目由 Rust 编写，采用 MIT 许可证，旨在帮助用户高效管理多个 AI 平台的账号资源，如 OpenAI、Anthropic 等，并提供灵活的本地代理能力，以满足开发者在多模型调用场景下的需求。

该项目由开发者 qxcnm 与 AI 协作开发，支持 Windows 平台的桌面端使用，其他平台的兼容性则依赖社区反馈与测试。CodexManager 提供了丰富的配置选项，包括端口设置、代理配置、数据库集成等，并支持插件扩展，便于用户根据需求定制功能。

## Key Points
- 提供 Codex CLI 账号管理、切换与本地网关能力。
- 支持多平台账号统一管理，包括 API Key 和使用量统计。
- 提供本地 HTTP 代理服务，增强 Codex CLI 的灵活性与可扩展性。
- 兼容多种 AI 平台，包括 GPT、Gemini、Claude 等。
- 开源项目，MIT 许可证，社区驱动开发与测试。
