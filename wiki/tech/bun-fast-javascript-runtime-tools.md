# Bun: Fast JavaScript Runtime & Tools

> Bun is an incredibly fast JavaScript runtime, bundler, test runner, and package manager, all in one tool. It is written in Zig and is designed as a drop-in replacement for Node.js, offering faster startup times and reduced memory usage. Last updated: 2026-05-15

## Overview

Bun is an all-in-one toolkit for JavaScript and TypeScript applications, bundled into a single executable called `bun`. The project is open-source and built using Rust and Zig, with support for various platforms including Linux, macOS, and Windows.

At the core of Bun is its JavaScript runtime, which is optimized for speed and efficiency. It is designed to replace Node.js in many use cases, offering significant improvements in performance and resource usage.

The `bun` command-line tool provides a suite of features:
- A fast JavaScript runtime
- A test runner
- A script runner
- A Node.js-compatible package manager

Bun supports TypeScript and JSX out-of-the-box, eliminating the need for multiple development tools. Its built-in tools are highly performant and can be used in existing Node.js projects with minimal changes.

Bun supports Linux (x64 & arm64), macOS (x64 & Apple Silicon), and Windows (x64 & arm64). Installation is straightforward, with several methods available including the install script, npm, Homebrew, and Docker.

## Key Points
- Bun combines a JavaScript runtime, test runner, package manager, and other tools into a single, fast executable.
- It is written in Zig and uses JavaScriptCore for performance optimization.
- It offers faster startup times and lower memory usage than traditional Node.js setups.
- Bun supports TypeScript and JSX natively, with minimal configuration required.
