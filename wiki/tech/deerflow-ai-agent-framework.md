# DeerFlow AI Agent Framework

> DeerFlow is an open-source super agent harness that enables AI agents to research, code, and create using sandboxes, memories, tools, and subagents to handle complex, long-horizon tasks. It has gained widespread attention, hitting #1 on GitHub Trending in February 2026 with a large and active community.  
> Last updated: 2026-04-19

## Overview

DeerFlow (Deep Exploration and Efficient Research Flow) is a next-generation AI agent framework designed to orchestrate sub-agents, memory, and sandboxes to perform complex, long-horizon tasks. It is built to be extensible and flexible, supporting multiple programming environments and AI models.

The framework is developed and maintained by ByteDance and has seen rapid adoption since its launch. DeerFlow 2.0 is a complete rewrite with no shared code from the earlier version, and it supports integration with powerful AI models such as Doubao-Seed-2.0-Code, DeepSeek v3.2, and Kimi 2.5.

DeerFlow includes a suite of tools such as InfoQuest for intelligent search and crawling, and supports various deployment options including Docker and local development environments. The project is open-source and available under the MIT license, with a strong focus on community contributions and real-world applications.

## Key Points

- DeerFlow is a **super agent harness** that allows AI agents to perform complex, multi-step tasks.
- It supports **extensible skills** and integrates with powerful AI models and tools.
- The framework is built in Python and includes **Node.js** components for multi-environment support.
- DeerFlow 2.0 is a **complete rewrite** with no code overlap from version 1.
- The project is open-source and licensed under **MIT**, with over **62,720 stars** and **8,122 forks** on GitHub.
- It includes integration with **InfoQuest**, an intelligent search and crawling toolset developed by BytePlus.
- Deployment options include **Docker** (recommended) and **local development**.
