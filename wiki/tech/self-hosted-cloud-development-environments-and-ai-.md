# Self-Hosted Cloud Development Environments and AI Agents

> This repository, Coder, provides a secure, self-hosted platform for cloud development environments and AI agents. It enables developers and AI agents to work on isolated cloud resources while reducing costs through automatic resource shutdowns and offers centralized governance for AI operations. It's notable for its integration with Terraform, AI agent support, and strong security and compliance features.
> Last updated: 2026-05-10

## Overview

The **coder/coder** repository hosts an open-source platform that enables users to deploy secure and scalable cloud development environments. It supports AI agents for coding tasks and offers centralized governance for infrastructure and AI activities. Workspaces are defined using Terraform, connected via Wireguard tunnels, and automatically shut down when not in use, reducing cloud costs. The system is designed to be self-hosted, granting organizations full control over their infrastructure and data.

## Key Points

- **Self-hosted**: All workspaces and AI agent operations can be deployed on your infrastructure with no reliance on third-party APIs.
- **AI agent support**: Coder allows for AI agents to execute tasks securely with centralized governance, cost tracking, and audit logging.
- **Terraform-based workspaces**: Define cloud environments using Terraform for VMs, Kubernetes, or Docker.
- **Secure connections**: Workspaces are connected via a Wireguard tunnel, ensuring secure access.
- **Auto-shutdown**: Idle resources are automatically shut down to save costs.
- **Rapid onboarding**: Developers can be set up in seconds, not days, using preconfigured templates.
