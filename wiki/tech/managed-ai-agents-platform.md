# Managed AI Agents Platform

> An open-source platform for managing AI agents as teammates, enabling task assignment, autonomous execution, and skill compounding. Notable for its vendor-neutral design and support for multiple AI models.  
> Last updated: 2026-04-10

## Overview
**Multica** is an open-source managed agents platform that integrates coding AI agents into collaborative workflows. It allows teams to assign tasks to AI agents as if they were human colleagues, with autonomous execution, progress tracking, and skill reuse. The platform supports multiple AI models (Claude Code, Codex, OpenClaw, OpenCode) and offers self-hosting, cloud access, and multi-workspace organization. Key features include:

- **Agents as Teammates**: Agents have profiles, participate in boards, and report blockers autonomously.  
- **Autonomous Execution**: Full task lifecycle management with real-time progress streaming.  
- **Reusable Skills**: Solutions become reusable team skills over time.  
- **Unified Runtimes**: Manages local and cloud compute resources with real-time monitoring.  
- **Multi-Workspace**: Isolated workspaces for team-level organization.  

The repository (5.9k stars, 715 forks) includes Docker-based self-hosting, a CLI for local integration, and a cloud interface at [multica.ai](https://multica.ai).  

## Key Points
- Open-source infrastructure for managing AI agents, supporting self-hosting and cloud deployment.  
- Enables AI agents to autonomously execute tasks, report progress, and compound reusable skills.
