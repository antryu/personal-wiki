# Garry Tan's AI-Powered Productivity Stack

> Garry Tan's `gstack` is an AI-powered productivity stack built around Claude Code that acts as a virtual engineering team of 23 opinionated tools. This open-source toolset significantly boosts his productivity, allowing him to ship more than 200 times the output of his 2013 year. The stack includes tools for CEO, engineering manager, designer, and QA roles, all integrated through slash commands and Markdown.
> Last updated: 2026-05-11

## Overview
Garry Tan, CEO of Y Combinator, developed `gstack` to revolutionize how developers build products using AI agents. The stack is built around Claude Code and includes 23 specialized AI tools designed to emulate a full team of engineers, designers, and QA leads. This toolset allows him to produce and ship more features than ever before, with a 240x increase in output compared to 2013.

Key features of `gstack` include:

- A structured environment for managing features and code using a virtual team of AI agents.
- Tools for tasks like product design, code review, QA testing, and security auditing.
- Integration with GitHub, allowing for automation and rapid deployment.
- Open-source under the MIT License, with an active community and 93,833 stars on GitHub.

## Key Points
- **AI-Powered Team Simulation**: gstack acts as a virtual team of 23 specialists, including a CEO, designer, engineering manager, and QA lead.
- **High Productivity Metrics**: Tan has achieved a 240× productivity increase compared to 2013 and a 230× improvement in 2026 alone.
- **Open Source and Free**: The tool is MIT licensed and open to anyone to fork, improve, and use.
- **Easy Setup**: The stack can be installed in under 30 seconds using bun and Git.
- **Real-Time Output Tracking**: The project includes visualizations of GitHub contribution rates to track productivity over time.
