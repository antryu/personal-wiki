# Craft Agents Open Source

> Craft Agents is a powerful tool designed to simplify working with AI agents, allowing users to connect to APIs, manage sessions, and customize workflows in a document-centric, no-code environment. It is built with Agent Native principles and is open source under Apache 2.0, enabling full customization and remixing.  
> Last updated: 2026-04-17

## Overview

Craft Agents is an open-source desktop application that enhances the experience of working with AI agents. It was developed to provide a more intuitive, visual, and non-CLI-based interface for managing tasks, connecting to APIs, and customizing workflows. Built using TypeScript and Electron, it supports multiple AI models and integrates with services like Linear, Slack, Gmail, and even local or custom APIs.

The project is actively maintained and has already gained traction with over 4,270 stars and 633 forks on GitHub. It is used by its creators at Craft.do to build their own tools using the same platform, making it highly adaptable and customizable.

Craft Agents combines the power of the Claude Agent SDK and Pi SDK to provide a seamless, fluid UI and a robust backend for handling complex tasks. It supports instant session management, real-time updates, and a variety of integration methods, including OpenAPI specs, MCP servers, and local subprocesses.

## Key Points

- Simplifies API and service integration without configuration files or setup wizards.
- Fully customizable using natural language prompts and no code editors.
- Supports both cloud-based and local MCP servers, as well as custom APIs.
- Built with Agent Native principles for a more intuitive, document-centric workflow.
- Open source under Apache 2.0, allowing for free remixing and modification.
