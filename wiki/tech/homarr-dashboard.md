# Homarr Dashboard

> A modern dashboard with 40+ integrations, 10K+ built-in icons, and drag-and-drop configuration. Notable for its user-friendly design, robust authentication, and compatibility with diverse hardware and self-hosted apps.  
> Last updated: 2026-04-18

## Overview
Homarr is an open-source dashboard designed for self-hosted application management, offering a highly customizable interface with over 40 integrations (e.g., AdGuard Home, Aria2, Codeberg) and 11K+ icons. It supports drag-and-drop configuration, real-time updates via WebSockets, and secure user management with roles and permissions. The project is actively maintained (3.4k+ stars, 208 forks) and compatible with x86, Raspberry Pi, and major OS platforms.

Key features include:
- **Extensible architecture**: Kubernetes support via Helm for scaling and reliability.
- **Security**: BCrypt and AES-256-CBC encryption, SSO via OIDC/LDAP.
- **Performance**: Real-time widgets, fast search across integrations.
- **Community**: Translations via Crowdin, active Discord community.

## Key Points
- Drag-and-drop UI with no YAML configuration required.  
- 40+ integrations and 11K+ icons for customization.  
- Supports hardware from Raspberry Pi to enterprise-grade systems.  
- Built-in authentication and user management with role-based access.
