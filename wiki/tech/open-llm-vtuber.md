# Open-LLM-VTuber

> Open-LLM-VTuber is a voice-interactive AI companion that allows real-time voice conversations and visual perception with a Live2D avatar, all running locally on your device. It supports a wide range of LLM backends and is designed to provide an engaging and customizable AI VTuber experience.
> Last updated: 2026-05-07

## Overview

Open-LLM-VTuber is a voice-interactive AI companion application that brings together natural language processing, real-time voice input, and visually engaging Live2D avatars into one platform. It enables users to interact with any large language model (LLM), such as Ollama, Llama, or GPT, via voice input, allowing for hands-free operation and dynamic visual feedback through an animated character.

The project is designed to be fully functional on a local machine and supports a wide range of customization and configuration options for both developers and end users. It is actively maintained and has a growing community of contributors.

## Key Points

- Supports real-time voice interaction with any LLM through voice input, including voice interruption for natural conversation flow.
- Features a Live2D avatar that reacts to user input and displays emotions and facial expressions.
- Runs entirely locally, ensuring privacy and offline functionality.
- Open-source and highly customizable, with support for multiple platforms and LLM backends.
- In development for version 2.0, which is a complete codebase rewrite with new features and improvements.
