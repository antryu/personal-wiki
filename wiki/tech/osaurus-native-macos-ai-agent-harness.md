# Osaurus: Native macOS AI Agent Harness

> Osaurus is a native macOS harness for AI agents, allowing users to run any model with persistent memory, autonomous execution, and cryptographic identity. Built in Swift, it operates fully offline and is open-source, making it a powerful tool for personal AI deployment.  
> Last updated: 2026-04-10

## Overview
Osaurus is a macOS-native AI harness designed to provide a personal, secure, and efficient environment for AI agents. It enables users to interact with any model — local or cloud-based — while ensuring all data and context remain on the user's machine. The system supports agents with persistent memory, autonomous execution, and customizable tools, all built using Swift for native performance on Apple Silicon. Osaurus is MIT-licensed and fully open-source, offering a self-hosted alternative to cloud-based AI platforms.

Key features include:
- A **fully offline** experience with support for local models (e.g., via MLX, Ollama, and MCP).
- Compatibility with cloud providers such as OpenAI, Anthropic, and Apple Foundation Models.
- Support for **autonomous agents** that can perform tasks, run code, and evolve over time.
- A **native macOS app** built in Swift, optimized for Apple Silicon with no Electron dependencies.
- A **plugin registry** for extending functionality via community-contributed tools.
- A **centralized agent management system** with customizable prompts, memory, and themes.

Osaurus is positioned as the "harness" for AI — the layer that makes AI personal, rather than relying on third-party services to manage context and execution. It allows users to maintain control over their AI systems, from the models to the tools and memory.

## Key Points
- **Fully offline and local-first** — All data and execution remain on the user's Mac.
- **Native Swift implementation** — Optimized for macOS and Apple Silicon, with no external dependencies.
- **Supports multiple models and APIs** — Works with OpenAI, Anthropic, Apple Foundation Models, Ollama, and more.
- **Autonomous agents with memory** — Agents can learn from interactions, execute code, and evolve over time.
- **Open-source and extensible** — MIT licensed with a plugin registry for community contributions.
