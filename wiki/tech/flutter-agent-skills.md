# Flutter Agent Skills

> The Flutter Agent Skills repository is a collection of skills designed to teach agents best practices and workflows for Flutter app development. These skills help reduce errors and ensure consistent, high-quality development. Last updated: 2026-05-08

## Overview
The **flutter/skills** repository is maintained by the Flutter team and offers a set of domain-specific skills for agents working with Flutter. Each skill is a small folder of files that provides step-by-step instructions for specific tasks, such as adding integration tests, setting up routing, or fixing layout issues. The skills complement toolchains like MCP, giving agents the knowledge to use tools effectively.

Installation is straightforward with the command `npx skills add flutter/skills --skill '*' --agent universal`, which adds all skills to your project. Developers can also update the skills using `npx skills update`. Skills are grouped by task and include example prompts to guide their usage.

## Key Points
- Offers structured workflows for common Flutter development tasks.
- Complements the use of toolchains like MCP.
- Skills are modular, making them easy to integrate and update.
- Helps enforce best practices and reduce common mistakes during app development.
- Maintained by the official Flutter team for reliability and compatibility.
