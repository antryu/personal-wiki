# WSABuilds: Windows Subsystem for Android with Root and GApps

> Run Windows Subsystem for Android (WSA) on Windows 10 and 11 with prebuilt binaries that include Google Play Store (GApps) and/or root solutions like Magisk or KernelSU. This project is notable for simplifying the use of Android apps and services on Windows, with support for customizations and troubleshooting.
> Last updated: 2026-05-13

## Overview
WSABuilds is a popular open-source project that provides prebuilt versions of the Windows Subsystem for Android (WSA) with various enhancements, including Google Play Store (GApps) and root access via tools like Magisk and KernelSU. These builds are designed for Windows 10 and Windows 11 users who want to run Android applications and services seamlessly on their PCs.

The project's GitHub repository contains extensive documentation, troubleshooting guides, and community support links to help users navigate potential issues. It also supports different versions of WSA, including older builds that may work better for specific configurations.

WSABuilds is maintained under the GNU Affero General Public License v3.0 and has a large community of contributors and users who actively report bugs and request features. The project is hosted on GitHub and includes integration with XDA Developers and other community resources.

## Key Points
- Provides prebuilt WSA images with GApps (Google Play Store) and root access via Magisk or KernelSU.
- Maintains compatibility with both Windows 10 and Windows 11.
- Includes documentation, bug tracking, and community support.
- Offers various WSA versions, including older builds that may be more stable.
- Hosted on GitHub under an open-source license.
