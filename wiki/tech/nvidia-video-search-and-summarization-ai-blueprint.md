# NVIDIA Video Search and Summarization AI Blueprint

> This repository provides reference architectures for building vision agents and AI-powered video analytics applications, combining accelerated vision microservices, Vision Language Models (VLMs), and Large Language Models (LLMs). It enables natural language interaction with video data for real-time analytics, alert verification, and long video summarization.
> Last updated: 2026-05-14

## Overview

The [NVIDIA AI Blueprint for Video Search and Summarization (VSS)](https://docs.nvidia.com/vss/latest/index.html) offers a suite of reference architectures for developing vision agents and AI-driven video analytics applications. These architectures integrate accelerated vision microservices, Vision Language Models (VLMs), and Large Language Models (LLMs), enabling users to enhance existing applications, deploy standalone microservices, or integrate components into larger vision agents.

The blueprint is structured around three core processing areas:
- **Real-time video intelligence**: Includes feature extraction, embeddings, and stream understanding with results published via message brokers.
- **Downstream analytics**: Enriches metadata into trajectories, incidents, and verified alerts.
- **Agentic and offline processing**: Orchestrates tools for search, Q&A, summarization, and clip retrieval, including via the Model Context Protocol.

This repository powers the [NVIDIA build experience](https://build.nvidia.com/nvidia/video-search-and-summarization), providing a foundation for natural-language video agents that leverage generative AI, VLMs, LLMs, and NVIDIA NIM microservices.

## Key Points
- Combines accelerated vision microservices, VLMs, and LLMs for video analytics.
- Provides reference agent workflows for Q&A, alert verification, real-time alerts, video search, and long video summarization.
- Targets use cases like smart space monitoring, warehouse automation, and SOP validation.
- Leverages NVIDIA NIM microservices and supports GPU acceleration for performance.
