# XcodeBuild MCP Server and CLI

> A Model Context Protocol (MCP) server and CLI that provides tools for agent use when working on iOS and macOS projects. It supports AI coding agents by offering a structured interface for Xcode build processes and development workflows.
> Last updated: 2026-05-16

## Overview
XcodeBuildMCP is a tool that provides a Model Context Protocol (MCP) server and command-line interface (CLI) tailored for iOS and macOS development. It is particularly useful for AI coding agents, offering a streamlined way to interact with Xcode and manage project builds, tools, and configurations. Developed in TypeScript, it supports macOS 14.5 or later and Xcode 16.x or later, and it is available under the MIT License.

The project is hosted on GitHub and has gained traction with over 5,600 stars and 273 forks since its creation in March 2025. It is actively maintained, with the last update occurring on May 2026. The tool is designed to enhance the productivity of developers and AI agents working within the Xcode ecosystem by simplifying build and project management tasks.

XcodeBuildMCP can be installed via Homebrew or npm (Node.js 18.x or later). It includes optional "skills" for AI agents, such as an MCP Skill for interacting with the server and a CLI Skill for terminal-based workflows. These skills help tailor the agent's behavior and improve its ability to handle Xcode-related tasks.

## Key Points
- Provides a Model Context Protocol (MCP) server and CLI for iOS and macOS development.
- Designed for integration with AI coding agents to streamline Xcode workflows.
- Supports macOS 14.5 and Xcode 16.x or later.
- Ships with optional agent skills to enhance AI interaction with the tool.
- Available via Homebrew or npm and is licensed under the MIT License.
