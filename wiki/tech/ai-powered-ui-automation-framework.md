# AI-Powered UI Automation Framework

> Midscene.js is an AI-powered, vision-driven UI automation framework for cross-platform testing and interaction. It enables natural language scripting and integrates with web, mobile, and custom interfaces using advanced computer vision and AI models.  
> Last updated: 2026-04-17

## Overview
**Midscene.js** (https://github.com/web-infra-dev/midscene) is a TypeScript-based framework for automating user interfaces across web, iOS, Android, and custom platforms. With 12,695 stars and 945 forks, it leverages AI models like UI-TARS-1.5-7B to enable natural language-driven automation, reducing the need for manual coding. The project supports integration with Puppeteer, Playwright, WebDriverAgent, and ADB for device control.

Key features include:
- **Natural language automation**: Write scripts using plain English or YAML.
- **Cross-platform support**: Web (Puppeteer/Playwright), Android (ADB), iOS (WebDriverAgent), and custom interfaces.
- **Computer vision**: Uses AI for visual recognition and interaction.
- **Developer-friendly APIs**: Interaction, Data Extraction, and Vision APIs for customization.

Showcases demonstrate real-world applications like GitHub form automation, iOS coffee ordering, Android hotel bookings, and robotic arm integration. The project maintains an active community via Discord (1.3M members) and Twitter.

## Key Points
- Cross-platform UI automation for web, iOS, Android, and custom interfaces.
- AI-powered natural language scripting with computer vision capabilities.
- MIT License and open-source ecosystem with 104 open issues.
- Integrates with Puppeteer, Playwright, WebDriverAgent, and ADB.
- Community-driven extensions via Midscene Skills and OpenClaw.
