# Self-Hosted AI Coding Assistant

> Tabby is a self-hosted, open-source AI coding assistant that serves as an on-premises alternative to GitHub Copilot. It enables developers to deploy a lightweight, GPU-compatible solution with OpenAPI integration for custom workflows.  
> Last updated: 2026-04-10

## Overview
**Tabby** (formerly TabbyML) is an open-source AI coding assistant designed for self-hosting, offering features like code completion, documentation integration, and GitLab merge request indexing. It requires no cloud services or database management systems, making it ideal for private infrastructure. Recent updates include:

- **2026-04-10**: v0.30.0 adds GitLab Merge Request context indexing.
- **2025-12-12**: Integration with Pochi for GitHub issue-to-PR automation.
- **2025-05-20**: v0.29.0 introduces REST APIs for custom documentation.
- **2025-03-31**: Enhanced `@` menu in the chat side panel for v0.27.0.

The project supports consumer-grade GPUs and provides Docker deployment options. A live demo is available at [tabby.tabbyml.com](https://tabby.tabbyml.com).

## Key Points
- Self-contained deployment with no DBMS or cloud dependencies.
- OpenAPI-compatible for integration with Cloud IDEs and custom workflows.
- Supports consumer GPUs (e.g., NVIDIA RTX series) for cost-effective inference.
- Active development with recent features like GitLab MR context indexing and persistent chat pages.
- High community engagement: 33,373 stars and 1,730 forks on GitHub.
