# Swift Terminal Emulator Library

> SwiftTerm is a VT100/Xterm terminal emulator library for Swift applications, designed for embedding in macOS, iOS, and other platforms. It's notable for its comprehensive terminal emulation and support for advanced features like Unicode rendering and GPU-accelerated rendering.
> Last updated: 2026-05-14

## Overview

SwiftTerm is a terminal emulator library written in Swift that allows developers to integrate a high-quality terminal emulator into their macOS, iOS, or headless applications. It is used in commercial SSH clients such as Secure Shellfish, La Terminal, and CodeEdit. The library provides both a UI-agnostic terminal engine and UI-specific front-ends for macOS (AppKit) and iOS (UIKit). It is also used in other related projects such as TermKit and Pane.

The project is actively maintained and has a strong set of features including support for Unicode, terminal multiplexing, and GPU acceleration. It is based on the original XtermSharp library and has been enhanced to surpass xterm.js and XtermSharp in many areas.

## Key Points

- Supports comprehensive VT100/Xterm terminal emulation
- Unicode rendering including emoji and combining characters
- Reusable and pluggable terminal engine with macOS and iOS UI front-ends
- Terminal session recording and playback
- GPU-accelerated rendering via Metal
