# Memory Shortage

> 메모리 부족(Memory Shortage)은 하드웨어 또는 소프트웨어에서 처리 가능한 데이터 용량 한계를 초과할 때 발생하는 문제입니다. 이는 시스템 성능 저하, 프로그램 충돌, 데이터 손실 등의 문제를 유발할 수 있습니다.  
> Last updated: 2026-04-12

## 개요
메모리 부족은 컴퓨터 시스템에서 RAM(임시 메모리) 또는 저장 공간(영구 메모리)이 요구되는 작업량을 감당하지 못할 때 발생합니다. 이 문제는 하드웨어 사양의 한계, 과도한 프로그램 실행, 불필요한 데이터 누적, 소프트웨어 최적화 부족 등 다양한 원인에 의해 발생할 수 있습니다. 메모리 부족은 시스템이 느려지거나 프로그램이 비정상적으로 종료되는 등의 영향을 미치며, 심각한 경우 데이터 손실이나 시스템 다운이 발생할 수 있습니다.

## 주요 포인트
- 하드웨어 사양(예: RAM 용량)이 작업 요구 사항을 충족하지 못함  
- 과도한 프로세스 실행 또는 메모리 누수(Memory Leak) 발생  
- 저장 공간 부족으로 인한 디스크 I/O 성능 저하
