# Oracle AI Developer Hub

> The Oracle AI Developer Hub is a technical resource repository designed to help AI developers build applications, agents, and systems using Oracle AI Database and OCI services. It provides complete working examples and interactive tutorials, making it a valuable tool for developers in the AI space.
> Last updated: 2026-05-09

## Overview

The Oracle AI Developer Hub provides a comprehensive set of tools and resources for AI developers and engineers aiming to build AI applications, agents, and systems using Oracle technologies. The repository is organized into key sections: **Apps**, **Notebooks**, and more.

The **Apps** section includes various AI-powered applications and reference implementations, each with source code, documentation, and deployment configurations. These examples demonstrate how to build end-to-end AI solutions using Oracle AI Database and OCI services.

The **Notebooks** section contains Jupyter notebooks and tutorials covering AI/ML model development, Oracle Database AI features, OCI AI service integrations, and agent development. These resources are invaluable for learning and experimentation.

## Key Points
- Offers working AI application examples using Oracle AI Database and OCI.
- Provides interactive Jupyter notebooks for AI/ML development and experimentation.
- Includes deployment configurations and documentation for production-grade AI systems.
- Maintains a strong focus on integrations, best practices, and architectural patterns.
