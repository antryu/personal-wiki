# AI-Powered Data Visualization Tool

> The Data Formulator is an AI-powered data visualization tool that enables users to explore and visualize data through natural language and AI agents. It offers a conversational interface, advanced visualization capabilities, and integration with various data sources, making it a powerful tool for data analysts and developers.
> Last updated: 2026-05-13

## Overview
The **Data Formulator** is an open-source project developed by Microsoft, which leverages AI to simplify and enhance data visualization. It supports a wide range of data sources and offers an interactive, conversational agent to help users explore and explain data. The tool is built with TypeScript and released under the MIT license. Recent updates have introduced features such as persistent sessions, a conversational AI agent with memory, and advanced charting capabilities.

## Key Points
- **AI-powered visualization**: Uses AI to create and refine charts and data insights.
- **Conversational agent**: A unified agent that supports fluid conversations for data exploration.
- **Multiple data connectors**: Connects to various data sources like Superset, Kusto, Cosmos DB, and more.
- **Persistent sessions & workspace management**: Saves sessions and workspaces for future access.
- **30+ chart types**: Supports a wide range of visualizations, including area, streamgraph, and radar charts.
- **Community contributions**: Actively seeks developer input for new agents and data loaders.
