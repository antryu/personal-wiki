# Kopia: Cross-Platform Backup Tool

> Kopia is a fast and secure open-source backup/restore tool that creates encrypted snapshots of your data and supports storage on cloud, network, or local drives. It offers both CLI and GUI interfaces and is notable for its robust features like end-to-end encryption, compression, deduplication, and error correction.
> Last updated: 2026-05-04

## Overview

Kopia is a versatile cross-platform backup tool designed for Windows, macOS, and Linux. It enables users to create fast, incremental backups of their data using a combination of client-side end-to-end encryption, compression, and data deduplication. Kopia provides both command-line and graphical user interfaces, making it accessible to a wide range of users—from developers to everyday individuals.

The project is maintained on GitHub and is written in Go, licensed under the Apache License 2.0. As of its latest update on May 4, 2026, it has accumulated 13,146 stars and 642 forks, with 788 open issues actively being addressed.

Kopia supports storage of encrypted snapshots to multiple cloud and local storage providers, including Amazon S3, Google Cloud Storage, Azure Blob Storage, Backblaze B2, WebDAV, SFTP, and Rclone-compatible services. These snapshots can be stored remotely, locally, or on network-attached storage, depending on user preference.

The project emphasizes user control, security, and performance, with key features such as error correction and support for both individual and enterprise-level use.

## Key Points
- Fast, incremental backups with client-side encryption and data deduplication
- Supports multiple cloud and local storage options, including Amazon S3, Google Cloud, and WebDAV
- Provides both CLI and GUI for ease of use across all user skill levels
- Open source and actively maintained with a large community presence
