# Payload: Open-source Fullstack Next.js Framework

> Payload is the open-source, fullstack Next.js framework, giving you instant backend superpowers. Get a full TypeScript backend and admin panel instantly. Use Payload as a headless CMS or for building powerful applications.  
> Last updated: 2026-05-06

## Overview

Payload is the first Next.js native CMS that can install directly in your existing `/app` folder, marking the beginning of a new era for headless CMS. It combines the advantages of a modern, open-source framework with a powerful admin panel and full TypeScript support. Payload is designed to be flexible and extensible, enabling developers to build both content-driven websites and custom fullstack applications.

### Benefits over a regular CMS
- It's both an app framework and headless CMS
- Deploy anywhere, including serverless on Vercel for free
- Combine your front+backend in the same `/app` folder if you want
- Don't sign up for yet another SaaS — Payload is open source
- Query your database in React Server Components
- Both admin and backend are 100% extensible
- No vendor lock-in
- Never touch ancient WP code again
- Build faster, never hit a roadblock

## Key Points
- Payload is open-source and built using TypeScript.
- It enables developers to create a full backend and admin panel instantly.
- It supports both MongoDB and PostgreSQL as databases.
- It is deployable on serverless platforms like Vercel and Cloudflare.
- The framework supports React Server Components and is designed for modern fullstack development.
