# Burn: Next-Gen Deep Learning Framework in Rust

> Burn is a next generation tensor library and Deep Learning Framework that doesn't compromise on flexibility, efficiency and portability. It leverages Rust for optimized numerical computing, model inference, and training while supporting multiple backends for cross-platform performance.
> Last updated: 2026-05-09

## Overview
Burn is a high-performance deep learning framework built in Rust, designed to be both fast and flexible. It offers a modern tensor library optimized for numerical computing and supports multiple hardware backends, including GPU and CPU options such as CUDA, ROCm, Metal, WebGPU, and more. Burn's modular architecture allows backends to be swapped or composed, with features like automatic differentiation (autodiff) and kernel fusion available across platforms. This makes it ideal for training models in the cloud and deploying them on diverse hardware environments.

Burn provides a clean, Rust-based API, which supports all major operating systems and even WebAssembly (Wasm) for portable deployment. It is particularly notable for combining the performance of static-graph frameworks with the flexibility of dynamic ones.

The framework is actively maintained and has a strong developer community, with a license under Apache 2.0.

## Key Points
- Burn is a Rust-based deep learning framework and tensor library that prioritizes performance and portability.
- It supports a wide range of hardware backends, including GPU options like CUDA, ROCm, Metal, WebGPU, and CPU options like CubeCL and Flex.
- Burn uses modular design to enable backend composition and swapping, with features like autodiff and kernel fusion.
- It is cross-platform and supports major operating systems, including WebAssembly for browser deployment.
- Burn is actively developed and has a growing community with 15k+ stars and an open-source Apache 2.0 license.
