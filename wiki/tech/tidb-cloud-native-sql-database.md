# TiDB Cloud-Native SQL Database

> TiDB is an open-source, cloud-native, distributed SQL database designed for modern applications, offering horizontal scalability and compatibility with MySQL. It is notable for its ability to handle large-scale data workloads and its integration with cloud environments.  
> Last updated: 2026-04-10

## Overview
- **Repository**: [pingcap/tidb](https://github.com/pingcap/tidb)  
- **Description**: A distributed SQL database optimized for cloud-native environments, combining the familiarity of MySQL with distributed scalability.  
- **Stars**: 39,944 (with **+20 stars** today)  
- **Language**: Primarily written in [Go](https://golang.org/).  
- **Key Features**:  
  - Distributed architecture for horizontal scaling.  
  - Cloud-native design for Kubernetes and containerized deployments.  
  - Strong consistency and ACID compliance.  

## Key Points
- Open-source and actively maintained by PingCAP, with a large community and enterprise support options.  
- Designed to handle both online transactional processing (OLTP) and analytical workloads (HTAP).
