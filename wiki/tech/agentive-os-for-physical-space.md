# Agentive OS for Physical Space

> Dimensional is the agentic operating system for physical space, enabling natural language control of humanoids, quadrupeds, drones, and other hardware platforms. It builds multi-agent systems that integrate seamlessly with physical inputs like cameras, lidar, and actuators.  
> Last updated: 2026-04-16

## Overview
Dimensional OS (dimos) is a pre-release beta platform designed as a next-generation SDK for generalist robotics. It eliminates the need for ROS (Robot Operating System) by providing a Python-centric framework for building physical applications that run across diverse hardware platforms. Key features include agent-native architecture, spatial memory systems, and integration with navigation, perception, and control capabilities. The system supports both local and hosted multi-agent systems, enabling tasks like natural language navigation ("hey Robot, go find the kitchen") and spatio-temporal reasoning.

**Repository Highlights**:
- **Platform**: Supports humanoids, quadrupeds, drones, and custom hardware.
- **Architecture**: Agent-native modules, spatial memory, and perception pipelines.
- **Development**: Built with Python, Nix, and CUDA support, with Docker compatibility.
- **Community**: 2,903 stars, 465 forks, and 266 open issues on GitHub.

## Key Points
- **Agentive Control**: Use natural language ("vibecoding") to direct robots and manage multi-agent systems.
- **Cross-Platform Hardware Integration**: Works with lidar, cameras, actuators, and major robot manufacturers.
- **Navigation & Mapping**: Includes SLAM, dynamic obstacle avoidance, and autonomous exploration.
- **Perception Capabilities**: 3D projections, VLMs (Vision-Language Models), and audio processing.
- **Spatial Memory**: Spatio-temporal RAG (Retrieval-Augmented Generation), object permanence tracking, and dynamic memory.
