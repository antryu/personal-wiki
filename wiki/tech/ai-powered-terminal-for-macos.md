# AI-Powered Terminal for macOS

> A Ghostty-based macOS terminal with vertical tabs and notifications for AI coding agents. Integrates AI tools like Claude Code Teams and offers features such as notification rings, in-app browser, and SSH support. Last updated: 2026-04-15

**Description**:  
A macOS terminal designed for AI-powered coding workflows, featuring vertical/horizontal tabs, real-time notifications for AI agents, and integrations with tools like Claude Code and Gemini. Includes a built-in browser, SSH workspace creation, and multilingual support (20+ languages).  

**Key Features**:  
- Notification rings for AI agent alerts  
- Vertical tabs with Git branch, PR status, and network metadata  
- Scriptable in-app browser with cookie/session import  
- Native SSH workspaces with drag-and-drop file transfers  
- Claude Code Teams integration via native splits  
- 20+ browser profile import support
