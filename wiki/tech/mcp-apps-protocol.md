# MCP Apps Protocol

> The MCP Apps Protocol provides a standardized framework for embedding interactive UIs within AI chatbots, such as Claude and ChatGPT. It enables developers to build and integrate tools like dashboards and forms directly into chat interfaces, making it easier to create rich, interactive experiences for users.
> Last updated: 2026-05-07

## Overview

This repository contains the specification and SDK for the MCP Apps protocol. The protocol enables developers to create apps and embed them inline in AI chat clients, using a standardized method that works across platforms compliant with the ModelContextProtocol (MCP). It also includes tools and documentation to help developers scaffold, migrate, and integrate apps into existing AI systems. The project includes a set of Agent Skills for AI coding agents, making it easier to create, modify, and deploy apps quickly.

Key features include:
- A TypeScript-based SDK for building interactive UIs.
- Integration with existing AI chat clients.
- Tools for converting and embedding apps into web or existing AI services.
- Support for documentation and API references.

The repository has seen significant development, with active GitHub contributions, a growing number of stars and forks, and an Apache 2.0 license for open source collaboration.

## Key Points

- Enables the creation of interactive UIs for AI chat clients.
- Provides a standardized way to embed dashboards, forms, and other tools.
- Includes Agent Skills for AI coding agents to simplify development and migration.
