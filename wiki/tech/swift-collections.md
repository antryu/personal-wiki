# Swift Collections

> Swift Collections is an open-source package providing commonly used data structures for the Swift programming language. It is notable for offering advanced and customizable implementations of collections, including deques, ordered dictionaries, and heaps, enhancing Swift's standard library capabilities.
> Last updated: 2026-05-13

## Overview

The **Swift Collections** repository is a Swift package developed and maintained by Apple, offering a set of commonly used data structure implementations. It includes modules like `BasicContainers`, `DequeModule`, and `OrderedCollections`, each offering specialized collection types such as deques, fixed-capacity arrays, and ordered dictionaries. The package is designed with a focus on performance, memory safety, and flexibility, allowing developers to work with advanced collection types not provided by the Swift standard library.

The repository is actively maintained, with the last update occurring on May 13, 2026. It is licensed under the Apache License 2.0 and is available on GitHub. The project also includes experimental features, such as unstable containers and sorted collections, for developers who want to contribute to future enhancements.

## Key Points

- Provides advanced data structures such as deques, ordered dictionaries, and heaps.
- Offers modules with specialized collections not found in Swift's standard library.
- Implements performance-optimized and memory-safe collection types.
- Includes both stable and experimental features for diverse use cases.
- Maintained by Apple with an active community on GitHub.
