# VHS Terminal Recorder

> VHS is a tool for recording terminal sessions as GIFs or videos, allowing users to create demos and integration tests for CLI tools. It's notable for its ease of use and flexibility in generating visual content directly from code.
> Last updated: 2026-04-16

## Overview
The VHS Terminal Recorder, developed under the `charmbracelet/vhs` GitHub repository, allows users to create terminal GIFs and videos programmatically. This makes it ideal for showcasing CLI tools, testing, and demonstrations. It is written in Go and is distributed under the MIT License. The project has gained significant traction, with over 19,388 stars and 405 forks as of 2026. VHS supports a variety of installation methods, including package managers, Docker, and direct downloads.

To use VHS, users write `.tape` files containing a sequence of commands that simulate terminal interactions. These commands are then executed by VHS to generate GIFs or videos. The tool is particularly useful for developers who want to showcase their CLI applications in a visual format.

## Key Points
- VHS allows users to create terminal GIFs and videos from code.
- It's ideal for integration testing and showcasing CLI tools.
- The project is actively maintained and has a large community.
- VHS supports multiple installation methods, including package managers and Docker.
