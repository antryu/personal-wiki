# CopilotKit: Frontend Stack for Agents & Generative UI

> CopilotKit is a frontend stack designed for building agent-native applications, offering tools for generative UI, shared state, and human-in-the-loop workflows. It is notable for its role in the AG-UI Protocol, adopted by major tech companies.  
> Last updated: 2026-05-08

## Overview

CopilotKit is a best-in-class SDK for developing full-stack agentic applications, Generative UI, and chat apps. It allows developers to integrate AI capabilities into their applications quickly and efficiently. The platform is backed by the **AG-UI Protocol**, adopted by Google, LangChain, AWS, Microsoft, and others, and is built with TypeScript and supports frameworks like React and Angular. 

The SDK includes features like a React-based chat UI, Backend Tool Rendering, Generative UI, Shared State, and Human-in-the-Loop interactions. Developers can begin integrating AI into their applications in as little as one minute.

### Key Features

- **Chat UI**: A React-based chat interface that supports message streaming, tool calls, and agent responses.
- **Backend Tool Rendering**: Enables agents to call backend tools and return UI components directly to the client.
- **Generative UI**: Dynamically generate and update UI components at runtime based on user intent.
- **Shared State**: Real-time synchronized state for both agents and UI components.
- **Human-in-the-Loop**: Allows agents to pause execution and request user input before continuing.

### Quick Start

For new projects:

```bash
npx copilotkit@latest create -f <framework>
```

For existing projects:

```bash
npx copilotkit@latest init
```

This command sets up CopilotKit in your app, configures the provider, connects agents and UI components, and prepares the app for deployment.
