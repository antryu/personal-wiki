# Agent Skills for AI Coding Agents

> This repository provides a secure and validated skill registry for professional AI coding agents, offering verified and safe capabilities for tools like Antigravity, Claude Code, Cursor, and more. It addresses the vulnerability issues found in many open-source skills and ensures that each skill is thoroughly tested and trusted before publication.  
> Last updated: 2026-05-16

## Overview

The `agent-skills` repository is a secure, managed skill registry designed to extend the functionality of AI-powered coding agents with verified and tested capabilities. With growing concerns around the safety and trustworthiness of third-party AI agent skills, this project ensures that all skills are open source, hardened, and rigorously scanned for vulnerabilities.

The repository includes a curated collection of skills, each organized in a structured format with detailed documentation, templates, and references. These skills act as plugins for AI agents, enhancing their ability to perform specialized tasks, follow new workflows, and access tailored knowledge.

Key features of the project include:

- A curated, hardened skill catalog to avoid common vulnerabilities found in open skill marketplaces.
- Integration with multiple AI agents such as Antigravity, Claude Code, and Cursor.
- A secure CLI tool with built-in protections like sandboxing, lockfiles, and audit trails.
- A public-facing documentation site and contribution guide for developers.
- Full transparency with all code and processes being open source.

## Key Points

- **Secure and Verified Skills**: Skills are validated and scanned for vulnerabilities before being published, ensuring a safe experience for developers.
- **Supported AI Agents**: Compatible with popular AI coding tools like Antigravity, Claude Code, and Cursor.
- **Modular Skill Structure**: Skills are organized by category and contain instruction files, templates, and references.
- **Active Community & Maintenance**: The project has 2,545 stars and is regularly updated, with 5 open issues reported on GitHub.
- **Open Source and Transparent**: All code is open source, with clear documentation and contribution guidelines.
