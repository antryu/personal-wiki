# AndroidMic: Use Android as PC Microphone

> AndroidMic allows you to use your Android phone as a high-quality microphone for your PC, supporting cross-platform operation and multiple connection methods. It is notable for its flexibility and audio processing features.  
> Last updated: 2026-05-08

## Overview

AndroidMic is an open-source project that enables users to use their Android smartphones as a microphone input for their PC. The application supports Windows, macOS, and Linux platforms and offers various connection options including TCP/UDP over WiFi, USB Serial, and USB ADB. The audio is streamed to a virtual audio cable, allowing the PC to treat it as a microphone input. Features include noise cancellation, audio wave visualization, and customizable audio settings.

The project is written in Rust and uses the GNU General Public License v3.0. It has 1,192 stars and 73 forks on GitHub, with active development and 53 open issues for ongoing improvements.

To use AndroidMic, a virtual audio cable device is required, with specific setup instructions provided for Windows, Linux, and macOS. Once set up, users can select the virtual audio device and connection method to start streaming audio from their Android phone to their PC.

## Key Points
- Cross-platform support for Windows, Linux, and macOS
- Supports multiple connection methods (WiFi, USB ADB, USB Serial)
- Audio processing features like noise cancellation and waveform visualization
- Requires a virtual audio cable for proper microphone input recognition
- Open-source and actively maintained
