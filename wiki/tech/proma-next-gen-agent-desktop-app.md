# Proma: Next-Gen Agent Desktop App

> Proma is a next-generation AI desktop application designed to bring a seamless agent experience into users' workflows. It enables proactive agent capabilities, offers native support for Lark group chat integration, and allows flexible access to any major AI model provider, making it a powerful, all-in-one agent platform for 100x productivity.  
> Last updated: 2026-05-13

## Overview
Proma is a locally-first AI desktop app that combines multi-model chat, general agent capabilities, workspaces, skills, MCP, remote bots, and memory in a single open-source client. It is not just a chat interface for casual use but a long-term platform for personal workflow automation. In chat mode, it handles straightforward tasks, while agent mode is for complex workflows involving file manipulation, multi-step task execution, and external context integration.

The application is built with TypeScript and is open source. It supports various AI models and provides both open-source and commercial versions for users, with commercial editions offering additional features and convenience.

## Key Points
- **Chat mode**: Multi-model chat, file parsing, image input, and markdown support.
- **Agent mode**: General agent capabilities using the `@anthropic-ai/claude-agent-sdk` for task execution and planning.
- **Remote bots and integrations**: Native Lark bot support and bridge interfaces for DingTalk and WeChat.
- **Local-first design**: Stores data in local JSON files, prioritizing user privacy and control.
- **Cross-platform support**: Available for macOS and Windows.
