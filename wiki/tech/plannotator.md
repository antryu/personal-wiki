# Plannotator

> Plannotator is a visual tool for annotating and reviewing AI coding agent plans and code diffs, enabling team collaboration and direct feedback to agents. It integrates with major AI coding platforms and offers secure, end-to-end encrypted sharing.  
> Last updated: 2026-04-15

### Key Features
- **Visual Plan Review**: Inline annotations for approving/denying agent plans  
- **Plan Diff**: Automatic visualization of revised plan changes  
- **Code Review**: Analyze git diffs/PRs with AI-powered annotations  
- **Annotation Tools**:  
  - `/plannotator-annotate` for markdown files  
  - `/plannotator-last` for structured feedback on agent responses  

### Integration Support
Works natively with **Claude Code**, **Copilot CLI**, **Gemini CLI**, **OpenCode**, **Pi**, and **Codex**. Installation guides provided for each platform.

### Security & Collaboration
- **Zero-knowledge sharing**: Large plans encrypted with AES-256-GCM before upload  
- **Self-hostable**: Open-source design mirrors [PrivateBin](https://privatebin.info/) security model  
- **Auto-delete**: Shared pastes expire after 7 days  

### Installation Highlights
- One-click plugin installation for Claude Code  
- Cross-platform CLI installers for macOS/Linux/Windows  
- Version pinning and [SLSA provenance verification](https://slsa.dev/) support  

## Key Points
- Visual UI for collaborative plan/code reviews with AI agents  
- End-to-end encrypted sharing with auto-expiration for sensitive plans  
- Native integrations with 6+ major AI coding platforms  
- Open-source and self-hostable for enterprise privacy
