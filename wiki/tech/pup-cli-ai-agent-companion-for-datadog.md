# Pup CLI: AI Agent Companion for Datadog

> Pup CLI is a powerful command-line interface designed for AI agents to interact with Datadog's observability platform, offering 200+ commands across 33+ products. It's notable for its structured output, self-discoverable commands, and secure access control.
> Last updated: 2026-05-07

## Overview

Pup CLI is a CLI companion built to allow AI agents to interact with Datadog's wide range of products, including monitoring, logs, metrics, RUM (Real User Monitoring), security, and more. It aims to provide a seamless, AI-friendly interface to access and manage the full breadth of the Datadog platform.

Built in Rust and licensed under the Apache License 2.0, Pup leverages Datadog's APIs to expose a structured and self-discoverable CLI with over 200 commands across more than 33 products. It is currently in preview mode, with the developers actively fine-tuning and improving its functionality.

Pup supports both humans and AI agents, offering clean JSON/YAML outputs, OAuth2 + PKCE authentication, and a well-documented command structure. It also includes a wide array of commands for metrics, logs, events, RUM, APM, traces, profiling, and database monitoring.

## Key Points
- **AI-Ready**: Designed for AI agents to access and manage Datadog observability tools.
- **Comprehensive**: Over 200 commands covering 33+ Datadog products.
- **Security-Focused**: Uses OAuth2 and PKCE for secure, scoped access.
- **Human-Friendly**: Usable by developers for direct CLI interaction.
