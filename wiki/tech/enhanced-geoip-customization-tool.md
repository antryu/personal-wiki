# Enhanced GeoIP Customization Tool

> The Enhanced GeoIP Customization Tool is a powerful tool for generating and customizing GeoIP files in multiple formats, such as V2Ray’s `geoip.dat`, MaxMind `mmdb`, sing-box `SRS`, mihomo `MRS`, Clash, and Surge rulesets. It is notable for its flexibility, allowing users to tailor GeoIP data for use in various proxy and routing applications.
> Last updated: 2026-05-14

## Overview
This tool is designed for users who need to create or modify GeoIP data for use with a wide range of applications including V2Ray, Xray-core, sing-box, Clash, mihomo, Shadowrocket, Quantumult X, Surge, hysteria, Trojan-Go, Leaf, and Nginx.

The repository generates standard GeoIP files weekly and provides a command-line interface (CLI) for users to create custom GeoIP files based on specific needs. It uses MaxMind’s GeoLite2 Country CSV data by default but enhances it with additional IP ranges and categories, such as those for Chinese operators, Cloudflare, Google, and more.

Users can download pre-built files from the GitHub repository, and the tool also supports multiple CDN fallbacks to ensure accessibility.

## Key Points
- Generates weekly GeoIP files in multiple formats for use with various proxy and routing software.
- Offers a CLI for custom GeoIP file generation.
- Enhances standard MaxMind GeoIP data with additional IP ranges and categories (e.g., Chinese operators, Cloudflare, Google).
- Provides multiple download options and fallback domains for reliability.
