# Waza - AI Agent Skills Evaluation Framework

> Waza is a CLI and framework for evaluating AI agent skills, enabling users to create, test, measure, and improve the quality and effectiveness of skills. It provides tools for benchmarking, comparing models, and measuring skill readiness.  
> Last updated: 2026-05-10

## Overview

Waza is a Go-based CLI framework developed by Microsoft to evaluate and manage AI agent skills. The framework supports the creation of evaluation suites, benchmarking, and comparing results across different AI models. Waza includes tools for scaffolding evaluations, running benchmarks, measuring performance, and suggesting improvements based on skill definitions and test data. The project is open-source under the MIT License and features support for both custom agent workflows and embedded model evaluation.

Key features include:
- **Skill Evaluation**: Create, test, and measure AI agent skills.
- **Benchmarking**: Run benchmarks to compare the performance of different models.
- **Custom Agent Support**: Enable custom agent workflows via `.agent.md` files.
- **Token Management**: Count and compare token usage across skill files.
- **Integration Options**: Install via binary, source, or as an extension for the Azure Developer CLI (azd).
- **Automatic Update Checks**: Built-in support to notify users of new versions without affecting performance.

Waza is built in Go and includes support for multiple operating systems (Linux, Darwin, Windows) and architectures (AMD64, ARM64). The CLI offers a streamlined installation process and comes with documentation for quick onboarding.

## Key Points

- Waza enables the creation and evaluation of AI agent skills through a comprehensive command-line interface.
- It supports benchmarking and comparing AI model performance using customizable evaluation suites.
- The framework is flexible, allowing for both pre-built skill scaffolding and custom agent workflows.
- Token usage and budgeting can be tracked and compared across different skill implementations.
- Waza integrates with the Azure Developer CLI (azd) for extended development and deployment workflows.
