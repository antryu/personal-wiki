# Matcha: Terminal Email Client

> Matcha is a beautiful and functional email client for the terminal, built with Go and the Bubble Tea TUI library. It allows users to manage their email directly from the command line, offering support for multiple accounts, rich content, and advanced terminal features, all while maintaining a modern and intuitive interface.
> Last updated: 2026-05-04

## Overview
Matcha is a terminal-based email client developed using Go and the Bubble Tea TUI library. It provides a modern and visually appealing interface for email management in the command line, making it easy for developers and power users to stay productive without leaving their terminal.

### Features

- **Modern UI**: Built with the Bubble Tea TUI framework for a clean and responsive experience.
- **Rich Content Support**: Send and read rich emails, including HTML and attachments.
- **Multiple Accounts**: Seamlessly manage multiple email accounts in one interface.
- **Plugin System**: Extend functionality with a plugin ecosystem that includes over 35 community plugins.
- **AI Integration**: Includes features like an AI rewrite plugin and the ability for AI agents to send emails autonomously.
- **Cross-Platform**: Compatible with various operating systems and terminal emulators.

### Usage

Matcha provides a CLI interface for managing email tasks. You can install plugins from its marketplace or directly from the command line:

```bash
matcha marketplace                # Browse plugins in the TUI
matcha install <url_or_file>      # Install a plugin
matcha config <plugin_name>       # Configure an installed plugin
```

For AI agent integration:

```bash
matcha send --to alice@example.com --subject "Hello" --body "Sent by my AI agent"
```
