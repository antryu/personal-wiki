# Apache DataFusion

> Apache DataFusion is an extensible, high-performance SQL query engine written in Rust, leveraging Apache Arrow for efficient in-memory data processing. It is notable for its support of multiple data formats, customizable execution engine, and integration with ecosystems like Python and Apache Spark.  
> Last updated: 2026-04-13

## Overview
**Apache DataFusion** is a query engine designed for building fast, scalable data systems, offering SQL and DataFrame APIs out-of-the-box. It is optimized for analytics workloads and supports formats like CSV, Parquet, JSON, and Avro. The project is part of the Apache Software Foundation and is licensed under the Apache License 2.0.

Key features include a columnar, multi-threaded execution engine, partitioned data source handling, and extensibility for custom workloads. Related projects include:
- **[DataFusion Python](https://github.com/apache/datafusion-python/)**: Python bindings for SQL and DataFrame operations.
- **[DataFusion Comet](https://github.com/apache/datafusion-comet/)**: A Spark accelerator built on DataFusion.

## Key Points
- Written in **Rust** for performance and memory safety.
- Uses **Apache Arrow** for efficient columnar in-memory data representation.
- Supports **SQL** and **DataFrame** APIs for analytics and data processing.
- Extensible architecture for custom query planning, execution, and data sources.
- Actively maintained with a large community (8.5k+ stars, 2k+ forks).
