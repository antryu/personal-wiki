# Generative UI Framework for Dynamic UIs

> json-render is a Generative UI framework that enables developers to create dynamic and personalized user interfaces from natural language prompts, all while maintaining reliability and control through customizable components and strict schemas. The framework is notable for its cross-platform capabilities and pre-built UI components.
> Last updated: 2026-05-08

## Overview

The **json-render** framework, hosted on GitHub by **vercel-labs**, is a powerful tool for generating user interfaces using AI prompts. It allows developers to define a catalog of components and actions, and then use natural language prompts to generate UIs that only utilize these predefined elements. This ensures predictability, safety, and cross-platform compatibility across web, mobile, and even terminal environments.

The framework supports multiple front-end libraries like React, Vue, Svelte, and React Native, as well as tools for creating PDFs, HTML emails, 3D scenes, and more. It is designed to be fast, streamable, and easy to use, with a variety of pre-built components and batteries-included convenience.

json-render uses TypeScript and is licensed under the Apache License 2.0. It has been created on January 14, 2026, and is actively maintained with the latest update on May 8, 2026.

## Key Points
- AI-generated UIs based on natural language prompts
- Predefined components and actions ensure predictable and safe output
- Cross-platform support for React, Vue, Svelte, React Native, and more
- Real-time rendering with streaming and progressive updates
- Pre-built components for quick setup and customization
