# React Video Creation Framework

> Remotion is a framework for creating videos programmatically using React, leveraging web technologies, programming logic, and React's component model to enable dynamic video generation. It has gained widespread adoption with over 43k stars and is notable for its integration of web development practices into video workflows.  
> Last updated: 2026-04-13

## Overview
**Remotion** (https://github.com/remotion-dev/remotion) is a TypeScript-based framework that allows developers to generate videos using React components. Key features include:

- **Repository Stats**:  
  - 43,109 stars | 2,845 forks | 84 open issues  
  - Created: 2020-06-23 | Last Updated: 2026-04-13  

- **Core Philosophy**:  
  - Combines React's component-based architecture with web technologies (CSS, SVG, WebGL) for dynamic video creation.  
  - Enables programmatic control via variables, functions, and APIs for effects and animations.  

- **Notable Use Cases**:  
  - Fireship's "This video was made with code" demo (source: [GitHub](https://github.com/wcandillon/remotion-fireship)).  
  - GitHub Unwrapped: Personalized Year-in-Review videos ([source](https://github.com/remotion-dev/github-unwrapped)).  

- **Getting Started**:  
  - Quick start via `npx create-video@latest`.  
  - Full documentation at [remotion.dev/docs](https://www.remotion.dev/docs).  

- **License**:  
  - Requires a company license for enterprise use; details in [LICENSE.md](https://github.com/remotion-dev/remotion/blob/main/LICENSE.md).  

## Key Points
- Leverages **web technologies** (CSS, Canvas, WebGL) for video composition.  
- Enables **programmatic effects** using variables, APIs, and algorithms.  
- Special license requirements for **enterprise/commercial use**.
