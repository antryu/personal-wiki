# Antigravity Account Manager

> Antigravity Account Manager 是一个专业的 AI 账号管理与协议中继工具，支持一键无缝切换多个 AI 账号，并提供跨平台、高性能的 API 接口转换功能。该工具基于 Tauri v2 和 Rust 构建，为开发者和 AI 用户提供了一个高效、稳定的本地调度网关。  
> Last updated: 2026-05-07

## Overview

Antigravity Account Manager 是一款专为 AI 开发者和 AI 爱好者打造的桌面应用，旨在简化 AI 账号管理并实现多种 AI 模型协议的统一调度。该工具通过集成多账号管理、协议转换和智能调度功能，为用户提供一个本地化的 AI 接口中转平台。

该项目目前托管在 GitHub 上，拥有 28,901 个 star 和 3,158 个 fork，活跃度较高。项目采用 Rust 语言开发，并结合 React 构建前端界面，使用 Tauri v2 进行跨平台打包。其核心功能包括智能账号切换、协议适配、API 中继和详细的监控仪表盘。

项目还得到了多个 API 中继服务商的赞助，包括 PackyCode、AICodeMirror 和 VisionCoder，它们为用户提供专属优惠和福利。

## Key Points

- **多账号管理**：支持一键无缝切换 Google、Anthropic 和 Gemini 等 AI 平台的账号，提供直观的界面和智能推荐功能。
- **协议中继**：将 Web 端 Session 转换为标准化 API 接口，适配 OpenAI、Anthropic 和 Gemini 等不同协议格式。
- **智能调度**：根据账号的配额使用情况，自动推荐最佳账号，提高调用效率并减少因配额不足导致的中断。
- **跨平台部署**：基于 Tauri 和 Rust 开发，支持 Windows、Linux 和 macOS 等主流平台。
- **开源与赞助**：项目开源，拥有多个赞助商和社区支持，提供稳定的服务和持续更新。
