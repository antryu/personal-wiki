### Introduction

UI-TARS-desktop is a desktop application that offers a native GUI Agent experience, leveraging the [UI-TARS](https://github.com/bytedance/UI-TARS) model. It provides two primary modes of operation: a **local operator** and a **remote operator**, allowing users to run agents on their own machines or connect to remote systems. This enables advanced multimodal AI capabilities in real-world application scenarios like web browsing, computer automation, and more.

### Features

- **Local and Remote Operators**: The application supports both local and remote execution environments, allowing agents to interact with desktop or remote systems.
- **Multimodal Capabilities**: UI-TARS-desktop integrates vision and GUI interaction features, making it capable of understanding and acting on both visual and textual inputs.
- **Browser Integration**: It can operate within browser environments, enabling AI agents to perform tasks like clicking, typing, and navigating web pages.
- **Custom Tool Integration**: The application is compatible with various real-world tools, allowing for seamless automation and task execution.
