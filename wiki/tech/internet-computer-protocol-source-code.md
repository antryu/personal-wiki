# Internet Computer Protocol Source Code

> The Internet Computer Protocol (ICP) is the world's first blockchain that runs at web speed and can scale infinitely. This repository contains the source code for the Rust-based "replica" software run by nodes in the network, making it a key foundation for the decentralized Internet Computer ecosystem.
> Last updated: 2026-05-14

## Overview

The **dfinity/ic** repository houses the source code for the Internet Computer Protocol, a decentralized blockchain platform designed for high performance and scalability. The software is written in Rust and is used by nodes across the network to execute the "replica" — the core client software that runs on each node.

This repository was created on May 9, 2021, and has since gathered significant community attention with over 1,755 stars and 397 forks on GitHub. It is maintained by the DFINITY Foundation and is open for contributions under a custom license.

The Internet Computer Protocol is notable for its ability to scale without limits and for its integration with the web. It introduces new features such as canister smart contracts, which are more powerful and flexible than traditional smart contracts.

## Key Points
- The Internet Computer Protocol is the first blockchain that operates at web speed and can scale infinitely.
- The repository contains the Rust implementation of the "replica", a core client used by nodes in the network.
- The ICP supports advanced features like canister smart contracts and the Network Nervous System (NNS) for decentralized governance.
