# MarkEdit: Lightweight macOS Markdown Editor

> MarkEdit is a lightweight, open-source macOS Markdown editor designed for speed and simplicity. It offers a native macOS experience with privacy-first principles, fast performance (handling 10 MB files effortlessly), and a 4 MB installer size.  
> Last updated: 2026-04-17

## Overview
MarkEdit is a free and open-source Markdown editor for macOS, built with Swift and MIT licensed. It emphasizes privacy, native macOS integration, and performance using [CodeMirror 6](https://codemirror.net/). Key features include:
- **Privacy-focused**: No user data collection
- **Native macOS experience**: Supports force-touch word lookup, inline predictions, and Writing Tools
- **High performance**: Edits 10 MB files and million-line files efficiently
- **Extensible**: Integrates with AppleScript, Shortcuts, and custom CSS/JavaScript extensions
- **GFM-compliant**: Adheres strictly to GitHub Flavored Markdown specifications

Installation options include downloading a `.dmg` file from [releases](https://github.com/MarkEdit-app/MarkEdit/releases/latest) or using Homebrew: `brew install --cask markedit`. The app supports macOS 12–15+ and checks for updates automatically.

## Key Points
- Built on CodeMirror 6 for fast, accurate Markdown editing
- 4 MB installer with no bloatware or ads
- Native macOS UI with full feature parity to TextEdit for Markdown
- Open-source with active community contributions (4.2k stars, 171 forks)
