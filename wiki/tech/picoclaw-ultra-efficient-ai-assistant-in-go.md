# PicoClaw: Ultra-Efficient AI Assistant in Go

> PicoClaw is an ultra-lightweight, open-source AI assistant written entirely in Go. It runs on $10 hardware with less than 10MB RAM, making it highly efficient and deployable in minimal environments. It’s ideal for automation tasks and creative workflows.
> Last updated: 2026-05-07

## Overview

PicoClaw is an open-source AI assistant developed by [Sipeed](https://sipeed.com). It was built entirely in Go and optimized through a self-bootstrapping process, meaning the AI agent itself helped in code optimization and architecture migration. The project is not a fork of OpenClaw, NanoBot, or any other existing project.

PicoClaw is designed to be extremely lightweight and efficient, with the ability to run on low-cost hardware, such as the Lichee Pi Zero, which costs around $10 and uses less than 10MB RAM. This makes it significantly more memory-efficient compared to similar AI assistants and much cheaper than high-end hardware alternatives.

The project supports a wide range of hardware architectures, including x86_64, ARM64, MIPS, RISC-V, and LoongArch. It is licensed under the MIT License and is continuously updated with new features and improvements.

Notable updates include Android support, integration with WeChat and WeCom, and the addition of new AI service providers like AWS Bedrock and Azure. The community is actively engaged with over 28,800 stars and 4,100 forks on GitHub.

## Key Points

- Runs on $10 hardware with less than 10MB RAM.
- Built entirely in Go from scratch, not a fork of any other project.
- Supports multiple hardware architectures (x86_64, ARM64, MIPS, RISC-V, LoongArch).
- Actively developed with a strong community and growing feature set.
- Includes integrations with WeChat, WeCom, and cloud providers like AWS and Azure.
