# OpenMAIC: Multi-Agent Interactive Classroom

> OpenMAIC is an innovative open-source project that transforms online learning through a multi-agent interactive classroom experience, enabling immersive education in one click. It leverages cutting-edge AI technologies to provide a rich and dynamic learning environment.  
> Last updated: 2026-05-11

## Overview
OpenMAIC (Open Multi-Agent Interactive Classroom) is an open-source platform developed by THU-MAIC, designed to revolutionize the way users engage with educational content through AI-driven interactivity. The platform utilizes multiple AI agents to simulate real-world classroom dynamics, offering an immersive and personalized learning experience. OpenMAIC is built with TypeScript and employs frameworks like Next.js and React for a robust and scalable architecture. It is released under the GNU Affero General Public License v3.0, ensuring open access and collaboration.

The project features live demos, a strong community presence (e.g., on Discord and Feishu), and integration with AI models and tools such as OpenClaw and Lemonade. OpenMAIC supports multiple AI providers and includes features like TTS with voice cloning, interactive simulations, and a modular classroom environment.

## Key Points
- **Multi-Agent Learning**: Utilizes AI agents to simulate interactive classroom environments for immersive learning.
- **Open Source**: Released under AGPL-3.0, allowing for community contributions and customization.
- **Live Demo and Community**: Offers a live demo and active community engagement through Discord and Feishu.
- **Customizable AI Integration**: Supports integration with various LLMs and local AI solutions like Lemonade and OpenClaw.
- **Regular Updates**: Frequent releases with features like 3D visualization, voice cloning, and enhanced classroom simulations.
- **Educational Focus**: Designed for diverse learning scenarios, including deep interactive mode for games, simulations, and mind maps.
