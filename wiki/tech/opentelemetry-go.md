# OpenTelemetry Go

> OpenTelemetry Go is the Go implementation of the OpenTelemetry project, offering APIs and tools for distributed tracing, metrics, and logging. It enables developers to collect and export telemetry data for observability, making it a key tool for monitoring cloud-native applications.
> Last updated: 2026-04-15

## Overview

The OpenTelemetry Go project provides a stable API and SDK for instrumenting Go applications to support distributed tracing, metrics collection, and logging. It is part of the broader OpenTelemetry ecosystem and supports sending telemetry data to various observability backends.

Key features include:

- **Tracing**: Captures and records the flow of requests through distributed systems.
- **Metrics**: Collects and exports performance metrics.
- **Logs**: Supports structured logging (in beta).

The project ensures compatibility with recent versions of the Go programming language and is actively maintained with continuous integration and testing across multiple platforms.

## Key Points

- OpenTelemetry Go provides APIs for tracing, metrics, and logs, enabling developers to monitor and debug complex systems.
- It is compatible with the latest supported versions of Go and runs on multiple operating systems and architectures.
- Traces and metrics are stable features, while logs are in beta.
- The project is open-source under the Apache License 2.0 and hosted on GitHub.
