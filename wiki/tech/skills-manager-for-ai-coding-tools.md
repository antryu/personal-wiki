# Skills Manager for AI Coding Tools

> A lightweight desktop app to manage, sync, and organize AI agent skills across 15+ coding tools. Notable for its centralized control and support for multiple platforms, including Cursor, Claude Code, Codex, and Copilot.
> Last updated: 2026-05-04

## Overview
The **Skills Manager** is a Rust-based desktop application designed to streamline the management of AI agent skills across various coding tools. It provides a unified skill library, enables synchronization across tools, and supports project-specific workspaces for custom skill management. The app also includes a marketplace for discovering and installing new skills, along with support for Git-based backup and restoration.

The project was launched on March 2, 2026, and has since gained traction with 1,075 stars and 96 forks. It is open-source and distributed under the MIT License.

## Key Points
- Unified skill management across 15+ coding tools.
- Centralized library and marketplace for skill discovery and installation.
- Multi-tool sync and project-specific workspaces.
- Git-based backup and restoration for version control.
- Custom tool support and flexible app settings.
