# Open Source Voice Agent Platform

> Dograh is an open-source voice agent platform that provides a self-hostable alternative to services like Vapi and Retell. With a drag-and-drop workflow builder, it allows developers to build voice agents quickly and offers full control and transparency due to its open-source nature.  
> Last updated: 2026-05-16

## Overview
The Dograh AI project is designed as a flexible and transparent open-source platform for creating voice agents. It supports inbound and outbound voice calls and enables users to build production-grade bots rapidly. Dograh offers full self-hosting capabilities, allowing customization of LLM, TTS, and STT integrations without vendor lock-in. It is actively maintained by Y Combinator alumni and exit founders, who are committed to open voice AI technology.

Key highlights include support for BSD 2-Clause licensing, compatibility with Python and Docker, and a vibrant ecosystem with 1,312 stars and 311 forks on GitHub. The platform features a web-based UI accessible via a local server, and includes documentation and community support through Slack.

## Key Points
- Fully open-source and self-hostable, offering flexibility and avoiding vendor lock-in.
- Easy setup with a single Docker command and drag-and-drop workflow builder.
- Provides integration with various LLMs, TTS, and STT providers.
- Maintained by experienced developers and founders with a strong commitment to open voice AI.
- Offers free self-hosted deployment with optional cloud-based, usage-based pricing.
