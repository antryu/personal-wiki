# Saleor Core API

> Saleor Core is a high-performance, composable, headless commerce API built for scalable and flexible e-commerce solutions. With over 22,800 stars and 5,974 forks, it offers a GraphQL-native, API-only architecture for modern, technology-agnostic commerce platforms.  
> Last updated: 2026-04-18

## Overview
**Saleor Core API** is an open-source, Python-based headless commerce platform designed for scalability and extensibility. Hosted on GitHub since 2013, it provides a GraphQL-native API for managing multichannel commerce operations, including cart, checkout, order management, payments, and product inventory. ## Key Points
- **Technology-Agnostic**: Works with any frontend or backend stack.
- **Headless & API-Only**: No frontend lock-in; full control via APIs.
- **Open Source**: Single, unrestricted version with active community support.
- **Native Multichannel**: Channel-specific control over pricing, stock, and product data.
- **High Performance**: Optimized for global commerce use cases.
