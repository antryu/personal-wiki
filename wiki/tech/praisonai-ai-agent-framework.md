# PraisonAI: AI Agent Framework

> PraisonAI is an AI agent framework that allows users to deploy autonomous, self-improving agents that can research, plan, code, and execute tasks across applications. It stands out for enabling the creation of an AI workforce with minimal code and support for over 100 large language models.
> Last updated: 2026-05-04

## Overview

PraisonAI is an open-source AI agent framework designed to simplify the deployment and management of autonomous AI agents. With PraisonAI, users can create self-improving agents capable of performing complex tasks such as research, code generation, data pipeline management, and workflow automation. The framework is written in Python and released under the MIT License. PraisonAI supports over 100 large language models and provides built-in memory and RAG capabilities for enhanced performance.

### Key Features
- **Autonomous Agents**: Deploy self-improving AI agents that can perform tasks across applications.
- **Rapid Deployment**: Agents can be deployed in under 1 minute using 5 lines of code.
- **Model Agnostic**: Supports over 100 large language models.
- **Built-in Capabilities**: Includes memory, RAG (retrieval-augmented generation), and task execution.
- **Multi-Agent Support**: Enables the creation of an entire AI workforce or organization.
- **Visual Dashboard**: Provides a user-friendly interface for monitoring and managing agents.
- **Cross-Platform**: Compatible with platforms like Telegram, Discord, and Slack for customer support use cases.

## Key Points
- PraisonAI is designed to reduce boilerplate code and automate complex tasks with AI agents.
