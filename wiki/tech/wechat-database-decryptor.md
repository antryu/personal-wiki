# WeChat Database Decryptor

> WeChat 4.0 database decryptor - extract keys from memory, decrypt SQLCipher 4 databases, real-time message monitor
> Last updated: 2026-05-11

## Overview
**WeChat 4.x Database Decryptor** 是一个用于解密微信 4.0（Windows、macOS、Linux）本地数据库的工具。它能够从微信运行时的进程内存中提取加密密钥，从而解密所有 SQLCipher 4 加密的数据库文件，并提供实时消息的监控功能。

该项目支持多平台使用，包括 Windows、Linux 和 macOS，并提供了一个 Web UI 用于查看消息和管理数据。同时，它也集成了 API 和 AI 功能（如 MCP Server），方便用户扩展使用。

## Key Points
- 支持跨平台（Windows、Linux、macOS）
- 可以实时监控消息流并展示在 Web UI 中
- 提供富媒体内容解析，如表情、链接卡片、视频号等
- 提供 API 用于访问和整合数据
- 与 AI 工具（如 Claude AI）集成，实现智能查询
