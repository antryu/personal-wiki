# AI-Powered React App Builder

> Clone and recreate any website as a modern React app in seconds using AI-powered generation. This notable tool integrates multiple LLM providers and offers instant app creation through conversational interfaces.  
> Last updated: 2026-04-17

## Overview
The **firecrawl/open-lovable** repository provides an AI-driven solution for building React applications instantly. Developed by the Firecrawl team, it allows users to chat with AI to generate functional React apps, with optional cloud deployment through Lovable.dev. The project supports integration with major LLM providers (Google Gemini, Anthropic, OpenAI, Groq) and sandbox environments (Vercel or E2B).

### Setup Process
1. **Clone & Install**  
   ```bash
   git clone https://github.com/firecrawl/open-lovable.git
   cd open-lovable
   pnpm install  # or npm install / yarn install
   ```

2. **Configure Environment Variables**  
   Create a `.env.local` file with required API keys, including:
   - `FIRECRAWL_API_KEY` (required)
   - Optional LLM provider keys (Gemini, Anthropic, OpenAI, Groq)
   - Sandbox provider configuration (Vercel default or E2B)

3. **Run Locally**  
   ```bash
   pnpm dev  # or npm run dev / yarn dev
   ```
   Access the app at [http://localhost:3000](http://localhost:3000).
