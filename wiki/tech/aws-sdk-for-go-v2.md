# AWS SDK for Go v2

> The AWS SDK for Go v2 is the second major version of the official software development kit for the Go programming language, allowing developers to interact with AWS services. It is notable for offering improved modularity, a more consistent API, and better support for Go language features.
> Last updated: 2026-05-05

## Overview
The AWS SDK for Go v2 is a major rewrite of the original AWS SDK for Go, designed to improve usability and maintainability. It is written in Go and supports a wide range of AWS services.

Key features of the SDK include modular service clients, support for shared configuration and credentials, and a simplified API with idiomatic Go patterns.

The v2 SDK requires Go 1.24 or newer, and follows a support policy that aligns with the Go language's release policy with an additional six months of support for deprecated versions.

Getting started with the SDK involves setting up a Go module, installing required dependencies, and writing code to interact with AWS services. The SDK provides a comprehensive set of tools for developers, including documentation, migration guides, and API references.

## Key Points
- Modular service clients for better code organization and reuse
- Supports Go 1.24 and newer with extended support for deprecated versions
- Comprehensive documentation and migration guides for easy adoption
- Simplified API design that aligns with idiomatic Go patterns
- Active community support and Apache License 2.0 for open-source use
