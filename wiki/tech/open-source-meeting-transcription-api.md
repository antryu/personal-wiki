# Open-source Meeting Transcription API

> Vexa is an open-source meeting transcription and bot API that supports real-time transcription and AI agent integration for Google Meet, Microsoft Teams, and Zoom. It provides a self-hostable alternative to commercial SaaS solutions like Otter.ai and Fireflies.ai.
> Last updated: 2026-05-04

## Overview
Vexa is an open-source, self-hostable meeting bot API and transcription API that supports real-time transcription for meetings on Google Meet, Microsoft Teams, and Zoom. With real-time WebSocket-based transcripts, auto-join bots, and an MCP (Meeting Control Protocol) server for AI agents, Vexa offers a flexible and customizable alternative to proprietary tools like Recall.ai, Otter.ai, and Fireflies.ai.

Users can choose to self-host Vexa on their own infrastructure, ensuring data sovereignty, or use the hosted service at [vexa.ai](https://vexa.ai). It supports multiple languages for transcription, interactive bots that can speak and share screen content during meetings, and a dashboard for managing meetings, transcripts, and AI agents.

## Key Points
- Real-time transcription with per-speaker identification using Whisper, supporting over 100 languages.
- Self-hostable API to ensure meeting data stays within your infrastructure.
- MCP server with 17 tools for AI agents like Claude and Cursor to interact with meetings.
- Supports meeting bot capabilities including auto-join, recording, speaking, and screen sharing.
- Multi-tenant architecture with scoped tokens for embedding in enterprise products.
- Built with TypeScript and can be extended or customized via open-source code.
