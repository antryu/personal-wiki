# RuVector: Self-Learning Vector AI System

> RuVector is a high-performance, real-time, self-learning AI vector system built in Rust. It features a graph neural network (GNN), memory database, and AI inference capabilities. It is designed to learn from usage and optimize itself, offering superior search and retrieval performance through self-tuning and adaptive algorithms.
> Last updated: 2026-05-06

## Overview
RuVector is a self-learning, vector memory system and agentic operating system developed in Rust. It combines a graph neural network (GNN) with a self-optimizing database that adapts to user queries and workloads in real-time. Unlike traditional vector databases, RuVector improves its search results over time by learning from user interactions. It supports hybrid search, graph-based RAG, and advanced attention mechanisms. RuVector integrates with PostgreSQL, supports low-latency inference, and can run locally without cloud dependencies. It is open-source under the MIT license and features a single-file deployment option.

Key features include:

- A GNN that improves search results over time
- Self-optimization via the SONA engine
- 50+ attention mechanisms including FlashAttention-3 and Mamba SSM
- Transfer learning across domains
- Hybrid search combining sparse and dense vectors
- Graph RAG for multi-hop queries
- DiskANN for large-scale SSD-backed ANN
- TurboQuant for memory-efficient quantization
- Support for multiple environments including browsers and PostgreSQL

## Key Points
- RuVector is a self-learning vector AI system built in Rust.
- It integrates graph neural networks (GNN), attention mechanisms, and memory optimization for real-time AI inference.
- It learns from user interactions and improves search results automatically.
- The system includes a self-optimizing routing engine (SONA) and 50+ attention mechanisms.
- It supports hybrid search, graph-based RAG, and high-performance disk-based ANN indexing.
- RuVector is open-source, MIT licensed, and designed for low-latency, local execution.
