# iOS Chat SDK in Swift

> The official iOS SDK for Stream Chat, providing a low-level client and reusable UI components for building chat applications. Notable for its support of UIKit, SwiftUI, and advanced features like offline messaging and structured concurrency.  
> Last updated: 2026-04-17

## Overview
The **GetStream/stream-chat-swift** repository offers an iOS SDK for integrating chat functionality using Stream Chat's API. Key components include:

- **Low-Level Client (LLC)**: A non-UI SDK for custom UI implementations.  
- **UIKit SDK**: UI components for iOS 13+ applications.  
- **SwiftUI SDK**: Modern UI components for iOS 14+ (hosted in a separate repository).  

The SDK supports offline messaging, native platform behaviors (e.g., dark mode, dynamic fonts), and integrates with Swift features like Combine and Structured Concurrency. It is open-source and compatible with Swift Package Manager (SPM).
