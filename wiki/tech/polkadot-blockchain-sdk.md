# Polkadot Blockchain SDK

> The Parity Polkadot SDK provides tools and components for building on Polkadot, a multi-chain blockchain platform enabling secure, scalable interoperability between blockchains. It includes Substrate, FRAME, Cumulus, and XCM frameworks for developing custom blockchains and parachains.  
> Last updated: 2026-04-11

## Overview
The **paritytech/polkadot-sdk** repository is a comprehensive toolkit for developers working on the Polkadot network. Hosted on GitHub with 2,743 stars and 1,178 forks, it is written in Rust and focuses on interoperability, modularity, and scalability for blockchain development. Key features include:

- **Core Components**: Integrates Substrate (blockchain framework), FRAME (modular runtime environment), Cumulus (parachain development), and XCM (cross-chain messaging).
- **Quickstart**: A script (`getting-started.sh`) allows rapid deployment of example nodes.
- **Build Targets**: Supports WASM (with `RUSTFLAGS="--cfg substrate_runtime"`) and PolkaVM (RISC-V architectures).
- **Documentation**: Extensive guides, API references, and templates for building FRAME pallets or starting new projects.
- **Community**: Active support via Telegram, Matrix, Discord, and StackExchange.

The SDK releases stable versions every three months, with badges tracking current and upcoming versions. It is maintained under an "N/A" license (likely open-source but unspecified) and has 2,235 open issues as of April 2026.

## Key Points
- Enables development of interoperable blockchains and parachains on Polkadot.
- Supports RISC-V and WASM architectures for runtime and PolkaVM compatibility.
- Includes modular frameworks (FRAME, Cumulus) and cross-chain communication (XCM).
- Provides detailed documentation, templates, and community-driven support channels.
- Regularly updated with quarterly releases and active contributor engagement.
