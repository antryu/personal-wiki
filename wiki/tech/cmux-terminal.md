# cmux Terminal

> A Ghostty-based macOS terminal with vertical tabs and notifications designed for AI coding agents, combining productivity features with seamless integration for collaborative AI workflows. Last updated: 2026-04-10

## Overview
**cmux** is a macOS terminal application built on Ghostty, optimized for developers working with AI coding agents. It introduces vertical tabs, real-time notification systems, and deep integration with tools like Claude Code Teams and remote SSH workflows. The project has gained significant traction with 13,448 stars and 961 forks since its January 2026 launch.

Key repository details:
- **Language**: Swift
- **License**: Custom (non-standard)
- **Topics**: AI coding (Claude, Gemini, Codex), terminal emulation, tmux alternatives
- **Active Development**: 1,171 open issues as of April 2026

Notable features include:
- **Visual Notification System**: Blue rings on active panes and a centralized notification panel
- **Integrated Browser**: Scriptable in-app browser with cross-browser session imports
- **Multi-Tab Architecture**: Vertical/horizontal tab splits with metadata (git status, ports, PRs)
- **SSH Workspaces**: Native remote session management with drag-and-drop file uploads
- **Claude Code Teams**: Native support for collaborative AI coding with split-pane teammates

The project maintains multilingual documentation (20+ languages) and offers community engagement via Discord and X/Twitter.

## Key Points
- Combines Ghostty's terminal capabilities with AI-specific workflows for code collaboration
- Features a unique visual notification system for managing AI agent interactions
- Supports remote development via SSH with integrated browser routing
- Provides native Claude Code Teams integration without requiring tmux
- Offers extensive customization through Swift-based development and plugin architecture
