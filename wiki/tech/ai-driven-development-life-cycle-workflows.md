# AI-Driven Development Life Cycle Workflows

> AI-DLC is an intelligent software development workflow that adapts to your needs, maintains quality standards, and keeps you in control of the process. This repository provides adaptive workflow steering rules for AI coding agents and is notable for its integration with multiple IDEs and tools to streamline AI-assisted software development.
> Last updated: 2026-05-07

## Overview

The AI-Driven Development Life Cycle (AI-DLC) repository, maintained by **awslabs**, introduces a set of adaptive steering rules for AI coding agents to guide and optimize the software development process. AI-DLC supports a three-phase workflow: plan, implement, and validate, and it integrates with several popular IDEs and tools, including Kiro, Amazon Q Developer, Cursor IDE, and more. It ensures developers remain in control while leveraging AI to enhance productivity and maintain high-quality code output.

The repository includes platform-specific setup instructions and a set of rules that help AI agents adapt their behavior based on the project's needs. It also supports version control best practices and encourages developers to review AI-generated output to ensure accuracy and cost efficiency.

## Key Points

- Provides adaptive workflow steering rules for AI coding agents.
- Supports multiple IDEs and platforms including Kiro, Amazon Q Developer, Cursor IDE, and more.
- Implements a three-phase workflow: plan, implement, and validate.
- Encourages developers to review AI-generated output for accuracy and cost efficiency.
- Includes version control recommendations and troubleshooting guides.
