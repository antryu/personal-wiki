# Authelia SSO & MFA Portal

> Authelia is an OpenID Certified™ Single Sign-On (SSO) and Multi-Factor Authentication (MFA) portal for securing web applications. It provides robust identity verification through methods like TOTP, WebAuthn, and YubiKey, making it a notable solution for enterprise-grade security.  
> Last updated: 2026-04-17

## Overview
Authelia is an open-source authentication portal designed to secure access to web applications using SSO and MFA. It supports a wide range of authentication protocols and integrations, including LDAP, OAuth2, OpenID Connect, and Kubernetes. The project, originally created in 2016, has gained significant traction with over 27.5k stars and 1.4k forks on GitHub. Authelia's OpenID Certification™ ensures compliance with industry standards for secure authentication workflows.

Key features include:
- **Multi-Factor Authentication (MFA)**: Supports TOTP, WebAuthn, YubiKey, and push notifications.
- **Single Sign-On (SSO)**: Centralized access control for multiple applications.
- **Integration Flexibility**: Compatible with Docker, Kubernetes, LDAP, and cloud providers.
- **OpenID Compliance**: Certified for secure, standardized authentication practices.

The repository is actively maintained, with 108 open issues as of the latest update, and is licensed under the Apache License 2.0.

## Key Points
- OpenID Certified™ SSO and MFA solution for web applications.
- Supports diverse authentication methods (TOTP, WebAuthn, YubiKey, etc.).
- Integrates with LDAP, Kubernetes, Docker, and cloud platforms.
- Apache License 2.0 ensures open-source flexibility.
