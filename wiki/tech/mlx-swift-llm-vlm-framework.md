# MLX Swift LLM/VLM Framework

> A framework for building and deploying large language models (LLMs) and vision-language models (VLMs) using Swift and MLX, enabling efficient AI development with a focus on performance and accessibility.  
> Last updated: 2026-04-11

## Overview
The `ml-explore/mlx-swift-lm` repository provides tools and libraries to develop LLMs and VLMs using Swift, leveraging the MLX library for optimized machine learning workflows. Hosted on GitHub, the project has gained 382 stars and recently saw a surge in popularity with 9 new stars in a single day. Built with Swift, it targets developers seeking high-performance AI solutions while benefiting from Swift's modern programming features. The framework integrates MLX's capabilities for streamlined model training, inference, and deployment.
