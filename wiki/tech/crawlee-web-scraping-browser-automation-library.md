# Crawlee - Web Scraping & Browser Automation Library

> Crawlee is a powerful and reliable web scraping and browser automation library for Node.js, designed to help developers extract data for AI, LLMs, RAG, or GPTs. It supports Puppeteer, Playwright, Cheerio, JSDOM, and raw HTTP, and works in both headful and headless modes with proxy rotation.
> Last updated: 2026-05-10

## Overview
Crawlee provides tools to crawl and scrape the web, extracting data and downloading HTML, PDF, JPG, PNG, and other file types. It helps build human-like scrapers that avoid detection by bot protections, even with the default configuration. Crawlee is available as the `crawlee` NPM package.

The project is actively maintained with a large community and has a comprehensive set of features for both developers and businesses. It is designed to be configurable and robust for a wide range of scraping and crawling tasks.

Crawlee can be used in its own standalone CLI form or manually integrated into existing Node.js projects. It also supports Python via a separate package, making it versatile for multi-language environments.

## Key Points
- Supports Puppeteer, Playwright, Cheerio, JSDOM, and raw HTTP for diverse use cases.
- Works in both headful and headless browser modes with proxy rotation for reliability.
- Designed to appear human-like, avoiding bot detection systems with default settings.
- Provides tools for extracting links, scraping data, and storing results in disk or cloud.
- Offers both CLI and manual installation options for quick setup and integration.
- Available in JavaScript/TypeScript with a Python version also available.
