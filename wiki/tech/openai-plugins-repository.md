# OpenAI Plugins Repository

> The OpenAI Plugins Repository is a curated collection of Codex plugin examples, offering developers a framework and tools for extending Codex's capabilities with custom integrations. It's notable for its structured approach and real-world implementation examples across various platforms.
> Last updated: 2026-05-07

## Overview
The `openai/plugins` repository serves as a centralized resource for plugin development using OpenAI's Codex platform. It provides developers with plugin templates and example implementations across a range of applications, including design, productivity, mobile and web app development, and cloud deployment. Each plugin is structured with a required manifest file and optional supporting components, such as skills, commands, and hooks. This repository is actively maintained with over 1,000 stars and is open for contributions.

## Key Points
- Offers a standardized structure for Codex plugin development.
- Includes examples for plugins targeting tools like Figma, Notion, and Google Slides.
- Contains advanced workflows for building iOS, macOS, and web applications.
- Designed for integration with Codex through a plugin manifest and skill-based extensions.
- Supports additional tools such as Expo, React Native, and Netlify for broader development needs.
