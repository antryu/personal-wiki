# Stripe iOS SDK

> The Stripe iOS SDK simplifies the integration of payment features into iOS apps, offering both prebuilt UI components and low-level APIs. It enables developers to create secure, SCA-compliant, and customizable payment experiences.  
> Last updated: 2026-04-16

## Overview
The **Stripe iOS SDK** is a powerful tool that allows developers to build secure and user-friendly payment experiences in iOS applications. It provides ready-to-use UI screens for collecting payment details and also exposes the underlying APIs for custom integrations. The SDK supports Apple Pay, 3D Secure authentication, and card scanning, and it is designed to ensure PCI compliance by handling sensitive data directly with Stripe's servers.

The project is actively maintained, with the latest update as of 2026-04-16, and has a strong community presence with over 2,500 stars on GitHub. It is written in Swift and is distributed under the MIT License.

## Key Points
- Simplified security by directly sending sensitive data to Stripe for PCI compliance.
- Supports Apple Pay and 3D Secure authentication for SCA compliance in Europe.
- Offers a prebuilt UI like PaymentSheet to streamline payment collection.
