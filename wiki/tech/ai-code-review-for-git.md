# AI Code Review for Git

> `git-lrc` is a free and unlimited AI-powered code review tool that automatically reviews every code diff during the `git commit` process. It helps developers catch bugs, logic changes, and security issues early by providing inline feedback directly on the code changes. This tool is notable for its seamless integration into Git workflows and its ability to prevent silent logic changes introduced by AI agents.
> Last updated: 2026-04-19

## Overview

`git-lrc` is a Go-based tool designed to provide real-time, AI-driven code reviews for every Git commit. It addresses the growing challenge of ensuring code quality and security in fast-paced development environments, especially where AI agents are used to generate or modify code.

The tool integrates directly into the Git workflow, ensuring that every change is reviewed before it is committed. This allows developers to catch issues such as logic changes, code removals, and security vulnerabilities early in the development cycle. By reviewing code at the point of commit, `git-lrc` reduces the risk of introducing bugs or security flaws into production systems.

Key features include inline comments, real-time feedback, and a lightweight setup process. The tool is open-source and supports multiple languages, with translations available in several languages. It is actively maintained and has a growing community on Discord.

## Key Points

- Automatically reviews code changes during the `git commit` process.
- Catches logic changes, code removals, and security issues before they reach production.
- Provides inline AI-powered comments for detailed feedback.
- 60-second setup and fully free to use.
- Integrates with any Git-based workflow, regardless of the development stack.
