# Modular LLM Applications in Rust

> Rig is a Rust library for building scalable, modular, and ergonomic LLM-powered applications. It supports 20+ model providers and 10+ vector store integrations under a unified interface, enabling developers to build sophisticated AI-driven solutions with minimal boilerplate. The project has gained notable traction on GitHub, with over 7,000 stars, and is used by organizations like St. Jude and Coral Protocol.
> Last updated: 2026-05-09

## Overview

`Rig` is a Rust-based framework developed by 0xPlaygrounds for building modular and scalable LLM applications. The project provides a unified interface to over 20 LLM providers and 10 vector store integrations. It supports features like multi-turn streaming, semantic compatibility with the GenAI OpenTelemetry specification, and full WASM compatibility. Rig is designed to simplify complex AI workflows, including completion, embeddings, image generation, and transcription, while offering ergonomic APIs for developers.

The project is actively maintained and has a growing ecosystem, with community support via Discord and a comprehensive API reference. However, due to its rapid development cycle, the library is currently in a volatile state and may include breaking changes in future updates.

## Key Points

- Provides a unified interface for 20+ LLM providers and 10+ vector store integrations.
- Supports semantic compatibility with the GenAI OpenTelemetry specification.
- Offers full WASM compatibility (core library only).
- Used by notable organizations like St. Jude and Coral Protocol for chatbots, AI agents, and genomics visualization tools.
