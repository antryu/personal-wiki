# Secure AI Platform with MCP Tools

> Enterprise AI platform with centralized MCP orchestration, security guardrails, and governance tools to mitigate data exfiltration risks and reduce AI costs. Combines observability with Kubernetes-native deployment for enterprise-scale AI management.
> Last updated: 2026-04-16

## Overview
- **Repository**: [archestra-ai/archestra](https://github.com/archestra-ai/archestra) (3.5k stars, 445 forks)
- **License**: GNU Affero General Public License v3.0
- **Language**: TypeScript
- **Key Features**:
  - Centralized MCP orchestration for enterprise teams
  - Private MCP registry with governance controls
  - Dual LLM architecture for security isolation
  - Kubernetes-native deployment and API key management
  - Tool guardrails to prevent prompt injection attacks
  - AI cost reduction up to 96% through optimization
  - Organization-wide observability for AI usage and data access

- **Quickstart**: Docker deployment with PostgreSQL and runtime configuration
- **Security Focus**: Mitigates "lethal trifecta" of prompt injections, data exfiltration, and uncontrolled MCP access
- **Adoption**: 1-click deployment for technical/non-technical users, integrates with OpenAI, Claude, Gemini, and other providers

## Key Points
- Centralizes MCP servers via orchestrator to eliminate chaos and ensure credential security
- Implements non-probabilistic security measures to block data leaks from private repos
- Provides private prompt registry for company-wide agent development and sharing
- Reduces AI costs through centralized management and usage optimization
- Offers live demo of security engine preventing GitHub repo data exfiltration
