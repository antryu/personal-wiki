# Lux Video Downloader

> Lux is a fast and simple video download library and CLI tool written in Go, supporting platforms like YouTube, Bilibili, and Youku. With over 31k stars, it offers multi-platform installation and advanced features like multi-threading and proxy support.  
> Last updated: 2026-04-17

## Overview
**Lux** is a Go-based video downloader and library with a focus on simplicity and performance. It supports downloading from major platforms including YouTube, Bilibili, Youku, and more. The tool provides a CLI interface and integrates with aria2 for advanced downloads.

### Installation
Lux offers multiple installation methods:
- **Go install**: `go install github.com/iawia002/lux@latest`  
- **Homebrew (macOS)**: `brew install lux`  
- **Arch Linux**: AUR package `lux-dl`  
- **Void Linux**: `xbps-install -S lux`  
- **Windows**: Scoop (`scoop install lux`), Chocolatey (`choco install lux`), or Cask  

### Key Features
- **Multi-threaded downloads** for faster transfers  
- **Playlist and batch downloading**  
- **Proxy and cookie support** for restricted content  
- **Cross-platform compatibility** (Linux/macOS/Windows)  
- **Integration with aria2** for advanced download management
