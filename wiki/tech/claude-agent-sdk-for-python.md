# Claude Agent SDK for Python

> The Claude Agent SDK for Python is a powerful tool that enables developers to interact with the Claude Code AI agent using Python. It provides a flexible and extensible interface to run queries, manage tools, and build custom integrations. The SDK is notable for its simplicity, performance, and compatibility with the full Claude Code ecosystem.
> Last updated: 2026-05-09

## Overview

The **Claude Agent SDK for Python** offers a Pythonic interface to the Claude Code AI assistant, allowing developers to build, customize, and run applications that leverage Claude Code's capabilities for code generation, file manipulation, and system interactions. The SDK supports both simple queries and advanced features such as custom tool integration, interactive sessions, and permission management.

Key features include:

- **Easy installation and setup**
- **Async support for real-time responses**
- **Permission controls for tool usage**
- **Custom tool implementation using Python functions**
- **Compatibility with the full Claude Code CLI and toolset**

This SDK is actively maintained and has a strong open-source community. It is widely used by Python developers for building AI-powered tools and workflows.

## Key Points

- Supports asynchronous queries using `query()` to interact with Claude Code.
- Enables the creation of **custom tools** that can be used in-process, simplifying integration.
- Offers robust **permission controls** (e.g., `allowed_tools`, `disallowed_tools`).
- Includes a bundled version of the Claude Code CLI, or you can use your own installation.
- Provides an `ClaudeSDKClient` for interactive sessions with Claude Code.
