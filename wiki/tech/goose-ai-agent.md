# Goose AI Agent

> A native open-source AI agent that runs on your machine for code, workflows, and automation, supporting 15+ LLM providers and 70+ extensions via open standards. Developed by the Agentic AI Foundation (AAIF) under the Linux Foundation.  
> Last updated: 2026-04-10

## Overview
The **Goose AI Agent** is a general-purpose, extensible AI agent built in Rust for performance and portability. It operates as a desktop app (macOS, Linux, Windows), CLI tool, and embeddable API. The project moved to the [Agentic AI Foundation (AAIF)](https://aaif.io/) in 2024, with active development and community engagement (40.7k stars, 4k forks). Key features include:

- **Cross-platform support**: Native desktop application and terminal CLI workflows.
- **LLM compatibility**: Works with 15+ providers (Anthropic, OpenAI, Google, Ollama, Azure, etc.) via API keys or subscriptions.
- **Open standards**: Connects to 70+ extensions using the [Model Context Protocol (MCP)](https://modelcontextprotocol.io/).
- **Customization**: Build custom distributions with preconfigured providers, extensions, and branding via `CUSTOM_DISTROS.md`.
- **Governance**: Open-source under Apache License 2.0, with transparent governance and active issue tracking (296 open issues).

The project emphasizes performance, flexibility, and integration with existing workflows, making it suitable for code, research, automation, and data analysis.

## Key Points
- Open-source under Apache License 2.0 with 40.7k stars and 4k forks.
- Supports 15+ LLM providers (e.g., Anthropic, OpenAI, Google) and 70+ extensions via MCP.
- Part of the Linux Foundation's Agentic AI Foundation (AAIF).
- Cross-platform: desktop apps for macOS, Linux, Windows, and terminal CLI.
- Active community via Discord, YouTube, and social media.
- Governance and contribution guidelines documented in `GOVERNANCE.md`.
