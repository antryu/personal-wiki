# Microservices Demo for Cloud Technologies

> The Online Boutique is a cloud-first microservices demo application composed of 10 services that demonstrate Kubernetes, Istio, and gRPC. It showcases how developers can modernize enterprise applications using Google Cloud technologies. Last updated: 2026-05-13.

## Overview

The **Online Boutique** is a web-based e-commerce application designed to demonstrate cloud-native architecture and microservices. The application allows users to browse products, add them to a cart, and complete a purchase. It is used by Google to demonstrate modern cloud technologies and practices using Kubernetes, Istio, gRPC, and more.

The repo is actively maintained and is a popular example of a microservices application written in various programming languages including Go, C#, Node.js, Python, and Java. The project is open source, licensed under the Apache License 2.0, and has strong community and enterprise adoption. It works on any Kubernetes cluster and is not limited to Google Cloud.

The architecture consists of 11 microservices, each handling a specific functionality such as shopping cart, product catalog, payment, shipping, and recommendations, which communicate with each other using gRPC. Developers can use this project to explore real-world deployment patterns, observability, and service mesh implementations.

## Key Points
- Demonstrates a complete microservices architecture with 11 services using Kubernetes and Istio.
- Uses gRPC and Protocol Buffers for service communication.
- Written in multiple languages including Go, C#, Node.js, Python, and Java.
- Integrates with a variety of Google Cloud services for deployment, monitoring, and scalability.
- Ideal for learning and demonstrating cloud-native development practices.
