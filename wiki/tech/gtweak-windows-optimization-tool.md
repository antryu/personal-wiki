# GTweak: Windows Optimization Tool

> GTweak is a portable Windows optimization utility that provides a wide range of system tweaks and configurations. It is notable for its ability to disable unnecessary services, telemetry, and bloatware, while also offering system activation and performance enhancements.  
> Last updated: 2026-05-13

## Overview

GTweak is a powerful system utility built in C# that allows users to customize and optimize their Windows operating system. It supports both Windows 10 and Windows 11 and provides over 150 system tweaks to help users create an ideal, streamlined Windows environment. The tool is designed to be portable and user-friendly, with an intuitive interface and multiple language support, including English, French, Hebrew, Hungarian, Italian, Korean, Polish, Portuguese (Brazil), Russian, Slovenian, Thai, Turkish, Ukrainian, and Chinese. Users can easily activate Windows using HWID and KMS methods, disable unnecessary services, telemetry, and data collection, as well as remove unwanted software like OneDrive and Microsoft Edge.

## Key Points

- Disables Windows Defender, SmartScreen, Antimalware, UAC, and unnecessary services.
- Blocks telemetry and keylogger data collection from Windows, NVIDIA, and Intel.
- Offers Windows activation via HWID and KMS methods.
- Disables unnecessary system updates, diagnostics, and personalized ads.
- Portable and lightweight application with no installation required.
- Supports multiple languages and is actively maintained with BSD 3-Clause licensing.
