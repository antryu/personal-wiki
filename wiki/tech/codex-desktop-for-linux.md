# Codex Desktop for Linux

> Run OpenAI Codex Desktop on Linux with an automated installer. This unofficial project allows users to run the macOS-only Codex app on Linux by converting the upstream `.dmg` into a runnable Electron app and providing native Linux package formats.
> Last updated: 2026-05-16

## Overview

The **Codex Desktop for Linux** project is an unofficial build that brings [OpenAI Codex Desktop](https://openai.com/codex/) to Linux users. As the official app is macOS-only, this project automatically converts the macOS `.dmg` file into a runnable Linux Electron application and provides native packages in `.deb`, `.rpm`, `.pkg.tar.zst`, and `.AppImage` formats. It includes a local auto-updater that detects and rebuilds newer versions of the app from upstream Codex updates. The project also supports various Linux distributions and package managers and provides additional Linux-specific features and integrations.

## Key Points

- Converts OpenAI Codex `.dmg` to a Linux Electron app.
- Supports native packaging for Debian, Ubuntu, Fedora, Arch, and more.
- Includes a local auto-updater for native packages.
- Provides optional Linux-specific features and integrations.
- Requires no system-wide Node.js installation for standard use.
