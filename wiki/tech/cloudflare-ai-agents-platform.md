# Cloudflare AI Agents Platform

> Cloudflare AI Agents Platform enables developers to build and deploy scalable, stateful AI agents using Cloudflare's infrastructure. It offers features like persistent state, real-time communication, and AI model integration, making it ideal for agentic workloads at scale.  
> Last updated: 2026-04-17

## Overview
The Cloudflare AI Agents Platform allows developers to create and manage AI agents using Cloudflare's Durable Objects. These agents are designed to be stateful, scalable, and efficient, with built-in support for real-time communication, scheduling, AI model calls, and more. Developers can easily create millions of agents for different use cases such as user sessions, game rooms, or workflows, with no cost incurred when agents are inactive. The platform offers a streamlined development experience with TypeScript and provides tools for both backend and frontend integration, including React hooks and a vanilla JavaScript client.

The platform supports a variety of features, such as persistent state, callable methods, scheduling, WebSockets, AI chat, workflows, email, SQL, and more. Developers can deploy agents using the `npm create cloudflare` CLI or integrate them into existing projects via `npm install agents`. The agents automatically hibernate when idle and wake up on demand, ensuring optimal performance and cost efficiency.

## Key Points
- Enables scalable, stateful AI agent development using Cloudflare Durable Objects.
- Offers built-in support for real-time communication, scheduling, AI model integration, and workflows.
- Supports persistent state that syncs across clients and survives restarts.
- Provides tools for frontend integration, including React hooks and a vanilla JavaScript client.
