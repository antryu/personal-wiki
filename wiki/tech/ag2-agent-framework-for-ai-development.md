# AG2 Agent Framework for AI Development

> AG2 is an open-source framework for building AI agents and enabling multi-agent collaboration to solve complex tasks. It evolved from AutoGen and is designed for both research and production use, with a strong community and active development.
> Last updated: 2026-05-04

## Overview

AG2 (formerly AutoGen) is an open-source framework for developing and researching AI agents. It supports the creation of autonomous agents capable of interacting with each other, using multiple large language models (LLMs), and integrating with external tools. The framework enables both autonomous workflows and human-in-the-loop systems, making it suitable for a wide range of AI applications.

AG2 is currently maintained by a group of volunteers from various organizations and is transitioning toward its v1.0 release. The new beta framework (`autogen.beta`) is expected to become the official version in the future. The project invites collaborators from all backgrounds to contribute and help shape the framework's future.

Key features of AG2 include:

- **Conversable agents** that can exchange messages and work together.
- **Multi-agent workflows** that can be orchestrated for complex tasks.
- **Integration with multiple LLMs** such as OpenAI, Anthropic, and more.
- **Tool support** for external APIs and data sources.
- **Human-in-the-loop functionality** to allow user interaction during agent workflows.

AG2 also provides extensive documentation, examples, and a growing community via Discord and GitHub.

## Key Points

- AG2 supports autonomous and human-in-the-loop AI agent workflows.
- The framework allows integration with various large language models and external tools.
- AG2 is transitioning to its v1.0 release, with the beta framework becoming the official version.
- The project is open-source under the Apache License 2.0 and is actively maintained by a community of contributors.
- AG2 offers detailed documentation and examples for developers and researchers.
