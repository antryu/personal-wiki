# Visual Studio Code

> Visual Studio Code is a popular open-source code editor developed by Microsoft, offering a lightweight yet powerful environment for software development. It combines advanced coding features with extensibility, making it a favorite among developers worldwide.  
> Last updated: 2026-04-16

## Overview
The **microsoft/vscode** repository hosts the open-source core of [Visual Studio Code](https://code.visualstudio.com), known as **Code - OSS**. This project is developed under the MIT License and serves as the foundation for the official Visual Studio Code distribution, which includes Microsoft-specific customizations under a separate license. Key details include:

- **Repository**:  
  - **Stars**: 183,927 | **Forks**: 39,236  
  - **Language**: TypeScript | **License**: MIT  
  - **Created**: 2015-09-03 | **Last Update**: 2026-04-16  
  - **Open Issues**: 16,387  

- **Features**:  
  - Lightweight debugging, intelligent code navigation, and multi-language support.  
  - Extensible via plugins and integrations with existing developer tools.  
  - Regular monthly updates and daily "Insiders" builds for early access.  

- **Distribution**:  
  - The **Code - OSS** repository is community-driven and open for contributions.  
  - The official **Visual Studio Code** distribution includes additional Microsoft branding and features, available for Windows, macOS, and Linux.  

- **Community & Development**:  
  - Active development with public roadmaps, iteration plans, and endgame strategies.  
  - Contributions welcome via bug reports, pull requests, and documentation improvements.  

## Key Points
- Open-source core under MIT License, with a widely adopted official distribution.  
- Cross-platform support (Windows, macOS, Linux) and regular feature updates.  
- Extensive community engagement, evidenced by 183k+ stars and 39k+ forks.  
- Actively maintained with transparent development workflows and public issue tracking.
