### Features
- Enforces widely accepted Swift style rules from guides like GitHub and Kodeco.  
- Built using **SwiftSyntax** for modern Swift analysis, with legacy Clang/SourceKit integrations for type information.  
- Supports multiple platforms and Swift versions (badges linked to [Swift Package Index](https://swiftpackageindex.com/realm/SwiftLint)).  
- Includes a video introduction and localized documentation (Chinese, Korean).  

### Installation
- **Swift Package Manager**:  
  Add `.package(url: "https://github.com/SimplyDanny/SwiftLintPlugins", from: "<version>")` to `Package.swift`.  
- **Homebrew**:  
  `brew install swiftlint`.  
- **Xcode**: Use the package dependency URL `https://github.com/SimplyDanny/SwiftLintPlugins`.
