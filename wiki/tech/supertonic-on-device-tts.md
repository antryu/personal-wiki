# Supertonic - On-Device TTS

> Supertonic is a fast, on-device text-to-speech system with minimal overhead. It operates natively via ONNX, offering high accuracy and support for multiple languages without cloud dependencies, ensuring user privacy and low latency.  
> Last updated: 2026-05-09

## Overview
Supertonic is a lightning-fast, on-device text-to-speech system that enables local inference with minimal overhead. It runs entirely on your device, leveraging ONNX Runtime, and is designed for use in environments where privacy and performance are critical. It supports multiple languages and offers tools like the Voice Builder to customize and deploy TTS models with edge-native compatibility.

The project has seen significant updates, including the release of **Supertonic 3**, which supports 31 languages and offers improved accuracy and fewer audio errors. Additionally, users can now generate audio in Python with a simple `pip install` and run the system across various platforms, including iOS, Android, and desktop environments, with support for multiple programming languages such as Swift, Go, Java, C#, and Python.

## Key Points
- Runs entirely on-device using ONNX Runtime for no cloud dependency and privacy preservation.
- Supports 31 languages with Supertonic 3 and includes optimized ONNX models.
- Includes a **Voice Builder** for turning custom voices into deployable TTS models.
- Offers SDKs in Python, Swift, Go, Java, C#, and more.
- Provides prebuilt models on Hugging Face and auto-download capabilities for easy setup.
