# Anthropic Agent Skills for Claude

> This repository provides a collection of skills for the Claude AI model, enhancing its ability to perform specialized tasks through dynamic loading of instructions, scripts, and resources. It showcases a wide range of skills from creative to technical domains and supports developers in building custom skills for Claude.  
> Last updated: 2026-04-17

## Overview

The **Anthropic Agent Skills** repository is a public resource offering a variety of skills for the Claude AI model. These skills enable Claude to perform specific tasks more effectively by incorporating domain-specific instructions, workflows, and tools. The repository includes both open-source and source-available skills, ranging from creative applications like art and design to technical tasks like web testing and document processing.

Each skill is self-contained in a folder, with a `SKILL.md` file that holds the instructions and metadata used by Claude. The repository also includes the specification for Agent Skills and a skill template for developers to build their own.

Skills can be used in **Claude Code**, **Claude.ai**, and the **Claude API**, with specific installation instructions for each platform. The repository serves as both a practical tool and an educational resource for developers and users looking to extend Claude's capabilities.

## Key Points
- Skills are dynamic modules that teach Claude how to perform specific tasks in a repeatable and customizable way.
- The repository includes a wide variety of skills across creative, technical, and enterprise domains.
- Skills can be used in Claude Code, Claude.ai, and the Claude API with detailed integration instructions.
- The repository contains both open-source and source-available skills, including document creation and editing capabilities.
- Skills are provided for demonstration and educational purposes, and should be tested thoroughly before use in production environments.
