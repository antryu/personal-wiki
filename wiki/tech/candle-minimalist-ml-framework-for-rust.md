# Candle: Minimalist ML Framework for Rust

> Candle is a minimalist machine learning framework for Rust, designed for performance and ease of use. It supports GPU acceleration and has been widely adopted since its creation in 2023. The framework allows users to run state-of-the-art models and includes online demos showcasing various ML tasks.  
> Last updated: 2026-05-05

## Overview

Candle is a Rust-based ML framework developed by Hugging Face. It is engineered to provide high performance and ease of integration for developers and researchers. The project is open-source under the Apache License 2.0 and has become popular due to its minimalistic design and powerful capabilities. Candle supports GPU and CUDA operations, allowing for efficient computation and model inference. It is used in various applications, including speech recognition, text generation, object detection, image segmentation, and more. The project also includes a collection of demos and command-line examples for popular models like LLaMA, Falcon, T5, and others.

Key features of Candle include its lightweight API, support for in-browser demos using WebAssembly (WASM), and a growing ecosystem of community-developed models. The project has an active development community, with over 20,000 stars and 1,500 forks on GitHub.

## Key Points

- **Minimalist design**: Candle focuses on simplicity and performance, avoiding unnecessary complexity.
- **GPU and CUDA support**: The framework enables fast computations using GPU acceleration, improving the performance of ML models.
- **Online demos**: Candle provides interactive in-browser demos for various ML tasks like speech recognition, object detection, and text generation.
- **WASM compatibility**: Many demos run entirely in the browser via WebAssembly, making them accessible without installing additional software.
- **Active community and development**: The project is actively maintained with a large number of contributors and an active issue tracker.
