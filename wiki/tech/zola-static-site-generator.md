# Zola Static Site Generator

> Zola is a fast, all-in-one static site generator written in Rust, notable for its single-binary design and built-in features for creating blogs, documentation sites, and more. It is widely used for its speed, simplicity, and rich feature set.  
> Last updated: 2026-05-05

## Overview

Zola is a modern static site generator originally known as Gutenberg. It is designed to be fast and efficient, offering a single binary with all required functionality built-in, eliminating the need for external tools or dependencies. The project was born out of the creator's dissatisfaction with complex and slow tools like Hugo. 

Zola supports a wide range of features including syntax highlighting, Sass compilation, multilingual site support, image processing, themes, shortcodes, and more. It also includes a powerful built-in template engine called [Tera](https://keats.github.io/tera/). Developers can easily deploy sites to platforms like Netlify, Vercel, or Cloudflare Pages. 

The tool is open source, with a dual licensing model: code added after version 0.22 is under the EU Public License 1.2, while older code remains under the MIT License. As of 2026, Zola has over 17,000 stars on GitHub and continues to be actively maintained.

## Key Points

- Single binary with all features included.
- Built-in support for syntax highlighting, Sass, and image processing.
- Multilingual site generation and custom taxonomies.
- Live reload and external link checking.
- Deployable to platforms like Netlify, Vercel, and Cloudflare Pages.
- Uses the Tera templating engine for flexible page layouts.
- Open source with a dual licensing model (EUPL-1.2 and MIT).
