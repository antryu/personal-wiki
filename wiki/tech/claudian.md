# Claudian

> Claudian is an Obsidian plugin that integrates AI coding agents such as Claude Code into your Obsidian vault, enabling seamless collaboration through file manipulation, inline editing, and advanced workflows. It is notable for its deep integration with AI tools, offering developers a powerful and intuitive environment for coding directly within Obsidian.
> Last updated: 2026-04-10

## Overview

Claudian is an Obsidian plugin designed to bring AI-powered coding assistance directly into your vault. It allows users to interact with AI coding agents like Claude Code and Codex, enabling file read/write, search, bash commands, and multi-step workflows without leaving Obsidian. The plugin features inline editing, slash commands, and integration with external tools via the Model Context Protocol (MCP). It supports multiple chat tabs, conversation history, and offers privacy-focused data handling.

Claudian requires the installation of the Claude Code CLI or compatible providers and is compatible with macOS, Linux, and Windows platforms. The plugin can be installed via GitHub releases, the BRAT plugin, or directly from source.

## Key Points

- **AI Integration**: Seamlessly integrates Claude Code and Codex into Obsidian for coding assistance.
- **Inline Editing**: Allows direct editing of notes with word-level diff preview using a hotkey.
- **Slash Commands & Skills**: Offers reusable prompt templates and Skills for enhanced productivity.
- **MCP Support**: Connects external tools via the Model Context Protocol for extended functionality.
- **Multi-Tab & Conversations**: Manages multiple chat tabs, conversation history, and allows forking and resuming conversations.
- **Privacy-Focused**: Limits data sent to the API to input, attached files, and tool outputs.
