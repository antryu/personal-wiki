# Intelligent Agent Tutorial

> A systematic open-source tutorial for building AI-native intelligent agents, bridging theoretical understanding with practical implementation. Notable for its comprehensive approach to transitioning from LLM users to agent system architects.
> Last updated: 2026-04-16

## Overview
**datawhalechina/hello-agents** is a 2025-launched project with 37.5k stars and 4.45k forks, offering a structured path to master AI-driven agent systems. The tutorial distinguishes between *software-engineering agents* (Dify/Coze/n8n) and *AI-native agents*, focusing on the latter's core principles, architecture, and implementation. Key features include:
- **Bilingual support** (Chinese/English)
- **Open-access learning** with online/offline reading options
- **Hands-on frameworks**: AutoGen, AgentScope, LangGraph, and a custom [HelloAgents](https://github.com/jjyaoao/helloagents) framework
- **Advanced topics**: Memory systems, context engineering, Agentic RL training, and multi-agent applications

The project tracks the 2025 "Agent Year" industry shift, providing practical projects like smart travel assistants and "cyber towns" while preparing learners for agent-related technical interviews.

## Key Points
- Systematic curriculum covering agent history, LLM foundations, and modern paradigms (ReAct, Plan-and-Solve)
- Combines low-code platform tutorials with framework development (LLM API integration, memory protocols)
- Open-source and community-driven with active maintenance (98 open issues as of 2026-04-16)
- Includes both theoretical foundations (transformers, prompt engineering) and applied techniques (GRPO training, reflection mechanisms)
