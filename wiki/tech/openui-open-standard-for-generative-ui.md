# OpenUI - Open Standard for Generative UI

> OpenUI is a full-stack Generative UI framework that provides a compact, streaming-first language, a React runtime with built-in component libraries, and ready-to-use chat interfaces. It is up to 67% more token-efficient than JSON and is notable for enabling developers to create interactive UIs directly from generative AI models.
> Last updated: 2026-05-05

## Overview

OpenUI is a full-stack framework designed to streamline the creation of Generative UIs. It offers a compact, streaming-first language known as **OpenUI Lang** and a React runtime that supports component libraries, making it significantly more efficient in token usage than JSON. OpenUI Lang allows developers to define UI components and generate prompt instructions directly from that component library, enabling the model to generate structured UI outputs as it streams.

The framework includes a variety of built-in UI components such as charts, forms, tables, and layouts, and supports prompt generation from these components. It also includes a streaming renderer that parses and renders the model’s output in real-time as tokens are received.

OpenUI is suitable for building assistants, copilots, and broader interactive product flows. Developers can quickly scaffold a new project using the CLI, which sets up a working example with OpenUI Lang support and streaming capabilities.

## Key Points

- OpenUI provides a compact language (OpenUI Lang) for structured UI generation with a focus on streaming output.
- It includes pre-built component libraries and prompt generation from those components.
- OpenUI is up to 67% more token-efficient than JSON.
- It supports real-time rendering of UIs as tokens arrive from the model.
- The framework includes a CLI for quick scaffolding and includes a working example to get started.
