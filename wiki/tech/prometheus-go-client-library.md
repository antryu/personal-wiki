# Prometheus Go Client Library

> This is the official Go client library for Prometheus, providing tools for instrumenting Go applications and interacting with the Prometheus HTTP API. Widely adopted and actively maintained, it plays a key role in monitoring systems built with Prometheus.
> Last updated: 2026-05-14

## Overview

The `prometheus/client_golang` repository is the official Go client library for Prometheus, enabling developers to instrument their applications and expose metrics for collection by Prometheus servers. It includes two main components: one for instrumenting Go applications and another for creating clients that communicate with the Prometheus HTTP API.

This library is used by many production systems for performance and health monitoring and is part of the CNCF ecosystem, which ensures high standards for quality and maintainability.

The library supports the two most recent major Go versions, and the team follows Semantic Versioning with some exceptions noted in the release policy. It is actively developed, with a focus on both stability and forward-compatible features.

## Key Points

- Provides a robust instrumentation library for exposing metrics from Go applications.
- Includes a client for querying the Prometheus HTTP API.
- Supports the latest two major versions of Go.
- Follows a versioning policy with exceptions for experimental APIs.
- Actively maintained with a large community and integration into CNCF Slack.
