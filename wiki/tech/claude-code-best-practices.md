# Claude Code Best Practices

> This repository provides best practices, patterns, and examples for leveraging the full potential of Claude Code, an AI-powered coding assistant. It's notable for its comprehensive approach to agentic engineering, including subagents, commands, and skills, and has become a top GitHub repository in its category.  
> Last updated: 2026-04-10

## Overview
The `claude-code-best-practice` repository is a community-driven resource that showcases how to effectively use Claude Code for development workflows. It includes structured documentation on subagents, commands, skills, and orchestration workflows, as well as practical examples and best practices. The repository is organized around key concepts and is maintained under the MIT License, with contributions and discussions encouraged through open issues and forks. It is designed to help developers integrate Claude Code into their coding routines, from simple prompt templates to complex agent-based systems.

Key components include:
- **Subagents**: Autonomous actors with isolated contexts, customizable tools, and persistent identities.
- **Commands**: User-invoked prompt templates for workflow orchestration.
- **Skills**: Configurable, preloadable knowledge units with context forking and progressive disclosure.
- **Workflows**: Structured sequences of commands and skills for common tasks.
- **Hooks**: User-defined event handlers that run outside the agentic loop.
- **MCP Servers**: Configuration for managing Claude Code server settings.
