# Web Data API for AI Agents

> Firecrawl is an open-source web data API that empowers AI agents with reliable, fast, and LLM-ready web data extraction. It handles complex tasks like JS-heavy pages, rotating proxies, and structured output formatting.  
> Last updated: 2026-04-12

## Overview  
**Firecrawl** is a production-grade API for AI agents to search, scrape, and interact with websites at scale. Hosted at [firecrawl.dev](https://firecrawl.dev), it offers:  
- **96% web coverage** with automatic handling of JavaScript-heavy pages  
- **3.4s P95 latency** for real-time agent workflows  
- **Clean markdown/JSON output** optimized for LLM input efficiency  
- **Zero-config infrastructure** (proxies, rate limiting, JS rendering)

### Core Features  
| Endpoint       | Description                              |
|----------------|------------------------------------------|
| **Search**     | Retrieve full-page content from search results |
| **Scrape**     | Convert URLs to markdown/JSON/screenshot data |
| **Interact**   | Execute AI-driven actions (clicks, scrolls) on pages |
| **Crawl/Map**  | Discover and scrape entire websites recursively |

## Key Points  
- Industry-leading reliability with automatic proxy rotation and JS rendering  
- Outputs structured data formats (markdown, JSON) to reduce LLM token usage  
- Agent-ready integration with MCP compatibility and batch processing
