# Telegram iOS Source Code

> The Telegram iOS Source Code repository provides the open-source codebase for the official Telegram iOS application. It allows developers to build, modify, and distribute their own versions of the app, while adhering to the project's licensing and branding guidelines.  
> Last updated: 2026-05-04

## Overview

The repository contains the source code for the Telegram iOS app, built primarily in Swift. It is actively maintained and updated, with the latest commit on May 4, 2026. The project includes detailed instructions for compiling and running the app, covering both basic and advanced setup scenarios. Developers are required to follow specific guidelines, such as creating their own app identifiers and ensuring user data privacy, to maintain compliance with Telegram’s policies.

The repository provides a comprehensive README with a step-by-step compilation guide, including setup for Xcode, configuration for code signing, and building for both development and distribution. It also includes a frequently asked questions (FAQ) section and tips for common issues, such as build failures after system restarts and simulator-only builds that do not require code signing.

## Key Points
- Requires developers to create their own app identifiers and follow branding and licensing guidelines.
- Offers detailed documentation for compiling the app for development and distribution.
- Uses Swift as the primary programming language.
- Includes advanced build options and troubleshooting tips.
