# Capacitor: Cross-Platform App Development Framework

> Capacitor enables developers to build cross-platform native progressive web apps (PWAs) for iOS, Android, and the web using a single web-based codebase. It offers a modern, extensible plugin architecture and first-class support for deploying to app stores and the mobile web.  
> Last updated: 2026-05-05

## Overview

Capacitor is a cross-platform app development framework maintained by the Ionic Team. It allows developers to write applications using modern web technologies like JavaScript, HTML, and CSS, and deploy them natively across iOS, Android, and the web. The framework supports the use of native device features via a plugin API and offers a more modern and flexible alternative to Cordova.

Capacitor integrates directly into any existing modern web application, making it easy to adopt without needing to rewrite an entire codebase. It supports writing custom native plugins using Swift for iOS and Kotlin or Java for Android, with the ability to package plugins for the broader community via npm.

Key components of the framework include:

- **Core Framework**: Enables native app execution and provides access to device APIs.
- **Plugin System**: Developers can build and share plugins, either as part of the app or as shared npm packages.
- **Tooling**: Offers CLI tools for initializing projects, adding native platforms, and syncing assets.
- **Progressive Web App Support**: Facilitates deployment both as native apps and as web apps.

The framework is designed to work seamlessly with the [Ionic Framework](https://ionicframework.com/), although it is not required for use.

---

## Key Points

- Capacitor allows a single web-based codebase to run natively on iOS, Android, and the web.
- It provides access to native device features via a plugin API.
- Plugins can be written in Swift (iOS) and Kotlin/Java (Android) and shared via npm.
- Capacitor is a modern and flexible alternative to Cordova with better tooling and support.
- It supports Progressive Web App (PWA) deployment alongside native app stores.
- The framework is open-source and licensed under the MIT License.
