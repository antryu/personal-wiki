# Clipy: Clipboard Extension for macOS

> Clipy is a clipboard extension app for macOS that enhances the clipboard experience by storing and managing copied content. Notable for its active development and community support, it provides developers and users an intuitive and efficient clipboard manager with a customizable interface.  
> Last updated: 2026-05-13

## Overview

Clipy is an open-source clipboard manager for macOS that extends the functionality of the system clipboard by storing multiple clipboard entries for quick access. It allows users to manage and retrieve copied text, images, and other content effortlessly.

The project, hosted on GitHub, has gained significant traction since its creation in 2015 and has been continuously updated, most recently on 2026-05-13. Clipy is written in Swift and is available under the MIT License, encouraging open contributions and forks.

The app requires macOS 10.10 Yosemite or higher and is distributed through the official website, [Clipy-App.com](https://clipy-app.com). It is currently under active development with a development environment based on macOS 10.15 Catalina, Xcode 12.2, and Swift 5.3.

Contributions are welcome, and the project provides detailed guidelines for contributing. Clipy also supports community translations and has a system in place for localizing the app. Backers and sponsors are acknowledged and supported through OpenCollective.

## Key Points
- Clipy enhances macOS clipboard functionality by storing multiple clipboard entries.
- It is built with Swift and is actively maintained under the MIT License.
- The project has a strong community with 8,519 stars and 782 forks on GitHub.
- It requires macOS 10.10 Yosemite or later to run.
- Open source and community-driven, with an active development team.
