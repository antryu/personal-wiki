# HyperFrames: HTML to Video Rendering Framework

> HyperFrames is an open-source video rendering framework that allows users to create, preview, and render HTML-based video compositions with first-class support for AI agents. Built for developers and AI tools alike, it provides an intuitive way to transform static HTML into dynamic videos using AI-powered coding agents.
> Last updated: 2026-05-11

## Overview
HyperFrames is designed to streamline the process of video rendering by letting users write HTML and see it transformed into videos. The framework supports AI coding agents such as Claude Code, Cursor, Gemini CLI, and Codex, empowering them to create video compositions, manage styles, and handle animations. HyperFrames includes features like Tailwind v4 browser-runtime styles and first-party adapter animations, making it a robust solution for developers and content creators alike.

HyperFrames is open source and distributed under the Apache License 2.0. It is actively developed and maintained, with the latest update as of 2026-05-11. The framework is built using TypeScript and is compatible with Node.js version 22 or higher. It has gained popularity with over 17,000 stars and 1,600 forks on GitHub.

## Key Points
- HyperFrames allows users to create video compositions by writing HTML, making the process of video rendering more accessible and efficient.
- The framework is built for and fully supports AI agents, enabling seamless collaboration between humans and AI tools.
- It provides integration with tools such as GSAP for timeline animations, Tailwind v4 for styling, and various animation runtimes like Lottie and Three.js.
- HyperFrames includes a suite of developer tools like linting, rendering, asset preprocessing (e.g., TTS and background removal), and a CLI for streamlining the workflow.
- The project is actively maintained, with a GitHub repository that hosts open issues, guides, and plugins for AI agents such as Claude Code, Cursor, and Codex.
