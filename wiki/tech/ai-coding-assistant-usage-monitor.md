# AI Coding Assistant Usage Monitor

> A macOS menu bar application that tracks AI coding assistant usage quotas for Claude, Codex, Antigravity, and Gemini. Notable for its real-time visibility into API token limits, helping developers optimize AI tool usage efficiency.  
> Last updated: 2026-04-11

## Overview
**ClaudeBar** is a Swift-based macOS utility that provides real-time monitoring of AI coding assistant API quotas through a streamlined menu bar interface. The application supports major platforms including Anthropic's Claude, GitHub's Copilot (Codex), and Google's Gemini, displaying current usage statistics, remaining tokens, and rate limits. Features include customizable alert thresholds, historical usage trends, and automatic token refresh tracking. The project has gained significant traction with 932 stars and 19 new stars in a single day, reflecting strong developer interest in AI tool transparency.

## Key Points
- Real-time monitoring of Claude, Codex, Antigravity, and Gemini API quotas  
- macOS menu bar interface for instant access to usage statistics  
- Tracks tokens consumed, rate limits, and historical usage patterns  
- Built with Swift for native macOS performance and integration  
- Community-driven development with rapid adoption (932 stars, +19 today)
