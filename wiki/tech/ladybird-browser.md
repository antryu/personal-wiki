# Ladybird Browser

> Ladybird is an independent web browser built with a novel engine based on web standards. It is designed for robustness and security, with a multi-process architecture, and is currently in a pre-alpha state.
> Last updated: 2026-05-06

## Overview
Ladybird is a pre-alpha stage web browser developed using a custom engine that adheres to modern web standards. It utilizes a multi-process architecture to isolate tasks such as image decoding, network operations, and rendering within separate processes, enhancing security and robustness. It is primarily aimed at developers and is built using a collection of core libraries inherited from the SerenityOS project. These include components like the Web rendering engine (LibWeb), JavaScript engine (LibJS), and WebAssembly support (LibWasm), among others.

The browser is open-source and licensed under the BSD 2-Clause "Simplified" License. It is actively developed on GitHub, and documentation and build instructions are available for developers looking to contribute or test the software. Ladybird runs on Linux, macOS, Windows (with WSL2), and various other \*Nix systems.

## Key Points
- Built on a novel engine based on web standards.
- Multi-process architecture improves robustness against malicious content.
- Uses core libraries inherited from SerenityOS.
- Open-source and BSD 2-Clause licensed.
- Currently in a pre-alpha state, not suitable for general users.
