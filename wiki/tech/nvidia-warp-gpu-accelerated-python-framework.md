# NVIDIA Warp: GPU-Accelerated Python Framework

> NVIDIA Warp is a Python framework for GPU-accelerated simulation, robotics, and machine learning. It enables developers to JIT compile Python functions to efficient kernel code that can run on CPU or GPU, providing high-performance computing capabilities for physics simulations and differentiable programming.
> Last updated: 2026-05-16

## Overview

NVIDIA Warp is a powerful framework that allows developers to write Python code and compile it efficiently for both CPU and GPU execution. It is especially useful for applications in simulation, robotics, and machine learning, where performance and scalability are critical. Warp supports differentiability, making it compatible with machine learning frameworks such as PyTorch, JAX, and Paddle. The framework is designed for developers who want to leverage GPU acceleration without delving into low-level CUDA programming.

Warp is open-source, available under the Apache License 2.0, and can be installed via PyPI for easy integration. It supports Windows, Linux, and macOS (including Apple Silicon), and requires a CUDA-capable NVIDIA GPU for GPU acceleration.

Key features include:

- Just-in-time (JIT) compilation for CPU/GPU execution.
- Extensive support for physics simulations, robotics, and geometry processing.
- Integration with machine learning frameworks via differentiable kernels.
- Comprehensive tutorials and documentation.
- Active development and maintenance since March 2022.

## Key Points
- NVIDIA Warp accelerates computation by JIT compiling Python functions to run on CPU or GPU.
- It supports differentiable kernels, enabling integration with machine learning frameworks.
- The framework is useful for robotics, machine learning, and physics-based simulations.
- It offers extensive documentation and tutorials for developers.
