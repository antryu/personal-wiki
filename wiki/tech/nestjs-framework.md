# NestJS Framework

> NestJS is a progressive Node.js framework that enables developers to build efficient, scalable, and enterprise-grade server-side applications using TypeScript or JavaScript. It is notable for combining best practices from various programming paradigms and offering a structured architecture for maintaining complex applications.
> Last updated: 2026-05-06

## Overview

The **NestJS** framework, maintained by the `nestjs/nest` repository on GitHub, is built using modern JavaScript with full support for TypeScript. It aims to provide a robust and structured approach to server-side development by leveraging the power of OOP (Object-Oriented Programming), FP (Functional Programming), and FRP (Functional Reactive Programming). NestJS uses Express under the hood but also supports other libraries such as Fastify.
