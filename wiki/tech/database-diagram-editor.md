# Database Diagram Editor

> A web-based, open-source database diagram editor that visualizes and designs databases using a single query, eliminating the need for installations or database passwords. Notable for its AI-powered migration tools and support for multiple database systems.  
> Last updated: 2026-04-14

## Overview
The **Database Diagram Editor** (ChartDB) is a web-based tool for creating and managing database schemas. Hosted on GitHub, it allows users to generate diagrams instantly via a "Smart Query," customize visualizations, and export SQL scripts for migrations. Key features include AI-driven DDL generation, interactive editing, and compatibility with major databases like PostgreSQL, MySQL, SQL Server, and SQLite. The project is in public beta, licensed under AGPLv3, and built with TypeScript and React.

The repository has 21.8k stars and 1.35k forks, indicating strong community engagement. It supports 12+ database systems, including Supabase, Timescale, and Cloudflare D1. Users can access a cloud demo or deploy the tool locally using Docker or npm. The project emphasizes no-password access and seamless schema visualization for documentation, team collaboration, or data understanding.

## Key Points
- **Instant Schema Import**: Retrieve database schemas as JSON with a single query for rapid visualization.
- **AI-Powered Migration**: Generate DDL scripts tailored to target databases (e.g., MySQL → PostgreSQL).
- **Interactive Editing**: Modify schemas and add annotations directly in the editor.
- **Public Beta**: Actively developed with 149 open issues for community contribution.
