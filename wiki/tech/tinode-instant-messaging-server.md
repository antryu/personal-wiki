# Tinode Instant Messaging Server

> Tinode is an open-source instant messaging platform that aims to deliver on the original vision of XMPP by providing a modern, decentralized, and federated messaging solution. It is notable for its robust architecture, support for multiple clients, and emphasis on privacy and interoperability.  
> Last updated: 2026-04-15

## Overview

Tinode is an instant messaging platform with a backend written in Go and a suite of clients including Android (Java), iOS (Swift), and web (ReactJS). It also supports gRPC clients in multiple languages such as C++, C#, Go, Java, Node, PHP, Python, Ruby, and Objective-C. Tinode uses JSON over WebSocket (or long polling) and Protobuf with gRPC for communication.

The project is designed to be a modern replacement for XMPP, aiming to enable a federated messaging ecosystem where users can communicate across different servers. It supports advanced features like video calling, chatbots, and web-based access. Tinode's backend is licensed under the GNU General Public License v3.0, while the clients are under Apache 2.0.

The project began in 2014 and has gained significant traction, with over 13,000 stars and 2,000 forks on GitHub. It's actively maintained, with the latest update on April 15, 2026.

Tinode is not intended to replace collaboration tools like Slack but to provide a decentralized and privacy-focused messaging infrastructure. It is not compatible with XMPP, but it seeks to address many of the shortcomings of existing messaging systems by offering a flexible and extensible API.

Installation instructions are available in the [INSTALL.md](INSTALL.md) file, and Docker-specific setup is also provided. A public web-based service is available at [https://web.tinode.co/](https://web.tinode.co/).

## Key Points

- Tinode is a modern, open-source instant messaging server and client platform.
- It supports multiple clients including Android, iOS, web, and gRPC clients in various languages.
- Tinode aims to replace XMPP with a more robust and federated messaging solution.
- The platform emphasizes privacy, decentralization, and extensibility.
- It supports advanced features like video calling, chatbots, and web-based access.
- Tinode is actively developed with a large and active GitHub community.
