# TanStack Router: A Type-Safe Full-Stack Web Framework

> TanStack Router is a modern, type-safe routing framework and full-stack web development tool designed for React and other JavaScript environments. It emphasizes client-first design, server capabilities, and provides seamless developer experiences through data-driven navigation and robust type inference. It's notable for its end-to-end type safety and integration with tools like TanStack Start for full-stack capabilities.
> Last updated: 2026-05-13

## Overview
TanStack Router is a powerful and modern framework that allows developers to build type-safe, client-first, and server-capable web applications. Initially created in 2019, it has become a go-to solution for developers seeking robust routing and data-handling capabilities. The project is hosted on GitHub, where it has amassed over 14,000 stars and 1,680 forks, and is currently maintained under the MIT License.

At its core, TanStack Router offers features such as end-to-end type safety, schema-driven search parameters, and built-in support for caching, prefetching, and invalidation. It supports nested layouts, transitions, and error boundaries, which make it highly flexible for complex web applications.

TanStack Router is part of a larger ecosystem that includes TanStack Start, a full-stack framework built on top of Router. TanStack Start allows for full-document SSR (Server-Side Rendering), server functions, and production-ready deployments, providing a comprehensive solution for full-stack development.

The project has an active community, with 589 open issues and dedicated channels for collaboration, such as GitHub Discussions and a Discord server. Developers are encouraged to contribute, and detailed instructions are provided in the [CONTRIBUTING.md](./CONTRIBUTING.md) file.

## Key Points
- End-to-end type safety for routes, parameters, and loaders.
- Schema-driven search parameters with validation.
- Built-in caching and prefetching for optimized performance.
- Support for nested layouts and error boundaries.
- Full-stack development with TanStack Start for SSR and server-side capabilities.
- Active and growing community with extensive documentation and support.
