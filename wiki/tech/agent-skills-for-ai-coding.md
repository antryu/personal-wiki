# Agent Skills for AI Coding

> This repository offers production-grade engineering skills for AI coding agents to consistently follow senior engineer best practices across the software development lifecycle. It's notable for its structured workflows and command-driven approach to ensure code quality and consistency.
> Last updated: 2026-05-06

## Overview
The **Agent Skills** project is designed to encode the workflows, quality gates, and best practices typically used by senior engineers and apply them systematically across AI coding agents. Each skill is organized around phases of the development lifecycle: define, plan, build, verify, review, and ship. Skills can be activated either via seven slash commands (`/spec`, `/plan`, `/build`, `/test`, `/review`, `/code-simplify`, `/ship`) or automatically based on the task at hand, such as API design or UI development.

The project supports integration with a wide range of AI and code tools, including Claude Code, Cursor, Gemini CLI, Windsurf, OpenCode, GitHub Copilot, and Kiro IDE & CLI. It is licensed under MIT and has been created and maintained by Addy Osmani. The repository includes setup instructions tailored for each supported platform.

## Key Points
- Encodes senior engineering best practices into workflows for AI coding agents.
- Uses slash commands to trigger specific development lifecycle stages.
- Skills are automatically applied based on context (e.g., API design, frontend work).
- Supports integration with tools like Cursor, Gemini CLI, GitHub Copilot, and more.
- Maintains high quality with 30k+ stars and active development.
