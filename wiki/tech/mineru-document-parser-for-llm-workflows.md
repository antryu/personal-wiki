# MinerU: Document Parser for LLM Workflows

> MinerU transforms complex documents like PDFs and Office files into structured markdown/JSON, making them ready for integration with LLM workflows. It is notable for its high performance and versatility across various document formats.
> Last updated: 2026-05-13

## Overview
MinerU is a powerful document parser designed to extract and convert content from various document formats, including PDFs, DOCX, XLSX, and PPTX, into LLM-ready markdown or JSON. It supports advanced layout analysis and OCR for scanned documents, making it ideal for agentic and RAG (Retrieval-Augmented Generation) workflows. The repository has gained significant traction, with over 62,900 stars and 5,298 forks.

Key features include:
- **High accuracy** in layout parsing and content extraction
- **Support for multiple document types**, including scanned PDFs
- **Integration with LLM pipelines**, enabling efficient data preprocessing
- **Open-source framework** with a permissive license and active community

## Key Points
- High-performance parsing of complex documents into structured data.
- Supports multiple file formats including PDF, DOCX, XLSX, and PPTX.
- Includes OCR and layout analysis for scanned or multi-layout documents.
- Designed for seamless integration into LLM and agentic workflows.
- Actively maintained and widely used, with a strong open-source community.
