# Pulumi: Infrastructure as Code in Any Language

> Pulumi enables infrastructure provisioning using any modern programming language, making infrastructure as code (IaC) more accessible and flexible. With over 25,000 stars and active development, it stands out for its language-agnostic approach and robust ecosystem.  
> Last updated: 2026-04-10

## Overview
- **Repository**: [pulumi/pulumi](https://github.com/pulumi/pulumi)  
- **Description**: A tool for defining cloud infrastructure using general-purpose programming languages (e.g., Python, JavaScript, TypeScript, Go).  
- **Key Stats**:  
  - Language: Go  
  - Stars: 25,016 (↑ +33 today)  
  - URL: https://github.com/pulumi/pulumi  

## Key Points
- Supports infrastructure as code in **any programming language**, allowing developers to use their preferred tools and workflows.  
- Actively maintained with a large community, making it a reliable choice for cloud infrastructure management.
