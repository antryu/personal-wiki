# Harbor Container Registry

> Harbor is an open-source, CNCF-hosted cloud-native registry for storing, signing, and scanning container images and Helm charts. It provides security, replication, and access control features essential for enterprise environments.  
> Last updated: 2026-04-10

## Overview  
Harbor extends the open-source Docker Distribution project with critical functionalities like role-based access control, policy-driven replication, and vulnerability scanning. Hosted by the Cloud Native Computing Foundation (CNCF), it supports cloud-native workflows by enabling efficient image transfer between build and runtime environments. Key repository details include:

- **Stars**: 28,223 | **Forks**: 5,188

- **Topics**: cloud-native, container-registry, Kubernetes, Helm, CNCF

The project emphasizes stability through regular releases (avoiding direct use of the `main` branch) and maintains community engagement via biweekly meetings. It integrates with CI/CD pipelines and supports advanced security practices like activity auditing and artifact scanning.

## Key Points  
- **Cloud-Native Registry**: Supports container images and Helm charts, compatible with Kubernetes and container orchestration platforms.  
- **Security & Compliance**: Includes vulnerability scanning, digital signing, and role-based access control for granular permissions.  
- **Replication & High Availability**: Enables policy-based replication between registries for load balancing, disaster recovery, and multi-datacenter workflows.  
- **CNCF Hosting**: Maintained under the CNCF umbrella, ensuring alignment with cloud-native standards and community-driven development.
