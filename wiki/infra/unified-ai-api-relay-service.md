# Unified AI API Relay Service

> Sub2API-CRS2 是一站式开源中转服务，统一接入 Claude、Openai、Gemini、Antigravity 等订阅接口，通过拼车共享机制高效分摊成本，同时保持原生工具无缝兼容性。该项目以 Go 语言实现，近期单日新增 267 星标，成为多模型 API 接入领域的高效解决方案。  
> Last updated: 2026-04-10

## Overview
该项目通过 API 中转服务整合 Claude、Openai、Gemini 和 Antigravity 等主流 AI 模型的订阅接口，提供统一的接入层。核心特性包括：
- **多模型统一接入**：支持 Claude、Openai、Gemini、Antigravity 等多个 AI 平台
- **拼车共享机制**：通过流量聚合降低单用户成本，实现资源高效利用
- **原生工具兼容**：保持与现有开发工具链的无缝对接，无需额外适配
- **开源架构**：采用 Go 语言实现，提供高并发处理能力和灵活扩展性

## Key Points
- 支持 Claude、Openai、Gemini、Antigravity 等多平台订阅统一接入
- 拼车共享机制显著降低 API 调用成本
- 开源设计（Go 语言）保障可维护性与扩展性
- 原生工具兼容避免二次开发成本
