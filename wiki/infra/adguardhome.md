# AdGuardHome

> A free and open-source network-wide DNS server that blocks ads and trackers across all devices, offering privacy protection without requiring client-side software. Notable for its high adoption (33.5k+ GitHub stars) and support for modern protocols like DNS-over-HTTPS and DNS-over-TLS.  
> Last updated: 2026-04-13

## Overview
AdGuardHome is a privacy-focused DNS server designed to filter ads, trackers, and malicious content at the network level. It operates by redirecting unwanted domains to a "black hole," ensuring all devices on a network benefit from filtering without requiring individual configuration. Key features include:

- **Multi-protocol support**: DNS-over-HTTPS (DoH), DNS-over-TLS (DoT), DNS-over-QUIC, and DNSCrypt.
- **Cross-platform compatibility**: Runs on Linux, macOS, FreeBSD, and OpenBSD.
- **Extensive filtering**: Uses shared codebase with AdGuard's public DNS service for real-time updates.
- **User-friendly deployment**: Offers automated installation scripts, Docker images, and Snap packages.

**Repository Highlights**:

## Key Points
- Blocks ads and trackers **network-wide**, eliminating the need for client-side software.
- Implements advanced DNS protocols (e.g., DoH, DoT) for enhanced security and privacy.
- Actively maintained with regular updates and a robust contributor community.
- Integrates with popular platforms via Docker, Snap, and automated installers.
