# Node Exporter

> A widely-used machine metrics exporter for Prometheus, enabling hardware and OS monitoring at scale. Last updated: 2026-04-16

## Overview
The `prometheus/node_exporter` repository provides a critical tool for collecting and exposing system-level metrics (CPU, memory, disk, etc.) to Prometheus. Key details include:
- **Repository**: [GitHub link](https://github.com/prometheus/node_exporter)
- **Stars**: 13,326 (with +5 stars added today)
- **Language**: Built in Go for cross-platform reliability and performance
- **Ecosystem**: Core component of the Prometheus monitoring stack

## Key Points
- Collects hardware, OS, and kernel metrics for Linux, Windows, and macOS
- Actively maintained with daily updates and community contributions
- Lightweight design enables deployment on thousands of nodes
- Essential for infrastructure monitoring in cloud-native environments
