# Goose Database Migration Tool

> Goose is a database migration tool that supports SQL migrations and Go functions, enabling developers to manage database schema changes effectively. It is notable for its flexibility across multiple databases and its ability to simplify migration workflows.
> Last updated: 2026-04-16

## Overview
Goose is a versatile database migration tool that functions both as a command-line interface (CLI) and a library. It allows developers to manage database schema changes by creating incremental SQL changes or Go functions. This makes it ideal for managing schema evolution in applications that require frequent database updates.

Goose supports multiple databases, including Postgres, MySQL, SQLite, and more, and it allows for out-of-order migrations, embedded migrations, and environment variable substitution. It is also highly customizable, enabling users to build a lightweight version by excluding unnecessary drivers.

## Key Points
- Supports a wide range of databases, including Postgres, MySQL, SQLite, and more.
- Allows for both SQL and Go-based migrations.
- Offers out-of-order migrations and embedded migrations.
- Provides environment variable substitution in SQL migrations.
- Highly customizable with the ability to build a lightweight version.
