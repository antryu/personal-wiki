# AI Model Gateway API

> A unified AI model hub for aggregation & distribution, enabling cross-conversion of various LLMs into OpenAI-compatible, Claude-compatible, or Gemini-compatible formats. Notable for its high adoption (26.7k+ stars) and enterprise-grade model management capabilities.  
> Last updated: 2026-04-14

## Overview
The **QuantumNous/new-api** project provides a next-generation LLM gateway and AI asset management system. It serves as a centralized platform for converting, deploying, and managing AI models across multiple frameworks while ensuring compliance with OpenAI's terms and regional regulations. Key features include:

- **Cross-format compatibility**: Converts models to OpenAI, Claude, and Gemini APIs  
- **Enterprise model management**: Centralized control for personal and organizational use  
- **Open-source infrastructure**: Built with Go under the GNU AGPLv3 license  
- **Active community**: 5.5k+ forks and 612 open issues for collaborative development  

The project includes deployment badges for Docker, GoReportCard, and version tracking, with multilingual documentation (English, Chinese, French, Japanese). It also integrates with cloud providers like Alibaba Cloud and UCloud through trusted partnerships.

## Key Points
- Supports conversion of LLMs to three major API formats (OpenAI, Claude, Gemini)  
- Centralized management system for AI models with enterprise-grade scalability  
- Compliant with OpenAI's Terms of Use and China's Generative AI Service Regulations  
- High adoption metrics: 26.7k stars, 5.4k forks, and active maintenance since 2023
