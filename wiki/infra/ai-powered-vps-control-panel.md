# AI-Powered VPS Control Panel

> 1Panel is a modern, open-source VPS control panel with native AI agent support, enabling users to run Ollama models, deploy OpenClaw agents, and manage server stacks through a unified web interface. With over 34,919 stars and 2 million+ global users, it offers a modern alternative to traditional panels like cPanel and Plesk.  
> Last updated: 2026-04-11

## Overview
1Panel is a Go-based, GPLv3-licensed open-source VPS control panel designed for developers and sysadmins. It stands out by integrating AI capabilities natively, allowing users to deploy large language models (LLMs) and AI agents directly within the dashboard. The platform supports one-click app deployment, Docker management, and multilingual documentation (14+ languages). Key highlights include:
- **Repository stats**: 34,919 stars, 3,088 forks, 367 open issues
- **User trust**: 2,000,000+ active self-hosters
- **Active ecosystem**: Weekly updates since 2022, Discord community (1.3M+ members)
- **Comparative advantages**: Modern UI, free/open-source, and AI-native features versus proprietary solutions

## Key Points
- Native AI runtime for Ollama models and OpenClaw agents
- One-click deployment for 165+ apps with automatic SSL and domain binding
- Full Docker/container management integrated into the web interface
- Multilingual support (English, Chinese, Japanese, Portuguese, Arabic, etc.)
- Actively maintained with 2026-04-11 as the latest update
