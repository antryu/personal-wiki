# Harbor - Cloud Native Container Registry

> Harbor is an open-source, CNCF-hosted container registry that stores, signs, and scans content, offering cloud-native features like security, access control, and replication. It is widely adopted for its extensibility and integration with container ecosystems.  
> Last updated: 2026-04-10

Harbor extends the open-source Docker Distribution with critical features like **security**, **identity management**, and **replication**. It supports container images and Helm charts, enabling efficient image transfer in cloud-native environments. Key capabilities include:
- **Role-based access control** for repositories and projects.
- **Policy-driven replication** across registries for high availability and multi-datacenter workflows.
- **Advanced security**: Vulnerability scanning, user management, and activity auditing.

> **Note**: The `main` branch may be unstable. Use [releases](https://github.com/goharbor/harbor/releases) for production.  

Community meetings occur bi-weekly, with schedules and recordings available in the [community repository](https://github.com/goharbor/community/blob/master/MEETING_SCHEDULE.md).

## Key Points
- Hosted by the **Cloud Native Computing Foundation (CNCF)** since 2018.
- Supports **OCI conformance** and integrates with Kubernetes, Helm, and container orchestration platforms.
- Provides **artifact scanning** for vulnerabilities and compliance.
