# VictoriaMetrics: Monitoring & Time Series DB

> VictoriaMetrics is a fast, cost-effective, and scalable monitoring solution and time series database designed for high performance and reliability. It supports single-node and cluster deployments and is widely adopted by companies like Grammarly, Roblox, and Spotify.  
> Last updated: 2026-04-16

## Overview
VictoriaMetrics offers a robust platform for managing time series data, combining monitoring capabilities with a time series database (TSDB). Key features include compatibility with Prometheus Query Language (PromQL), integration with observability tools like Grafana and OpenTelemetry, and support for Kubernetes environments. The project provides both single-node and cluster deployment options, making it adaptable to diverse use cases. It is licensed under the Apache License 2.0 and built in Go for performance optimization.
