# NVIDIA NCX Infra Controller for AI Cloud Infrastructure

> The NVIDIA NCX Infra Controller (NICo) automates zero-touch lifecycle management for bare-metal systems in AI cloud infrastructure, leveraging DPU-enforced isolation and multitenant networking to secure datacenter foundations.  
> Last updated: 2026-04-16

## Overview
The **NVIDIA NCX Infra Controller** (NICo) is an API-based microservice designed for site-local, zero-trust management of bare-metal hardware lifecycle operations. It enables secure, scalable deployment of AI cloud infrastructure by automating provisioning, configuration, and isolation of systems using Data Processing Units (DPUs).  

Key repository details include:  
- **Repository**: [GitHub](https://github.com/NVIDIA/ncx-infra-controller-core)  
- **Stars/Forks**: 122 / 80  
- **Language**: Rust  
- **License**: Apache License 2.0  
- **Status**: Experimental (preview release)  

The project provides documentation for architecture overviews, site setup, container building, and local development using DevSpace with mock systems. Users are advised to test thoroughly in non-production environments due to potential API/config changes in future releases.

## Key Points
- **Zero-touch automation**: Streamlines bare-metal lifecycle management for AI cloud infrastructure.  
- **DPU-enforced isolation**: Ensures zero-trust security with hardware-level multitenant networking.  
- **API-driven microservice**: Enables programmatic control and integration with cloud orchestration tools.  
- **Experimental status**: Preview release with potential breaking changes; not recommended for production without testing.
