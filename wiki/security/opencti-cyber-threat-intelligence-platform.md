# OpenCTI Cyber Threat Intelligence Platform

> OpenCTI is an open source cyber threat intelligence platform designed to help organizations manage and analyze cyber threat knowledge. It supports structured data management using STIX2 standards and integrates with tools like MITRE ATT&CK and MISP, making it a powerful solution for cybersecurity professionals.
> Last updated: 2026-05-14

## Overview

OpenCTI is a comprehensive platform for managing cyber threat intelligence, allowing users to store, structure, and visualize both technical and non-technical cyber threat information. It uses the STIX2 standard for data structuring and features a modern web interface powered by a GraphQL API. The platform is open source and offers a Community Edition under the Apache 2.0 license and an Enterprise Edition with enhanced features.

Key features include:
- A robust knowledge management system with links between threat data and their sources.
- Support for MITRE ATT&CK framework integration.
- Automation of import and export functions for formats such as CSV and STIX2 bundles.
- Extensive connectivity via connectors to integrate with other platforms and tools.

The platform also supports community collaboration and has a large number of contributors and active discussions through its GitHub repository, Slack, and other community channels.

## Key Points
- OpenCTI is built using TypeScript and offers a GraphQL-based API for flexible integration.
- It supports both technical and non-technical threat data, with features for source attribution, first and last seen timestamps, and confidence levels.
- The platform includes a modular architecture with connectors for integration with tools like MISP, TheHive, and MITRE ATT&CK.
- It is available in two editions: Community Edition (open source) and Enterprise Edition (commercial with advanced features).
