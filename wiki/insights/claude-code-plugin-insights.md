# Claude Code Plugin Insights

> This repository automatically collects and curates adoption metrics for Claude Code plugins across GitHub repositories using n8n workflows. It provides a ranked list of top repositories using these plugins, offering insights into current trends and popular tools in the AI coding ecosystem.
> Last updated: 2026-04-16

## Overview
The **quemsah/awesome-claude-plugins** repository maintains a dynamic list of the top 100 GitHub repositories that utilize Claude Code plugins. It leverages automated workflows to index and rank these repositories based on metrics like stars, subscribers, and the number of plugins implemented. The project is built with TypeScript and uses n8n for automation. The data is updated regularly and includes repositories ranging from large-scale frameworks like `next.js` to specialized tools like `mempalace`. The goal is to provide developers and researchers with a centralized, up-to-date view of the most impactful and widely adopted Claude Code plugin integrations.

## Key Points
- Automatically indexes over 12,000 repositories to track Claude Code plugin usage.
- Maintains a real-time list of the top 100 repositories, sorted by stars and plugin count.
- Shows growing adoption of Claude Code plugins in major development frameworks and AI agent tools.
- Highlights projects like `agents` and `antigravity-awesome-skills`, which have a high number of plugins and integrations.
- Offers insights into emerging trends in agentic development and AI-assisted coding workflows.
