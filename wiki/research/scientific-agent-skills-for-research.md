# Scientific Agent Skills for Research

> A repository providing 134 ready-to-use AI agent skills for scientific research across biology, chemistry, medicine, and engineering. Notable for its broad compatibility with AI agents and integration with 100+ scientific databases.  
> Last updated: 2026-04-11

## Overview
The **K-Dense-AI/scientific-agent-skills** repository offers a standardized toolkit of skills for AI agents to execute complex scientific workflows. Originally developed for Claude, it now supports any AI agent adhering to the open [Agent Skills](https://agentskills.io/) standard. Key features include:

- **134 curated skills** spanning bioinformatics, cheminformatics, proteomics, clinical research, materials science, and more.
- Integration with **100+ scientific databases** (e.g., GenBank, PubChem, PDB) and specialized tools for tasks like RNA-seq analysis, molecular docking, and geospatial modeling.
- **K-Dense BYOK**: A free, open-source desktop research workspace enabling local execution with user-provided API keys, cloud scalability via Modal, and access to all 134 skills.
- Compatibility with **Cursor, Claude Code, Codex**, and other AI agents, enabling multi-step workflows from data analysis to hypothesis generation.

The skills simplify interactions with scientific libraries and APIs, offering structured documentation and examples for reliability in research contexts.

## Key Points
- Over **134 skills** covering 10+ scientific domains, including drug discovery, genomics, and machine learning.
- Broad compatibility with AI agents via the open Agent Skills standard.
- Integration with **100+ scientific databases** and tools for reproducible research.
- Free, MIT-licensed open-source tools like [K-Dense BYOK](https://github.com/K-Dense-AI/k-dense-byok) for local and cloud-based research workflows.
- Focus on domains like clinical trials, proteomics, materials science, and AI/ML for scientific applications.
