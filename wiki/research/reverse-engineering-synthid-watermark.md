# Reverse-Engineering SynthID Watermark

> This project reverse-engineers Google's SynthID watermarking system embedded in Gemini-generated images, achieving 90% detection accuracy and developing a surgical bypass method with 43+ dB PSNR. It introduces a resolution-dependent spectral analysis approach, making it a critical tool for understanding and manipulating AI-generated image watermarks.  
> Last updated: 2026-04-11

## Overview  
The `reverse-SynthID` repository focuses on analyzing, detecting, and bypassing Google's SynthID watermark through signal processing and spectral analysis. Key achievements include:  

1. **Watermark Discovery**: Revealed SynthID's resolution-dependent carrier frequency structure, where watermark positions vary across image resolutions (e.g., 1024x1024 vs. 1536x2816).  
2. **Detection System**: Built a detector achieving 90% accuracy using spectral coherence analysis.  
3. **V3 Bypass Method**: A multi-resolution "SpectralCodebook" approach that reduces watermark energy by 75%, phase coherence by 91%, and maintains 43+ dB PSNR across all resolutions.  

### Community Contribution Initiative  
The project actively collects **pure black/white images** generated by Gemini's Nano Banana Pro to refine the codebook. Contributors are requested to:  
- Generate images of solid black (`#000000`) or white (`#FFFFFF`) using Gemini.  
- Upload to [Hugging Face](https://huggingface.co/datasets/aoxo/reverse-synthid).  
- Submit pull requests to expand resolution coverage.  

### Technical Differentiation  
Unlike brute-force methods (e.g., JPEG compression), the V3 bypass uses a **resolution-specific codebook** to surgically target watermark frequencies. This enables precise removal without degrading image quality.  

## Key Points  
- SynthID watermarks use **resolution-dependent carrier frequencies**, requiring separate profiles for different image sizes.  
- The V3 bypass achieves **43+ dB PSNR** while removing 75% of watermark energy.  
- Community contributions of black/white reference images are critical for improving cross-resolution robustness.
