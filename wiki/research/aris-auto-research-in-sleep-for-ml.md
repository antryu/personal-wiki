# ARIS: Auto-Research-in-Sleep for ML

> ARIS (Auto-Research-In-Sleep) is a lightweight framework for autonomous machine learning research, enabling cross-model review loops, idea discovery, and experiment automation. Designed as a Markdown-first system, ARIS works seamlessly with Claude Code, Codex, OpenClaw, or any LLM agent, offering flexibility and zero lock-in.

> Last updated: 2026-05-11

## Overview

ARIS is a research automation tool designed to streamline the process of machine learning research by enabling AI agents to work autonomously. It supports workflows in markdown-based skill files, allowing researchers to integrate different LLMs and tools without being restricted to any specific framework or platform.

The project is hosted on GitHub and has attracted significant attention, with over 8,000 stars and 800 forks since its launch in March 2026. It is actively developed under the MIT License, and the team regularly updates the system with new features and enhancements.

ARIS offers a CLI version, ARIS-Code, for those who prefer a standalone experience. This CLI supports local models like LM Studio and Ollama, and includes features such as a Research Wiki, self-evolution capabilities, and proxy support for Anthropic-compatible models.

## Key Points

- **Cross-Model Workflow**: ARIS supports a variety of LLMs including Claude Code, Codex, OpenClaw, and more.
- **Markdown-based Skills**: Each skill is a readable and portable `.md` file, making it easy to swap agents or platforms.
- **Autonomous Research**: Agents can review papers, identify weaknesses, run experiments, and rewrite narratives while the user sleeps.
- **Lightweight and Flexible**: No framework, no database, no Docker required.
- **Self-Evolution and Auto-Compaction**: ARIS can analyze logs and propose improvements to skill files automatically.
