# Self-Improving AI Agent

> A self-evolving AI agent with autonomous skill creation, persistent memory, and cross-platform adaptability. Notable for its closed learning loop, multi-model compatibility, and serverless deployment options.  
> Last updated: 2026-04-10

## Overview
**Hermes Agent** is a self-improving AI agent developed by Nous Research. It features a closed learning loop that autonomously creates and refines skills from experience, persists knowledge through periodic memory nudges, and builds a deep user model across sessions. The agent operates on diverse infrastructures (e.g., $5 VPS, GPU clusters, serverless platforms) and integrates with multiple communication channels (Telegram, Discord, CLI). It supports over 200 models via OpenRouter, OpenAI, and custom endpoints, enabling seamless switching without code changes. Key capabilities include:

- **Adaptive Learning**: Autonomous skill generation after complex tasks, with real-time skill improvement during use.
- **Cross-Platform Presence**: Unified gateway for Telegram, Discord, Slack, WhatsApp, Signal, and CLI, with voice memo transcription and session continuity.
- **Deployment Flexibility**: Six terminal backends (local, Docker, SSH, Daytona, Singularity, Modal) and serverless hibernation for cost efficiency.
- **Research Integration**: Trajectory generation, RL environments, and tool-calling model training support.

## Key Points
- Closed learning loop with agent-curated memory and autonomous skill creation/improvement  
- Multi-model compatibility (200+ models via OpenRouter, OpenAI, etc.) with zero-code switching  
- Serverless deployment options (Daytona/Modal) for near-zero idle costs  
- Cross-platform communication via Telegram, Discord, CLI, and more  
- Research-ready features for trajectory analysis and model training
