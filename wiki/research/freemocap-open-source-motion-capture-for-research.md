# FreeMoCap: Open-Source Motion Capture for Research

> FreeMoCap is an open-source, low-cost, research-grade motion capture (MoCap) system designed to democratize access to MoCap technology. It enables decentralized scientific research, education, and training using minimal hardware and open-source software.
> Last updated: 2026-05-07

## Overview

FreeMoCap is a community-driven project that offers a full-stack solution for motion capture, combining both software and hardware in an accessible and affordable manner. The system allows users to track human movement with multiple cameras and provides tools for 3D reconstruction and data analysis.

The software is written in Python and is fully open-source, licensed under the GNU Affero General Public License v3.0. The project is actively maintained with community contributions welcome. It is compatible with Python versions 3.10 to 3.12, with Python 3.12 being recommended.

The FreeMoCap platform includes a user-friendly GUI and supports both pip installation and source code installation. Detailed installation and usage instructions are available in the [official documentation](https://freemocap.github.io/documentation/).

The project has a strong focus on education and open science, aiming to make motion capture accessible to a broader audience beyond traditional research labs. It also includes a vibrant Discord community where users can seek help, share experiences, and contribute to the development of the project.

## Key Points
- Open-source and free-to-use, licensed under AGPLv3.
- Minimal-cost and research-grade motion capture solution.
- Supports both pip installation and source code deployment.
- Community-driven with contributions and documentation encouraged.
- Available with a graphical user interface (GUI) for easy use.
