# UI-TARS: Automated GUI Interaction with Native Agents

> UI-TARS is a groundbreaking open-source framework for automated GUI interaction with native agents, leveraging advanced multimodal models to perform tasks across virtual environments. It is notable for its integration with reinforcement learning-based reasoning and its ability to handle complex GUI and game interactions with high accuracy.
> Last updated: 2026-05-11

## Overview

UI-TARS is a research project developed by ByteDance, offering an open-source framework for automated GUI and virtual world interaction. It is built on a powerful vision-language model and enhanced with reinforcement learning, enabling it to reason and make decisions in real-time. The model is capable of handling diverse tasks such as GUI interaction, game playing, and tool use, all while maintaining high performance across various benchmarks. The latest version, **UI-TARS-2**, provides an "All In One" agent with enhanced capabilities in GUI, code, and tool use, making it a versatile solution for complex automation needs.

UI-TARS also provides a desktop version and web automation support via the Midscene.js project, ensuring adaptability across different platforms and environments.

## Key Points
- **Multimodal Architecture**: Utilizes a vision-language model combined with reinforcement learning for robust reasoning and interaction.
- **High Accuracy in Tasks**: Achieves state-of-the-art results in GUI interactions, game playing, and tool use.
- **Open Source**: Provides open-source models and inference scripts for easy deployment and testing.
- **Cross-Platform Support**: Offers desktop and web-based versions for local and web automation scenarios.
- **Comprehensive Documentation**: Includes deployment guides, technical reports, and coordinate processing guides.
