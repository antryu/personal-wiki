# GPT 전체 코드 리뷰

## 한줄 요약
nanoGPT의 model.py를 한 줄씩 분석. 300줄 미만의 코드로 GPT-2가 완전히 구현됨.

## 코드 구조 분석

### 1. 설정 (Config)

```python
@dataclass
class GPTConfig:
    block_size: int = 1024      # 최대 시퀀스 길이
    vocab_size: int = 50304     # 토큰 수 (64의 배수로 패딩)
    n_layer: int = 12           # 트랜스포머 블록 수
    n_head: int = 12            # 어텐션 헤드 수
    n_embd: int = 768           # 임베딩 차원
    dropout: float = 0.0
    bias: bool = True
```

### 2. CausalSelfAttention

```python
class CausalSelfAttention(nn.Module):
    def forward(self, x):
        B, T, C = x.size()
        
        # Q, K, V를 한번에 계산 (효율!)
        q, k, v = self.c_attn(x).split(self.n_embd, dim=2)
        
        # 멀티헤드로 reshape
        q = q.view(B, T, self.n_head, C // self.n_head).transpose(1, 2)
        k = k.view(B, T, self.n_head, C // self.n_head).transpose(1, 2)
        v = v.view(B, T, self.n_head, C // self.n_head).transpose(1, 2)
        
        # Flash Attention (PyTorch 2.0+) or 수동 구현
        y = F.scaled_dot_product_attention(q, k, v, is_causal=True)
        
        # 헤드 합치기
        y = y.transpose(1, 2).contiguous().view(B, T, C)
        return self.c_proj(y)
```

### 3. MLP (Feed-Forward)

```python
class MLP(nn.Module):
    def __init__(self, config):
        self.c_fc = nn.Linear(n_embd, 4 * n_embd)      # 확장
        self.gelu = nn.GELU()
        self.c_proj = nn.Linear(4 * n_embd, n_embd)     # 축소
```

### 4. Block (트랜스포머 블록)

```python
class Block(nn.Module):
    def forward(self, x):
        x = x + self.attn(self.ln_1(x))   # Pre-LN + 어텐션 + 잔차
        x = x + self.mlp(self.ln_2(x))    # Pre-LN + FFN + 잔차
        return x
```

### 5. GPT 전체

```python
class GPT(nn.Module):
    def forward(self, idx, targets=None):
        tok_emb = self.transformer.wte(idx)        # 토큰 임베딩
        pos_emb = self.transformer.wpe(pos)        # 위치 임베딩
        x = self.transformer.drop(tok_emb + pos_emb)
        for block in self.transformer.h:            # 트랜스포머 블록들
            x = block(x)
        x = self.transformer.ln_f(x)               # 최종 정규화
        logits = self.lm_head(x)                   # 출력 헤드
        
        if targets is not None:
            loss = F.cross_entropy(logits.view(-1, logits.size(-1)), 
                                   targets.view(-1))
        return logits, loss
```

## 핵심 최적화 포인트
1. **가중치 공유**: `wte`와 `lm_head`의 가중치 공유 → 파라미터 절약
2. **vocab_size 패딩**: 64의 배수로 맞춰서 GPU 연산 최적화
3. **Flash Attention**: O(N²) 메모리 → O(N) 메모리
4. **가중치 초기화**: 잔차 경로의 프로젝션에 1/√(2N) 스케일링
