# GPT 실습 가이드

## 한줄 요약
nanoGPT를 직접 실행하고 다양한 실험을 통해 GPT를 체화하는 실습 매뉴얼.

## 환경 설정

### 최소 요구 사항
```bash
# Python 3.10+, PyTorch 2.0+
pip install torch numpy transformers datasets tiktoken wandb tqdm

# nanoGPT 클론
git clone https://github.com/karpathy/nanoGPT.git
cd nanoGPT
```

### GPU 별 가능한 실험

| GPU | VRAM | 가능한 실험 |
|-----|------|------------|
| 없음 (CPU) | - | 셰익스피어 문자 레벨 (매우 느림) |
| Colab 무료 (T4) | 16GB | 셰익스피어, 소규모 실험 |
| RTX 3090 / 4090 | 24GB | GPT-2 small 파인튜닝 |
| 8×A100 | 640GB | GPT-2 124M 처음부터 학습 |

## 실습 1: 셰익스피어 (입문)

```bash
# 데이터 준비
python data/shakespeare_char/prepare.py

# 학습 (~3분 on GPU)
python train.py config/train_shakespeare_char.py

# 생성
python sample.py --out_dir=out-shakespeare-char
```

### 실험 변수
```python
# config 수정해서 실험!
n_layer = 6      # 4, 8, 12로 바꿔보기
n_head = 6       # 1, 4, 8로 바꿔보기  
n_embd = 384     # 128, 256, 768로 바꿔보기
learning_rate = 1e-3  # 1e-2, 1e-4로 바꿔보기
dropout = 0.2    # 0.0, 0.1, 0.5로 바꿔보기
```

## 디버깅 체크리스트

- [ ] 손실이 첫 스텝에서 `-ln(1/vocab_size)` ≈ 예상값인가?
- [ ] 손실이 지속적으로 감소하는가?
- [ ] 생성 텍스트가 점점 나아지는가?
- [ ] `torch.manual_seed(42)`로 재현 가능한가?
- [ ] gradient norm이 폭발하지 않는가?
