# 셀프 어텐션 (Self-Attention) 👀

## 한줄 요약
문장 안의 각 단어가 다른 모든 단어를 "둘러보고" 관련 있는 단어에 집중하는 방식이야.

## 왜 중요한가?
"그 학생이 시험을 잘 봤는데, 그는 기뻐했다"에서 "그"가 누구인지 알려면 앞의 "학생"을 봐야 하잖아? 셀프 어텐션은 모든 단어가 다른 모든 단어를 살펴보면서, 관련도를 점수로 매겨. 이게 트랜스포머와 GPT의 핵심 비밀이야!

## 핵심 개념

### 1. 모든 단어가 서로를 바라본다

```
"고양이가  생선을  먹었다"

  고양이가 ←──→ 생선을    (관련 높음! 🔥)
  고양이가 ←──→ 먹었다    (관련 높음! 🔥)
  생선을   ←──→ 먹었다    (관련 높음! 🔥)
```

각 단어가 "나랑 관련 있는 단어가 뭐지?" 하고 모든 단어를 살펴봐.

### 2. Query, Key, Value — 도서관 비유

```
📚 도서관에서 책 찾기:

  Query (검색어):  "고양이 음식" ← 내가 찾고 싶은 것
  Key (책 제목):   "생선 요리법" ← 각 책이 가진 라벨
  Value (책 내용): "참치, 연어..." ← 실제 내용

  Q와 K가 비슷하면 → 그 V를 많이 참고해!
```

모든 단어가 Q, K, V 세 가지 역할을 동시에 해.

### 3. 어텐션 점수 — 집중도 점수표

```
             고양이가  생선을  먹었다
  고양이가:   0.1     0.5    0.4     (생선에 집중!)
  생선을:     0.3     0.1    0.6     (먹었다에 집중!)
  먹었다:     0.5     0.3    0.2     (고양이에 집중!)
             ↑ 합이 항상 1.0 (확률!)
```

각 행의 합이 1이야. "내 관심의 100%를 어떻게 나눌까?"를 보여줘.
