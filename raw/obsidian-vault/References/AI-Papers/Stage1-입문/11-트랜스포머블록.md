# 🧱 트랜스포머 블록 — GPT의 레고 블록

## 한줄 요약
멀티헤드 어텐션 + 피드포워드 네트워크를 합친 기본 블록. 이걸 여러 개 쌓으면 GPT가 돼!

## 🏗️ 레고 비유

레고로 건물을 만들 때, 같은 블록을 반복해서 쌓잖아? GPT도 마찬가지야!

```
GPT-2:  블록 × 12개  🏠
GPT-3:  블록 × 96개  🏢🏢🏢
```

## 🔧 블록 하나의 내부

```
입력
 ↓
 ├──────────────────────┐
 ↓                      │ (지름길! 잔차 연결)
[레이어 정규화]           │
 ↓                      │
[멀티헤드 어텐션]  👀      │
 ↓                      │
 + ◀────────────────────┘  ← 입력을 더해줘!
 ↓
 ├──────────────────────┐
 ↓                      │ (지름길! 잔차 연결)
[레이어 정규화]           │
 ↓                      │
[피드포워드 네트워크] 🧠   │
 ↓                      │
 + ◀────────────────────┘  ← 또 더해줘!
 ↓
출력
```

## 🧩 각 부품 설명

### 멀티헤드 어텐션 👀
"단어끼리 관계 파악" — 이전 장에서 배웠지!

### 피드포워드 네트워크 (FFN) 🧠
어텐션으로 모은 정보를 **소화**하는 단계:

```
768차원 → 3072차원(4배!) → 768차원
         ↑                ↑
       확장해서          다시 압축!
       깊이 생각
```

### 잔차 연결 (지름길) 🔗
입력을 출력에 그대로 더해주는 것. 중요한 정보가 사라지지 않게!

```
지름길 없이:  입력 → [처리] → 출력  (원본 정보 사라질 수 있음 😢)
지름길 있으면: 입력 → [처리] → 출력 + 입력  (원본 보존! 😊)
```

### 레이어 정규화 📏
값이 너무 크거나 작아지지 않게 조절. 시험 점수를 표준점수로 바꾸는 것과 비슷해!

## 🎯 핵심 정리

- **트랜스포머 블록** = 어텐션 + FFN + 잔차 연결 + 정규화
- 이 블록을 **N번 쌓으면** GPT!
- 각 블록이 정보를 점점 더 깊이 이해해
- GPT-2는 12블록, GPT-3는 96블록!
