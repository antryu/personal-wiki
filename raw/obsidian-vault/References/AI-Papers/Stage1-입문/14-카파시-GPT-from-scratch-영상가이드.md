## 🎯 영상의 핵심 포인트

### 1단계: 가장 바보 같은 모델부터
```
Bigram: "이전 글자 하나만 보고 다음 글자 예측"
→ 결과: "Thens ber ilaved..." 😵 (엉망!)
```

### 2단계: 셀프 어텐션 추가
```
"이전 글자들 전부를 보고 관계를 파악해서 예측"
→ 결과: 조금 나아짐!
```

### 3단계: 전체 트랜스포머 조립
```
임베딩 + 멀티헤드 어텐션 + FFN + 잔차연결
→ 결과: 셰익스피어 스타일 텍스트 생성! 🎭
```
