# ✍️ 텍스트 생성 — GPT는 어떻게 글을 쓸까?

## 한줄 요약
학습된 GPT가 다음 단어를 하나씩 골라가면서 글을 써나가는 과정이야.

## 🎰 단어 뽑기 기계

GPT는 매번 **확률 뽑기**를 해:

```
입력: "오늘 날씨가"

GPT의 예측:
  ┌─────────┬────────┐
  │  단어    │  확률   │
  ├─────────┼────────┤
  │  좋다   │  45%   │ ████████████████████
  │  나쁘다  │  20%   │ ████████
  │  덥다   │  15%   │ ██████
  │  춥다   │  10%   │ ████
  │  흐리다  │  7%    │ ███
  │  맑다   │  3%    │ █
  └─────────┴────────┘

→ "좋다" 선택! (확률이 제일 높으니까)
→ "오늘 날씨가 좋다" 완성
→ 이제 "오늘 날씨가 좋다" 다음 단어는?
→ 반복!
```

## 🌡️ 온도 (Temperature)

```
온도 낮음 (0.1) — 안전하고 예측 가능:
  "오늘 날씨가 좋다."  (항상 비슷한 답)

온도 중간 (0.7) — 적당히 창의적:
  "오늘 날씨가 좋아서 공원에 갔다."

온도 높음 (1.5) — 창의적이지만 이상할 수도:
  "오늘 날씨가 보라색 고양이처럼 춤을 춘다."  😅
```

> 💡 ChatGPT의 온도는 보통 0.7 정도!

## 🎯 Top-k와 Top-p

확률이 너무 낮은 이상한 단어가 선택되지 않게 막아줘:

```
Top-k = 5: 확률 상위 5개 단어에서만 고르기
  좋다(45%), 나쁘다(20%), 덥다(15%), 춥다(10%), 흐리다(7%)
  → "우주선"(0.001%) 같은 건 제외!

Top-p = 0.9: 확률 합이 90%가 될 때까지만
  좋다(45%) + 나쁘다(20%) + 덥다(15%) + 춥다(10%) = 90%
  → 이 4개 중에서 선택!
```

## 🔁 자기회귀 = 이어쓰기

```
Step 1: "오늘"          → 다음: "날씨가"
Step 2: "오늘 날씨가"    → 다음: "좋아서"
Step 3: "오늘 날씨가 좋아서" → 다음: "기분이"
Step 4: "오늘 날씨가 좋아서 기분이" → 다음: "좋다"
...
```

내가 쓴 글을 다시 읽고 → 다음 단어 쓰고 → 반복!
ChatGPT가 글자를 하나씩 출력하는 이유가 이거야!

## 🎯 핵심 정리

- GPT는 **확률 분포에서 단어를 하나씩 뽑아서** 글을 써
- **온도** = 창의성 조절 (낮으면 안전, 높으면 창의적)
- **Top-k / Top-p** = 이상한 단어 걸러내기
- **자기회귀** = 쓴 글을 다시 읽고 다음을 이어쓰기
