# 멀티헤드 어텐션 (Multi-Head Attention) 👀👀👀

## 한줄 요약
여러 개의 어텐션을 동시에 돌려서, 다양한 관점에서 단어 관계를 파악해.

## 왜 중요한가?
한 문장을 국어 선생님, 수학 선생님, 미술 선생님이 각각 분석하면 더 풍부한 이해가 되잖아? 멀티헤드 어텐션이 바로 그거야. 어텐션 헤드가 8개면, 8가지 다른 관점에서 동시에 단어 관계를 분석해.

## 핵심 개념

### 1. 하나 vs 여러 개

```
싱글 헤드 (1개의 관점):
  "나는 사과를 먹었다"
  → 헤드1: "나는"↔"먹었다" (주어-동사 관계만 봄)

멀티 헤드 (8개의 관점):
  → 헤드1: "나는"↔"먹었다"  (문법 관계)
  → 헤드2: "사과를"↔"먹었다" (의미 관계)
  → 헤드3: "나는"↔"사과를"  (행위자-대상)
  → ...
  → 결합! 훨씬 풍부한 이해 🎉
```

### 2. 차원 나누기 — 계산은 똑같아

```
768차원 벡터 ÷ 8헤드 = 각 헤드가 96차원 담당

  [────────── 768차원 ──────────]
  [헤드1][헤드2][헤드3]...[헤드8]
   96     96    96        96

계산량은 싱글 헤드랑 같은데, 관점은 8배! 🎯
```

### 3. 합치기 — 모든 의견 종합

모든 헤드의 결과를 이어 붙이고, 하나의 벡터로 합쳐. 선생님 8명의 성적표를 종합하는 것처럼!
