# GPT-3 — 이게 왜 대단해? 🤯

## 한줄 요약
1750억 개의 파라미터를 가진 초거대 AI로, 몇 가지 예시만 보여주면 새로운 작업도 바로 해내.

## 왜 중요한가?
새 게임을 배울 때, 규칙서를 다 읽지 않아도 몇 판 보면 감이 오잖아? GPT-3는 그런 능력을 AI에서 구현했어. 2-3개 예시만 보여주면 번역, 요약, 코딩을 별도 학습 없이 바로 해! 이 논문이 ChatGPT로 이어지는 시작이었고, "크기가 곧 지능"이라는 것을 보여줬어.

## 핵심 개념

### 1. 퓨샷 러닝 — 몇 개만 보여주면 OK

```
프롬프트에 이렇게 써주면:

  "사과 → apple
   바나나 → banana
   포도 → "

GPT-3 답: "grape" ✅

👉 학습(가중치 업데이트) 없이, 예시만 보고 이해!
```

### 2. 세 가지 모드

```
🔴 제로샷: 예시 없이 바로 수행
   "다음을 한국어로 번역해: Hello" → "안녕하세요"

🟡 원샷: 예시 1개
   "happy → 행복한. sad → " → "슬픈"

🟢 퓨샷: 예시 2~3개
   가장 정확함!
```

### 3. 크기의 힘 — 양이 질로 변한다

```
모델 크기 →→→→→→→→→→→→→→→→→→→→

  작은 모델:  번역 ❌  코딩 ❌  추론 ❌
  중간 모델:  번역 ✅  코딩 ❌  추론 ❌
  GPT-3:     번역 ✅  코딩 ✅  추론 ✅ 🎉

  → 크기가 임계점을 넘으면 갑자기 새 능력이 등장!
```
