# RNN과 LSTM 🧠💭

## 한줄 요약
RNN은 이전 내용을 기억하면서 읽는 신경망이고, LSTM은 중요한 것만 골라 기억하는 업그레이드 버전이야.

## 왜 중요한가?
"어제 비가 왔기 때문에 오늘은 ___을 가져갔다." 이 빈칸을 채우려면 "비"라는 앞부분 정보를 기억해야 해. 일반 신경망은 입력을 한 번에 보고 끝이라 "기억"이 없어. RNN은 한 단어씩 읽으면서 기억을 업데이트해. 번역, 음성인식, 챗봇 등 순서가 중요한 모든 AI의 기본이 됐어.

## 핵심 개념

### 1. RNN — 돌려막기 신경망

```
"나는 학교에 간다"

  "나는" → [RNN] → 기억₁
              ↓ (기억 전달)
  "학교에" → [RNN] → 기억₂
              ↓ (기억 전달)
  "간다" → [RNN] → 기억₃ → 최종 출력
```

같은 신경망을 재활용하면서, 이전 기억을 계속 넘겨줘. 일기장을 한 줄씩 읽으면서 내용을 요약하는 것과 같아.

### 2. RNN의 문제 — 금붕어 기억력 🐟

```
아주 긴 문장:
"작년 여름에 제주도에서 ... (200단어) ... 그래서 나는 비행기를 탔다"

RNN: "제주도? 뭐였더라..." → 앞부분을 까먹어!
```

문장이 길어지면 앞부분 정보가 점점 사라져. 이걸 "장기 의존성 문제"라고 해.

### 3. LSTM — 똑똑한 기억 관리자

```
┌──────────────────────────────┐
│          LSTM 셀             │
│                              │
│  🚪 망각 게이트: "이건 잊자"   │
│  🚪 입력 게이트: "이건 기억!"  │
│  🚪 출력 게이트: "이건 출력!"  │
│                              │
│  📦 기억 셀: 장기 보관함       │
└──────────────────────────────┘
```

LSTM은 3개의 "게이트"(문)으로 뭘 기억하고 뭘 잊을지 결정해:
- **망각 게이트**: 필요 없는 정보를 버려 (오래된 알림 삭제)
- **입력 게이트**: 새로운 중요 정보를 저장 (즐겨찾기 추가)
- **출력 게이트**: 지금 필요한 정보만 꺼내 (검색)
