# AI란 무엇인가? 🤖

## 한줄 요약
AI는 사람의 뇌를 흉내 낸 "인공 뉴런"을 연결해서, 데이터로부터 스스로 배우는 프로그램이야.

## 왜 중요한가?
스마트폰의 얼굴 인식, 유튜브 추천, 번역 앱... 이 모든 게 AI야. 옛날 프로그래밍은 "이럴 때 이렇게 해"라고 규칙을 일일이 적어줬어. 하지만 AI는 수만 개의 예시를 보여주면 스스로 규칙을 찾아내. 강아지 사진 1만 장을 보여주면 "아, 이런 게 강아지구나!"를 알아서 배우는 거야.

## 핵심 개념

### 1. 뉴런 — AI의 가장 작은 단위

사람의 뇌에는 약 860억 개의 뉴런(신경 세포)이 있어. AI의 뉴런은 이걸 아주 단순하게 흉내낸 거야:

```
  입력1 ──→ ┌─────────┐
  입력2 ──→ │  뉴  런  │──→ 출력
  입력3 ──→ └─────────┘
             (가중합 → 활성화)
```

각 입력에 "가중치"(중요도)를 곱해서 더하고, 일정 기준을 넘으면 신호를 보내. 마치 투표처럼, 찬성이 많으면 통과!

### 2. 신경망 — 뉴런을 연결한 네트워크

뉴런 하나로는 할 수 있는 게 적어. 하지만 수천, 수만 개를 층층이 연결하면?

```
입력층        은닉층        출력층
 ○ ─────┐   ┌── ○ ──┐   ┌── ○  강아지 (95%)
 ○ ──┐  ├── ○ ──┤   ├── ○  고양이 (3%)
 ○ ──┤  ├── ○ ──┤   └── ○  토끼 (2%)
 ○ ──┘  └── ○ ──┘
(픽셀들)  (특징 추출)   (판단)
```

- **입력층**: 데이터를 받는 곳 (사진의 픽셀, 문장의 단어 등)
- **은닉층**: 특징을 찾는 곳 (가장자리, 색깔, 패턴 등)
- **출력층**: 최종 답을 내는 곳

### 3. 학습 — 실수하면서 똑똑해지기

```
[데이터] → [신경망] → [예측] ──→ 정답과 비교
                                    │
              가중치 조정 ←─── [오차 계산]
              (역전파)
```

1. 신경망이 답을 내놔 (처음엔 틀려)
2. 정답과 비교해서 얼마나 틀렸는지 계산해
3. 틀린 만큼 가중치를 조금씩 조절해
4. 이걸 수만 번 반복하면 점점 잘 맞춰!

시험 오답 노트를 계속 만드는 것과 같아. 틀린 문제를 분석하고 고치기를 반복하면 점수가 오르지?

## 쉬운 비유로 정리

| AI 용어 | 일상 비유 |
|---------|----------|
| 뉴런 | 투표하는 사람 한 명 |
| 신경망 | 투표하는 사람들의 모임 |
| 가중치 | 각 사람의 영향력 |
| 학습 | 오답 노트 만들기 반복 |
| 데이터 | 연습 문제집 |
