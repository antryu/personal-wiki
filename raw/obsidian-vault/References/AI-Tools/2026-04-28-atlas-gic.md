# ATLAS by General Intelligence Capital

> ATLAS — Self-improving AI trading agents using Karpathy-style autoresearch
> Built by [General Intelligence Capital](https://generalintelligencecapital.com)
> 1,469 ⭐ · 270 forks · MIT license · Python · Created 2026-03-11

## What it is

A framework for multi-agent LLM trading where **agent prompts are the model weights** and **Sharpe ratio is the loss function**. Agents debate markets daily, get scored against real outcomes, and the worst performer's prompt is rewritten via autoresearch loop. Git commit if Sharpe improves; git revert if not.

Inspired by:
- [Karpathy autoresearch](https://github.com/karpathy/autoresearch) — pattern adapted to financial markets
- [Soros reflexivity](https://en.wikipedia.org/wiki/Reflexivity_(social_theory)) — markets change reality
- [MiroFish swarm](https://github.com/666ghj/MiroFish) — parallel digital worlds

## Architecture — 25+ agents in 4 layers

| Layer | Agents | Role |
|-------|--------|------|
| L1 Macro | 10 (CB, geopolitical, China, USD, yield, commodities, vol, EM, news, flow) | Set regime — risk on/off |
| L2 Sector | 7 (semi, energy, biotech, consumer, industrials, financials, relationship-mapper) | Sector views + supply chain |
| L3 Superinvestor | 4 (Druckenmiller, Aschenbrenner, Baker, Ackman) | Style-specific theses |
| L4 Decision | 4 (CRO, Alpha Discovery, Auto Execution, CIO) | Final synthesis + sizing |

## Key mechanisms

### Autoresearch loop
- Worst Sharpe agent → 1 prompt mod → 5 days test → keep (commit) or revert
- $20/mo VM replaces H100 (LLM cost-efficient)

### Darwinian weights
- Each agent: 0.3 (min) to 2.5 (max)
- Top quartile × 1.05 daily, bottom × 0.95
- CIO weighting input by these scores

### Agent spawning
- Detects recurring blind spots (3+ in 5 days) → autonomously creates new specialist
- Test (Jul-Dec 2024): 9 spawned, 3 extinct, 6 survived

### PRISM regime training
5 separate cohorts, each evolved on different historical regime:
| Cohort | Period | Return | Mods Kept | Key Learning |
|--------|--------|--------|-----------|--------------|
| Bull/Low Vol | 2016-18 | +7.7% | 35% | Exit vol longs at peace |
| Crisis (COVID) | 2020 Q1-Q2 | -13.1% | 0% | Too fast for autoresearch |
| Rate Tightening | 2022-23 | -30.2% | 43% | Don't flip-flop in Fed weeks |
| Recovery | 2020 Q2-Q4 | -29.0% | 0% | Same problem as crisis |
| Euphoria | 2021 | +14.3% | 49% | Momentum confirm before short |

→ **Convergent evolution**: All 5 independently learned cap conviction, VIX as filter, hard position limits.
→ **Divergent evolution**: Same volatility agent grew bull→121KB, tight→10KB, euphoria→2KB. Different markets shape different agents.

### JANUS meta-layer
- Sits above all PRISM cohorts, weights by recent accuracy
- Weight differential = emergent regime detector (didn't program it)

### Soros reflexivity engine
5 feedback loops modeled:
1. Price → fundamentals (drop >15% triggers downgrades, flight)
2. P&L → behavior (drawdown → forced selling)
3. Narrative → flows (analyst convergence → retail follows)
4. Market → policy (drawdown → CB easing)
5. Reflexive reversal detection (5+ rounds same direction → fragility flag)

### MiroFish integration
- Overnight generates branching futures (geopolitical, Fed, earnings shocks, black swans)
- Trading agents trained inside simulated futures
- "Druckenmiller agent scores 1.0 in simulated crashes, 0.22 in melt-ups" — system knows which instincts to trust before regime arrives
