## 프레임워크 요약 — 3권을 1쪽으로

### Book 1: *Principles for Navigating Big Debt Crises* (2018)
**핵심 명제**: 모든 대형 채무 위기는 동일한 메커니즘으로 작동하며, 역사 패턴을 이해하면 예측 가능하다.

3가지 부채 사이클 (자세히 → [[debt-cycles]]):
- **단기 (5-8년)**: 신용/경기 사이클. 중앙은행이 조절 가능.
- **장기 (50-75년)**: 세대를 넘는 부채 누적. 구조적 디레버리징 불가피.
- **통화 사이클 (100-200년)**: 기축통화 부상과 몰락. 현재 USD가 하락 국면.

**2024-2026 위치**: 단기 1C/1D, 장기 2C(피크), 통화 3C(하락). 3개 사이클 동시 스트레스 구간.

---

### Book 2: *Principles for Dealing with the Changing World Order* (2021)
**핵심 명제**: 제국·기축통화의 흥망은 반복 가능한 패턴이다. 17개 지표로 측정 가능하며, Big Cycle 6단계로 위치 식별 가능.

Big Cycle (→ [[big-cycle-stages]]):
1. New Order → 2. Peace & Prosperity → 3. Excess → 4. Top → 5. Decline → 6. Bottom

17개 지표 (→ [[world-order-17-measures]]):
- **현재 US 포지션**: 119/170점, 11/17개 지표 하락 추세
- **China**: 107/170점, 대부분 상승 추세
- **Korea**: 89/170점, 지정학 노출이 최대 리스크

**자산 함의**: Stage 4-5 전환 구간에서 하드 자산(금·은·실물)이 금융 자산(주식·채권) 대비 outperform.

---

### Book 3: *The Changing World Order* (2021, 심화판)
*Principles for Dealing with the Changing World Order*와 동일 저작의 심화 버전. 역사 사례 심층 분석 추가.

추가 핵심 사례:
- 네덜란드 황금기 (1600s-1700s): VOC → Amsterdam 금융 중심 → 영국에 추월
- 대영제국 (1815-1945): 파운드 기축 → 2차 세계대전 후 달러에 이전
- 미국 (1945-현재): 브레턴우즈 → 닉슨 쇼크 → 현재 달러 하락 추세

**Dalio의 결론**: 현재 미국은 영국이 1920-1945에 겪은 것과 유사한 경로에 있다. 쇠퇴가 예정된 것은 아니나, 부채·불평등·내부 갈등 3요소가 동시에 악화 중.

## Andrew Asset에 이 프레임워크가 중요한 이유

**쉽게 말해**: Dalio 프레임워크는 "지금이 어느 시대인지"를 알려주는 GPS다. 개별 종목을 고를 때 "NVDX 지금 사도 되나?"가 아니라 "지금이 Stage 2-3인지 Stage 4-5인지"를 먼저 판단하면, 개별 종목 판단이 훨씬 단단해진다.

**구체적으로 가능해지는 의사결정**:

1. **비중 조정 근거 문서화**: "UGL 비중 올린다" → "현재 Stage 4-5 전환, USD 통화 사이클 3C 확인, 17개 지표 11개 하락 → 금 보유 framework 완전 지지" — 이런 식으로 감이 아니라 프레임으로 기록.

2. **포지션 변경 트리거 사전 정의**: "Stage 4→5 진입 신호(CAPE 지속 고점·신용 스프레드 급확대·정치 불안 지수 신고점) 시 NVDX 축소, UGL/AGQ 확대" — 미리 조건 세팅.

3. **17개 지표 periodic review**: Phase B (매크로 인디케이터 페처)에서 이 17개 지표를 실시간 데이터로 업데이트하면 → 의장이 분기마다 "지금 US score 얼마인지" 30초에 확인 가능.

4. **US-China 패권 전환 시나리오 준비**: 단순한 "중국이 뜰 것 같다"가 아니라 "China가 17개 지표 중 몇 개에서 US를 추월하는 시점"을 측정 가능하게 만듦 → 아시아 자산 로테이션 타이밍 근거.

## 현재 매크로 포지션 요약 (2024-2026)

```
Big Cycle    : Stage 4→5 전환 (US Top → Decline 초기)
Long Debt    : Stage 2C → 2D 압박 (장기 부채 피크, 다음 침체 = 구조적 디레버리징 위험)
Currency     : Stage 3C (USD 준비통화 비중 하락 추세, 58%→목표 미상)
17 Measures  : US 119점(하락), China 107점(상승), 2041±10년 추월 가능 추정
```

**포트폴리오 함의**: 하드 자산 (UGL, AGQ, BTC) = 현재 프레임워크 가장 강한 지지. 고성장 레버리지 (NVDX) = 조건부 유지, top signal 시 축소. 퀀텀/New Order (IONQ, RGTI) = 장기 positioning, 소규모.
