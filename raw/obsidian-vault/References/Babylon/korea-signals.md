3. **AI가 주니어 개발자를 쓸모없게 만들고 있다** (32pts, 18h)

4. **에이전틱 엔지니어링 시대의 생존 스킬 9가지** (67pts, 3d) ⭐ 인기
   - Karpathy의 에이전트 활용 사례 기반, 한국 개발자 커뮤니티 반향 큼
   - "바이브 코딩" 개념의 한국 확산

5. **MCP는 죽었다. CLI 만세** (29pts, 2d)
   - MCP(Model Context Protocol) 관심 하락 → CLI 기반 접근이 더 실용적
   - 도구/프로토콜 피로감 시그널

6. **Anthropic Courses 무료 온라인 강의 공개** (83pts, 2d) ⭐ 최고 인기
   - Claude 기반 교육 콘텐츠에 한국 개발자 관심 폭발적

7. **Alibaba zvec — 초경량 인프로세스 벡터 DB** (28pts, 1d)

## 📰 아웃스탠딩 (outstanding.kr) — 스타트업 분석

### 주요 기사: "플랫폼의 시대는 끝나지 않았다. 다만 룰이 바뀌었을 뿐"

**핵심 인사이트:**
- 한국 유니콘의 대부분이 플랫폼(쿠팡, 배민, 쏘카, 야놀자, 토스)이던 시대 종료
- 티몬·위메프 붕괴 (위메프 2025년 파산 선고) → "플랫폼 = 성공 방정식" 믿음 붕괴
- **VC들의 플랫폼 기피 현상** 심화
- 2025년 기준 국내 스타트업 투자:
  - 전체 투자 금액: 전년 수준 유지
  - **딜 수 30% 이상 감소**
  - **초기 단계 투자 비중: 38.3% → 26.7% 급감**
- AI, 반도체 등 **딥테크가 투자 비중 빠르게 확대**
- 일반 ICT서비스/플랫폼 스타트업 투자금 감소

## 🔍 종합 분석: 한국 AI/스타트업 시그널

### 1. 투자 환경 변화
- **플랫폼 → 딥테크 전환** 가속 (아웃스탠딩)
- 초기 투자 위축, 딜 수 30% 감소 → 선별적 투자
- AI/반도체가 투자 파이의 주력으로 부상

### 2. AI 도구 활용 폭발 (Claude 중심)
- GeekNews 상위권 대부분이 Claude 관련 프로젝트
- Claude Code 해커톤, 학습 도구, 사용량 추적기 등 생태계 형성
- **한국 개발자 커뮤니티가 Claude를 사실상 표준 도구로 채택** 중

### 3. "바이브 코딩" 한국 확산
- 비개발자도 AI로 서비스 제작 (eo: 15분만에 바이브코딩)
- 에이전틱 엔지니어링 스킬 논의 활발
- 주니어 개발자 역할 재정의 논쟁

### 4. 1인 기업 / 솔로프리너 AI 모델
- eo에서 "AI로 10억 1인 기업" 콘텐츠가 1위
- AI 자동화 기반 소규모 사업 모델에 대한 관심 폭증

### 5. AI Slop → 역설적 기회
- AI 생성 저품질 콘텐츠가 새로운 산업(필터링, 큐레이션)을 만든다는 관점
- 한국 시장에서 AI 콘텐츠 품질 관리 니즈 부상

### 6. 한국 시장 특이점 (글로벌 대비)
| 한국 | 글로벌 |
|------|--------|
| Claude가 GPT 대비 개발자 선호도 급상승 | GPT가 여전히 시장 지배적 |
| 플랫폼 붕괴 트라우마 (티몬/위메프) → VC 딥테크 전환 | 플랫폼 여전히 건재 (미국) |
| MCP의 로컬 적용 (다이소 재고 등) | MCP 피로감/CLI 회귀 |
| AI 자격증·학습 도구 수요 폭발 | AI 도구 직접 사용 중심 |
| 정부지원금 활용 가이드가 인기 콘텐츠 | VC 펀딩 중심 |
| AI 대체율 논의가 "쇼크" 프레임 | 점진적 변화 프레임 |
