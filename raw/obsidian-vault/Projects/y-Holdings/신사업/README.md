## 평가 프레임워크 (참고)

각 신사업 검토 시 다음 7개 차원 평가:

1. **Demand evidence** — 기존 비용 지불 사실, segment-wide 시그널 유무
2. **Andrew 자산 재활용** — M1 기존 자산으로 % 커버 가능?
3. **Solo build feasibility** — 솔로 MVP 시간·기술 가능성
4. **ROI 시나리오** — 5/30/100 고객 단계별 매출
5. **Anti-pattern check** — Bid 패턴, scope creep, "하나만 더 만들고 팔자"
6. **Concurrent load** — Cap/Bid/Vita 부하 위에서 신규 라인 가능?
7. **Lawyer/regulatory** — 마켓 ToS, 도메인 규제 (의료·금융·세무 등)
