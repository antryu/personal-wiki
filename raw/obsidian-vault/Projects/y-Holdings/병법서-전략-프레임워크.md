# 병법서 전략 프레임워크 — _y Holdings

> 동서양 병법서의 핵심 원리를 _y Holdings 사업 전략에 매핑

## Phase 1: 핵심 원리 추출 (기획조정실)

### 대상 병법서
1. 손자병법 (孫子兵法)
2. 오자병법 (吳子兵法)
3. 삼십육계 (三十六計)
4. 육도삼략 (六韜三略)
5. 클라우제비츠 전쟁론 (Vom Kriege)
6. 마키아벨리 군주론 (Il Principe)
7. 미야모토 무사시 오륜서 (五輪書)

### 기획조정실 합의사항
- 병법서 원리를 기업 전략에 적용하되, **재무 성과와 법적 리스크 관리를 최우선**으로 고려
- 각 원리가 _y Holdings의 OKR 달성에 기여하고 준법 경영을 저해하지 않도록 분석
- Tasky: 전략적 연결성 분석
- Finy: 재무 영향 및 KPI 분석
- Legaly: 법적 리스크 및 방어 전략

## Phase 2: 계열사별 매핑 (실무 부서)

### _y Builder (SW개발본부)
- **"공격할 곳을 알되 방어할 곳도 알라"**
- 핵심 기능에 자원 집중 + 기술 부채 예측/방어
- 코드 품질 지표 90% 이상 유지
- Buildy: Agent Framework + Persona Management에 집중

### _y Trading (투자사업부)
- **"지피지기 백전불태"**
- 시장 분석(지피) + 자사 전략 분석(지기) → 알파 극대화, 변동성 최소화
- 퀀트 모델의 상태 공간 정의 → 최적 전략 전환 로직
- Quanty: 샤프 비율 2.0 이상 목표

### _y Media (콘텐츠본부)
- **"정병으로 싸우고 기병으로 이겨라 (以正合 以奇勝)"**
- 정병: 핵심 스토리 콘텐츠 (AI 회사 운영 일지) — 신뢰 구축
- 기병: 바이럴 밈/챌린지 — 예측 불가능한 폭발력
- Buzzy: 매주 바이럴 기획 회의 + A/B 테스트

### _y SaaS (운영)
- **"보급로 확보" + "견고한 진지 구축"**
- 안정적 인프라 운영 = 전쟁의 보급선
- 99.9% 가동률 목표

### 전사 공통
- **리스크 관리**: Skepty — 각 원리 적용 전 Pre-mortem + 블랙스완 시나리오 검증
- **측정**: Growthy — 모든 원리 적용에 A/B 테스트 필수, p-value 유의미해야 채택

## Phase 3: Playbook 구체화

### _y Trading — "지피지기" Playbook
| 항목 | 내용 |
|---|---|
| 트리거 | VIX 20%↑ 또는 섹터 변동성 2σ 초과 |
| 소집 | Quanty + Tradey + Globy |
| 실행 | ① Quanty: 변동성 요인 분석 ② Tradey: 베타/헷지 전략 ③ Globy: 매크로 영향 평가 |
| 지표 | VaR 갱신율 감소, 샤프비율 0.1↑ 유지 |

### _y Media — "허실편" Playbook
| 항목 | 내용 |
|---|---|
| 트리거 | 콘텐츠 반응률 20%↓ 또는 트렌드 지수 30%↓ |
| 소집 | Buzzy + Wordy + Edity + Searchy |
| 실행 | ① 밈 분석 ② A/B 숏폼 3개 긴급 제작 ③ 바이럴 포인트 재설정 |
| 지표 | 24h 내 조회수 50%↑ 회복, 공유율 15%↑ |

### _y Builder — "군쟁편" Playbook
| 항목 | 내용 |
|---|---|
| 트리거 | 마일스톤 3일↑ 지연 또는 Critical 버그 3개↑ |
| 소집 | Buildy + Pixely + Testy + Skepty(Red Team) |
| 실행 | ① 기술부채 분석 ② MVS 재정의 ③ 우회경로 설계 ④ Red Team 검증 |
| 지표 | 24h 내 정상화, Critical 버그 50%↓, 코드 커버리지 80%↑ |

### Skepty 미결 이슈
- 지연 판단 기준의 Critical Path Impact 계산 필요
- 의사결정 편향(확증편향, 매몰비용) 통제 메커니즘
- MVS 재정의 시 이해관계자 기대치 관리

## Phase 4: 에이전트 학습 반영 (예정)
- 각 에이전트의 시스템 프롬프트에 해당 병법 원리 추가
- 의사결정 시 자동으로 병법 프레임워크 참조
- 시간 경과에 따른 전략 진화 추적
