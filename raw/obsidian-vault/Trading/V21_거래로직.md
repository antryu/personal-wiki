## [1] 레짐 판단 (`CalculateRegime`)

매 30분봉 완성 시 + 초기화 시 호출

**입력:**
- MA20 = SMA(종가, 20) → 셀 DA25
- MA200 = SMA(종가, 200) → 셀 DB25 (유효 데이터 100개 미만 시 MA20 fallback)
- ATR_20_AVG = SMA(ATR(20), 20) → 셀 DC25
- ATR_14 = ATR(14) → 셀 BP26

**레짐 결정:**
| 조건 | 레짐 | 코드 | 허용 방향 |
|------|------|------|----------|
| MA20 > MA200 AND ATR_14 < ATR_AVG × 1.3 | UP_LOW | 1 | LONG only |
| MA20 > MA200 AND ATR_14 ≥ ATR_AVG × 1.3 | UP_HIGH | 2 | LONG only |
| MA20 ≤ MA200 AND ATR_14 < ATR_AVG × 1.3 | DOWN_LOW | 3 | SHORT only |
| MA20 ≤ MA200 AND ATR_14 ≥ ATR_AVG × 1.3 | DOWN_HIGH | 4 | LONG(하단) + SHORT(상단) |

## [3] 진입 조건

### LONG (BB 하단 이탈)
```vb
If flagRealBS = 0                              ' 포지션 없음
  And AJ26 = 0                                 ' BB 내부 카운트 = 0
  And flag_logicstart = 1                       ' 로직 시작됨
  And DailyTradeCount < MAX_DAILY_TRADES Then   ' 하루 1회 제한
    
    ' BB 하단 이탈 감지
    If BB_Closek < BB_LOk Then
        If flag304 >= 1 And RSIADX.Value > 35 Then   ' RSI(14) > 35 필터
            If flagLogic = 0 Then
                If SPTHM > 520 And SPTHM < 700 Then  ' 서머타임 시간대
                    → BuyFX 호출
                    → DailyTradeCount + 1
```

### SHORT (BB 상단 이탈)
```vb
If BB_Closek > BB_UPk Then
    If flag304 >= 2 Then   ' RSI 필터 없음 (SHORT)
        If flagLogic = 0 Then
            If SPTHM > 520 And SPTHM < 700 Then
                → SellFX 호출
                → DailyTradeCount + 1
```

## [4] 포지션 관리 (`CalcAdaptiveTrailingStop`)

```vb
Function CalcAdaptiveTrailingStop(entryPrice, currentPrice, isBuy)
    actualSL = DF25 값  ' ATR 동적 SL
    
    ' 수익 계산 (틱 단위)
    profitPt = (currentPrice - entryPrice) × 4  ' LONG
    
    If profitPt >= RegimeTrailingStart Then  ' 15틱 이상 수익
        trailingLevel = currentPrice - (RegimeTrailingStep / 4)  ' 현재가 - 1.25pt
    Else
        trailingLevel = entryPrice - (actualSL / 4)  ' 진입가 - SL
    End If
End Function
```

**매 틱마다:**
1. `CalcAdaptiveTrailingStop` 호출 → `BS25` (트레일링 레벨) 업데이트
2. `G13` = 현재 스탑 가격 표시

## [5] 청산 조건

### 손절 (SL)
```vb
regimeSL = DF25  ' ATR 동적 SL (틱)
If ZZ <= -regimeSL Then  ' LONG: 현재가 - 진입가 ≤ -SL
    → SellFX (청산)
    → flag304 = 0
```

### 트레일링 스탑
- `BS25` 값(CalcAdaptiveTrailingStop 결과)에 의해 관리
- 15틱 이상 수익 시 활성화
- 매 틱 업데이트, 가격이 트레일링 레벨 이하로 떨어지면 청산

### BB 복귀 (TP)
- 종가가 BB 반대편 도달 시 청산
