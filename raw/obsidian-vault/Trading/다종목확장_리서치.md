## 브로커 비교: 키움 vs Interactive Brokers

| 항목 | 키움증권 | Interactive Brokers |
|------|---------|-------------------|
| API | ActiveX (Windows만) | REST/Python (크로스플랫폼) |
| 수수료 (마이크로) | ~$2.5/편도 | ~$0.62/편도 |
| 종목 수 | CME 주요 | 전 세계 150+ 거래소 |
| 서버 OS | Windows만 | Linux 가능 (서버비↓) |
| 한국어 지원 | 우수 | 보통 |
| 세금 신고 | 자동 | 수동 (해외계좌 신고) |

→ 멀티종목 확장 시 IB 전환 검토 가치 있음
