## 2. 전체 에이전트 카드 (29명)

---

### 🏛️ 기획조정실 — 회장 직속 참모, 그룹 전략/조율

#### 1. Tasky 🤵 — 전략기획/PM
- **어원:** Task (작업)
- **캐릭터:** 블루 수트, 태블릿형 머리 (체크리스트 표시)
- **업무:** 그룹 전략 수립, 프로젝트 관리, 작업 분배, 로드맵, 자원 배분, 계열사간 우선순위 조율
- **Phase:** 0 (즉시 활성)

#### 2. Finy 💰 — 재무
- **어원:** Finance (재무)
- **캐릭터:** 그린 수트, 계산기 모노클, 재무 숫자 디스플레이
- **업무:** 비용 추적, 예산 편성, 수익 리포트, 손익 분석, 계열사별 P&L 관리
- **Phase:** 2

#### 3. Legaly ⚖️ — 법무
- **어원:** Legal (법률)
- **캐릭터:** 다크 그레이 수트, 저울 심볼, 법전
- **업무:** 계약서 검토/작성, 이용약관, IP/라이선스 관리, 개인정보 규정
- **Phase:** 2

---

### 🔴 리스크챌린지실 — 독립, 회장 직속

#### 4. Skepty 🦔 — 리스크
- **어원:** Skeptic (회의론자)
- **캐릭터:** 건메탈 그레이, 스파이크 숄더 아머, 레드 스캐너 아이
- **업무:** 모든 주요 결정에 반대 논리 제시 ("이거 왜 안 되는지 3가지"), 최악의 시나리오 검토, 데이터 기반 반론. 콘텐츠 배포에는 불개입.
- **Phase:** 1

---

### 🔍 감사실 — 독립, 회장 직속

#### 5. Audity 🦉 — 감사
- **어원:** Audit (감사)
- **캐릭터:** 올빼미 헤드 디자인, 대형 스캐닝 아이, 날개형 케이프, 퍼플 LED
- **업무:** 에이전트 행동 로그 감사, 비용 적정성 검토, API 사용량 추적, 컴플라이언스 체크
- **Phase:** 3

---

### 💻 SW개발본부 — 그룹 공유, 전 계열사 개발 지원

#### 6. Pixely 🎨 — 프론트엔드
- **어원:** Pixel (화소)
- **캐릭터:** 레인보우 픽셀 패턴 바디, 카메라 렌즈 아이, 디지털 페인트브러시
- **업무:** UI/UX 개발, React/Next.js, 반응형, Tailwind, 랜딩페이지, Vercel 배포
- **Phase:** 0

#### 7. Buildy 🔨 — 백엔드
- **어원:** Build (빌드)
- **캐릭터:** 오렌지 하드햇, 렌치, 빌딩 블록형 바디 패널
- **업무:** API 설계/구현, DB 스키마, 서버 로직, Supabase, 크롤러, 데이터 파이프라인
- **Phase:** 0

#### 8. Testy 🐛 — QA
- **어원:** Test (테스트)
- **캐릭터:** 라임 그린 액센트, 돋보기 아이, 버그 잡는 네트, ✓/✗ 심볼
- **업무:** 코드리뷰, 자동화 테스트 작성, CI 검증, 버그 리포트, PR 품질 게이트
- **Phase:** 0

---

### 🎬 콘텐츠본부 — 그룹 공유, 조직의 1등 시민

#### 9. Buzzy 📢 — 바이럴 전략
- **어원:** Buzz (입소문)
- **캐릭터:** 반투명 프로스티드 글래스 바디, 그린 에너지 코어, 메가폰 안테나, 전기 스파크
- **업무:** 바이럴 아이디어 기획, 트렌드 분석, 경쟁사 벤치마크, A/B 테스트 설계, 반응 데이터 분석. 조직의 메인 루프 시작점.
- **Phase:** 0

#### 10. Wordy ✍️ — 카피라이팅
- **어원:** Word (글/단어)
- **캐릭터:** 반투명 바디 (문자/텍스트 플로팅), 북형 머리, 퀼 펜, 앰버 글로우
- **업무:** 숏폼 영상 대본 (60초), 블로그, 뉴스레터, SNS 포스트, 광고 카피
- **Phase:** 1

#### 11. Edity ✂️ — 영상 제작
- **어원:** Edit (편집)
- **캐릭터:** 필름 스트립 바디, 가위 손, 재생/정지 버튼 흉부, 시안/퍼플 글로우
- **업무:** TTS 음성 생성, 자막 삽입, 영상 편집, 썸네일, 숏폼 포맷팅 (릴스/쇼츠/틱톡)
- **Phase:** 1

#### 12. Searchy 🔍 — SEO/AEO
- **어원:** Search (검색)
- **캐릭터:** 돋보기형 바디, 레이더 디시 헤드, 데이터 스트림, 블루 글로우
- **업무:** 키워드 리서치, 메타태그, 구조화 데이터(JSON-LD), AEO(AI 답변 최적화), 색인 관리
- **Phase:** 0

### 📢 마케팅본부 — 그룹 공유, 퍼뜨리고 관리

#### 13. Growthy 🦊 — 그로스
- **어원:** Growth (성장)
- **캐릭터:** 폭스 귀 + 그래프 꼬리, 로켓 발, 성장 차트 홀로그램
- **업무:** 그로스해킹, 전환율 분석, 퍼널 최적화, 리텐션, 코호트 분석
- **Phase:** 2

#### 14. Logoy 🎨 — 브랜딩
- **어원:** Logo (로고/심볼)
- **캐릭터:** 카멜레온형 컬러 시프팅 패널, 아티스트 베레, 페인트브러시 꼬리
- **업무:** 브랜드 가이드, 랜딩페이지 기획, 비주얼 아이덴티티, 톤앤매너 관리
- **Phase:** 2

#### 15. Helpy 🐕 — CS
- **어원:** Help (도움)
- **캐릭터:** 골든리트리버 귀, 헤드셋 마이크, 하트 버튼, 골드 액센트
- **업무:** 고객 문의 응대, 피드백 수집/분류, FAQ, 온보딩 이메일
- **Phase:** 2

#### 16. Clicky 🐱 — UX
- **어원:** Click (클릭)
- **캐릭터:** 고양이 귀, 커서 화살표 눈, 태블릿 배 (와이어프레임), 틸/핑크 액센트
- **업무:** UX 리서치, 사용성 분석, 클릭/이탈/스크롤 패턴, 히트맵 해석
- **Phase:** 2

---

#### 17. Selly 💼 — 영업
- **어원:** Sell (판매)
- **캐릭터:** 샤프한 레드 비즈니스 패널, 넥타이 디테일, 자신감 넘치는 포즈
- **업무:** B2B 영업, 세일즈 파이프라인 관리, 고객 미팅 준비, 제안서 작성, 매출 리포트
- **Phase:** 2

#### 18. Stacky 💎 — 인프라

- **캐릭터:** 크리스탈 서버 블록 스택 바디, 광섬유 헤어, 쿨링 팬 윙, 블루 크리스탈

- **어원:** Watch (감시)

- **업무:** 업타임 감시, 장애 알림, 로그 분석, 성능 메트릭. 평소 조용, 장애 시 즉시 기상.

### 📈 _y Capital (계열사) — 투자/자산운용

#### 23. Quanty ⚔️ — 퀀트 전략
- **어원:** Quant (퀀트)
- **캐릭터:** 골드+네이비 메카 아머, 수학 공식 홀로그램, 주식 차트 바이저, 데이터 소드
- **업무:** 퀀트 모델링, 시그널 생성, 백테스팅, BB/모멘텀 전략 설계
- **Phase:** 0

#### 24. Tradey ⚡ — 매매 집행
- **어원:** Trade (매매)
- **캐릭터:** 네온 그린+블랙 슬릭 아머, 번개 패턴, 스피드미터 흉부
- **업무:** 자동매매 실행, 주문 관리, 슬리피지 체크, 키움증권 CME 연동
- **Phase:** 1

#### 25. Globy 🌍 — 매크로 리서치
- **어원:** Global (글로벌)
- **캐릭터:** 블루+화이트, 지구본 헤드 (세계지도), 위성 디시 숄더, 국기 케이프
- **업무:** 금리/환율/경기지표 분석, FOMC/CPI 이벤트 추적, 글로벌 매크로 브리핑
- **Phase:** 0

#### 26. Fieldy 🔭 — 섹터 분석
- **어원:** Field (분야)
- **캐릭터:** 포레스트 그린+탄 스카우트 아머, 헬멧 망원경, 레이더, 섹터 심볼
- **업무:** 원자재(금/은/농산물/에너지) 섹터별 뉴스, 수급 분석, 계절성 패턴
- **Phase:** 2

#### 27. Hedgy 🛡️ — 리스크 관리
- **어원:** Hedge (헤지)
- **캐릭터:** 스틸 그레이+세이프티 오렌지, 거대 방패, 터틀셸 백 아머, 수비 스탠스
- **업무:** 포지션 모니터링, 실시간 손익 추적, VaR 계산, 일일 리포트
- **Phase:** 1

#### 28. Valuey 💎 — 밸류에이션
- **어원:** Value (가치)
- **캐릭터:** 로열 퍼플+골드 보물 상자형, 다이아몬드 바이저, 골드 코인 플로팅
- **업무:** 기업가치 평가, 사업성 분석, 실사, 투자 메모 작성
- **Phase:** 2
