# 로직변경 히스토리

## 권장 순서

### 1단계: 현재 데이터로 다른 전략 빠르게 테스트

- 추세 추종 전략 (MA 크로스, 브레이크아웃)
- RSI 과매수/과매도
- VWAP 기반 전략
- **목표**: 수익 가능성 있는 전략 유형 찾기

### 2단계: 유망한 전략 심화 분석

- WF 검증 통과하는 파라미터 탐색
- 시장 상태별 성과 분석 (상승장/하락장/횡보장)

### 3단계: 복합/하이브리드 전략

- 여러 전략 조합
- 시장 상태에 따른 전략 전환 로직

### 4단계: 실전 적용 검토

- 페이퍼 트레이딩
- 리스크 관리 규칙 설정

---

**1단계부터 시작할까요?**

현재 코드에 추세 추종이나 RSI 전략이 구현되어 있는지 확인하고, 없으면 빠르게 추가해서 테스트해볼 수 있습니다.

## 결과 요약

| 분석 | 결과 |
| --- | --- |
| **전체 기간 백테스트** | +2,443pt (Sharpe 1.14) ✅ |
| **연도별** | 2022년만 손실, 나머지 수익 |
| **최근 6개월 (단순)** | +1,081pt, 승률 69% ✅ |
| **WF 홀드아웃 검증** | -402pt, Sharpe -1.50 ❌ |

**문제점:**

- 단순 백테스트와 WF 검증 결과가 크게 다름
- WF는 **폴드별로 파라미터를 다시 최적화**하기 때문에 결과가 달라짐
- 전략 자체보다 **시장 환경(레짐)**에 따른 변동이 큼

**현실:**
BB 이탈 전략은 **특정 시장 환경에서만 작동**합니다:

- 횡보장/약한 추세: 좋음
- 강한 추세장: 손실

**대안:**

1. **시장 레짐 필터 추가** - ADX, 변동성으로 시장 상태 판단 후 진입
2. **다른 전략 유형 탐색** - 추세 추종, 모멘텀
3. **수동 매매 보조** - 자동매매 대신 신호만 참고
