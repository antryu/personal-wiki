# PF & Sharpe Ratio

## PF (Profit Factor)

**수익 / 손실 비율**

`PF = 총 수익 거래 합계 / 총 손실 거래 합계 (절대값)`

| PF | 의미 |
| --- | --- |
| < 1.0 | 손실 > 수익 (손해) |
| 1.0 | 손익분기 |
| 1.5+ | 양호 |
| 2.0+ | 우수 |

예: 수익 1000pt, 손실 500pt → PF = 2.0

---

## Sharpe Ratio (샤프 비율)

**위험 대비 수익률**

`Sharpe = 평균 수익률 / 수익률 표준편차`

| Sharpe | 의미 |
| --- | --- |
| < 0 | 손실 (나쁨) |
| 0~0.5 | 저조 |
| 0.5~1.0 | 보통 |
| 1.0~2.0 | 양호 |
| 2.0+ | 우수 |

**의미**: 변동성 대비 얼마나 안정적으로 수익을 내는가

- Sharpe 1.0 = 1% 변동성당 1% 수익
- 높을수록 안정적인 수익
