### 🔧 수동 명령어

**즉시 백업 다운로드:**

```bash
/Users/andrew/Thairon/govrfp-alerts/download_backups.sh

```

**백업 상태 확인:**

```bash
ls -lh /Users/andrew/Thairon/govrfp-alerts-backup/

```

**자동화 작업 확인:**

```bash
launchctl list | grep govrfp

```

**자동화 중지:**

```bash
launchctl unload /Users/andrew/Library/LaunchAgents/com.govrfp.backup.plist

```

**자동화 재시작:**

```bash
launchctl load /Users/andrew/Library/LaunchAgents/com.govrfp.backup.plist

```
