# [HWP]

### 현재 방식의 한계

| 방식 | HWP 읽기 | HWP 쓰기 | 누름틀 지원 |
| --- | --- | --- | --- |
| pyhwp (Python) | ✅ | ❌ | ❌ |
| pypandoc-hwpx | ❌ | ✅ (HWPX만) | ❌ |
| hwplib (Java) | ✅ | ✅ | ✅ |

### 3. hwplib 장점

1. **누름틀(FormField) 완벽 지원** - 정부 양식에서 핵심
2. **HWP 바이너리 직접 수정** - 원본 양식 유지
3. **표 구조 접근** - 셀 단위 데이터 삽입 가능

### 아키텍처 제안

`┌─────────────────┐     REST API      ┌─────────────────┐
│  Python FastAPI │ ←───────────────→ │  Java Spring    │
│  (메인 서비스)   │    JSON 통신       │  (HWP 처리)     │
└─────────────────┘                   └─────────────────┘
        │                                     │
        └─── Claude AI                        └─── hwplib`

바로 구현해 드릴까요? 먼저 Java 프로젝트 구조를 만들겠습니다.

1. 워드(`.docx`), 마크다운(`.md`), HTML(`.html`,`.htm`) ⇒ 한글(`.hwpx`) 변환해주는 파이썬 도구   * pypandoc-hwpx · PyPI : https://share.google/NYF0hCuW4dzxocaE4

## HWP 처리 도구 3종 비교

- 정부 입찰 양식처럼 누름틀이 필수인 경우 **hwplib**가 유일한 선택지입니다.
- 단순 보고서 생성은 **pypandoc-hwpx**로 Python만으로 해결 가능하고요.

| **원본 양식 유지** | ❌ | ❌ | ✅ |

### 1. pyhwp (Python)

**GitHub**: https://github.com/mete0r/pyhwp

| 장점 | 단점 |
| --- | --- |
| Python 네이티브 - 기존 스택과 통합 용이 | **쓰기 불가** - 읽기 전용 |
| HWP 바이너리 파싱 가능 | 누름틀 지원 없음 |
| 텍스트/이미지 추출 | 개발 중단 상태 (업데이트 오래됨) |
| 설치 간단 (`pip install`) | 복잡한 양식 처리 한계 |

**적합한 용도**: HWP → 텍스트 추출, 문서 분석, 검색 인덱싱

---

### 2. pypandoc-hwpx (Python)

**GitHub**: https://github.com/msjang/pypandoc-hwpx

| 장점 | 단점 |
| --- | --- |
| **LLM 출력 → HWPX 직접 변환** | HWP 읽기 불가 |
| Markdown/HTML/DOCX 입력 지원 | 바이너리 HWP 미지원 (HWPX만) |
| Pandoc 기반 - 안정적인 변환 | 누름틀/양식 지원 없음 |
| Python 네이티브 | **새 문서 생성만 가능** (기존 양식 수정 불가) |
| 최신 개발 (2025.12) | 표 구조 세밀한 제어 어려움 |

**적합한 용도**: AI 분석 결과 → 새 HWPX 문서 생성, 보고서 자동 생성

---

### 3. hwplib (Java)

**GitHub**: https://github.com/neolord0/hwplib, https://github.com/neolord0/hwpxlib

| 장점 | 단점 |
| --- | --- |
| **읽기 + 쓰기 모두 지원** | Java 환경 필요 |
| **누름틀(FormField) 완벽 지원** | Python과 별도 서비스 필요 |
| 바이너리 HWP 직접 수정 | 학습 곡선 있음 |
| **원본 양식 100% 유지** | 배포/운영 복잡도 증가 |
| 표 셀 단위 데이터 삽입 |  |
| 정부 양식 처리 최적 |  |

**적합한 용도**: 정부 입찰 양식 자동 작성, 누름틀 기반 문서 자동화

추가 개발예정 기능

- 암호화된 HWP 파일에 대한 읽고 쓰기
- 이미지 파일, PDF 파일, HTML 파일로 변환
- 특정 페이지 내용 가져오기 나 전체 페이지 수를 구하기


### 권장 조합

```
Python FastAPI (메인)
├── pypandoc-hwpx  → 새 문서 생성 (단순 케이스)
├── pyhwp          → HWP 텍스트 추출/분석
└── REST API ────→ Java Spring + hwplib
                   → 누름틀 양식 처리 (복잡한 케이스)

```
