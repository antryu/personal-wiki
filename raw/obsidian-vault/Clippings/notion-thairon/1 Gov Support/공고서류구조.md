# 공고서류구조

## 조사 결과 요약

### 데이터 분석

- **G2B API는 구비서류 필드를 제공하지 않음** (나라장터 상세페이지에서만 확인 가능)
- bid_notices 테이블에 구비서류 관련 컬럼 없음
- 하위 유형 판별 가능 필드:
    - `srvce_div_nm` (용역 분류)
    - `dtil_prdct_clsfc_no_nm` (물품 분류)
    - `main_cnstty_nm` (공사 분류)

### 조달청 공식 자료 기반 구비서류 목록

### 공통 필수서류 (모든 입찰)

- 
    
    ```
    - 입찰서 (가격제안서)
    - 사업자등록증 사본
    - 법인등기부등본 (법인의 경우)
    - 인감증명서 또는 본인서명사실확인서
    - 입찰참가자격 등록증
    - 중소기업확인서 (해당 시)
    
    ```
    

### 용역입찰 (service) 추가서류

- 
    
    ```
    - 기술제안서 (협상에 의한 계약의 경우)
    - 과업수행계획서
    - 용역수행 실적증명서
    - 관련 면허/등록증 (기술용역, 엔지니어링 등)
    - 투입인력 자격증 사본
    - 4대보험 가입자 명부
    
    ```
    

### 물품입찰 (goods) 추가서류

- 
    
    ```
    - 물품 규격서 (규격입찰서)
    - 물품 카탈로그
    - 공급실적증명서
    - 공장등록증 (제조업체)
    - 품질인증서 (KC인증, ISO 등)
    - 시험성적서 (해당 시)
    - 독점공급확약서 (공급업체)
    
    ```
    

### 공사입찰 (construction) 추가서류

- 
    
    ```
    - 시공능력평가서
    - 건설업 등록증
    - 건설공사 실적증명서
    - 기술자 보유현황
    - 건설기술인 배치계획서
    - 경영상태 확인서 (재무제표)
    - 안전관리계획서
    - 하도급 관리계획서 (해당 시)
    ```
