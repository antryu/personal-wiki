# API (조달청)

**조달청 / 나라장터 관련 공공데이터개방표준서비스** 및 관련 API 안내

데이터 포맷: XML/JSON 

기준일: 2025-08-13

---

**1. 나라장터 공공데이터개방표준서비스**

- **서비스특징**: 입찰, 낙찰, 계약정보 데이터.
- **표준**: 행안부 고시 공공데이터 개방표준 적용(공통 개방 기준, 분야별 개방 항목 및 속성, 제공형식 등)
- **API유형**: REST
- **데이터포맷**: JSON, XML
- **활용신청**: 무료, 기본 10,000 트래픽(운영계정: 활용사례 등록시 증설 가능)
- **대표키워드**: 조달통계, 종합분석, 물품, 공사, 용역, 외자, 공공데이터표준

**활용 방법 및 참고 사항**

- 웹 기반 Swagger 명세서 제공(주요 브라우저 지원)
- 업무별 데이터 및 현황 조건별 상세 제공
- 개발계정 기본 트래픽(10,000), 활용사례 등록 시 운영계정 증설 가능
- 참조: 공식문서, 활용가이드 포함

**요약**

조달청 나라장터 OpenAPI는 **국가행정기관의 주요 조달/입찰/계약/통계 관련 업무데이터**를 XML/JSON으로 무료 개방하고 있습니다. 각 서비스별로 세부정보, 서류/금액/기관/시기 등 현황을 표준화해 제공하는 포털로, 국가중점 공공 데이터 개방의 대표적인 사례입니다.
