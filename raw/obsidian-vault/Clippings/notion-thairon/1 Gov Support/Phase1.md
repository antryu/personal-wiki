# Phase1

**Phase 1 (MVP - 3개월)**: 핵심 1-2개 기능으로 시작

- **스마트 공고 발굴**: 중소기업의 관심사 입력 → 맞춤 공고 자동 필터링 + 일일 이메일/앱 알림
- 기대효과: 정보 폭주 문제와 시간 낭비 해결, 사용자가 즉시 체감하는 가치

**1️⃣ 핵심 기능 정의 (2가지)**

**기능 A: 스마트 공고 필터링**

- 사용자 가입 시 프로필 설정: 업종, 지역, 용역액 범위, 관심사(예: "시스템 개발", "유지보수" 등)
- 나라장터 API에서 실시간 공고 수집 → 사용자 프로필과 매칭
- "당신에게 딱 맞는 공고 3개 발견됨" 같은 결과 제시

**기능 B: 일일 맞춤 알림**

- 매일 아침 이메일 or 앱 푸시: "어제 등록된 공고 중 당신에게 맞는 5개"
- 클릭 시 나라장터 공고 상세페이지로 이동
- 마감 24시간 전 리마인더 알림

**3️⃣ 사용자 여정 (User Flow)**

`가입 → 프로필 설정 → 매일 알림 수신 → 공고 클릭 
      (업종/지역/예산)   (이메일/앱)    → 나라장터 입찰`

**구체적 시나리오:**

- 월요일 아침, 건설 중소기업 A사 대표가 앱 열음
- "이번주 어제 등록된 공고: 5개 발견"
- 지역별, 가격대별로 정렬된 공고 리스트 확인
- 관심 공고 클릭 → 나라장터로 이동해 바로 입찰

**6️⃣ MVP 검증 전략**

**초기 사용자 모집 (100-200명):**

- 중소기업 커뮤니티, SNS, 나라장터 포럼에 공지
- "베타 테스터 모집" 형식으로 무료 제공

**측정 지표:**

- 일일 활성사용자(DAU)
- 알림 클릭률 (목표: 20% 이상)
- 사용자 만족도 (NPS 스코어)
- 실제 입찰 전환율

**피드백 수집:**

- 월 1회 사용자 인터뷰 (10명씩)
- "가장 불편한 점은?" → Phase 2 기능 우선순위 결정

**8️⃣ 성공 기준**

Phase 1 완료 후 다음 조건을 만족하면 Phase 2로 진입:

- ✅ 100명 이상 활성사용자 확보
- ✅ 월 30% 이상 사용자 증가
- ✅ NPS 스코어 40 이상
- ✅ "템플릿/자동화 기능 필요" 피드백 30% 이상
