# Hugging Face

- AI 모델, 데이터셋, 관련 도구를 공유/협업하는 온라인 플랫폼 및 커뮤니팅
    - 오픈소스전략: 강력한 커뮤니티 구축, 빠른 시장니즈 파악 등 AI 처럼 빠르게 변화하는 분야에서 개방형 협업 모델의 강력한 힘을 증명
- 구성
    - 허깅페이스 허브(Hugging Face Hub) - 중심 플랫폼
        - AI 빌더용 다양한 도구와 리소스 제공
        - AI 모델/데이터 검색에서 AI 어플리케이션 생성/공유 전과정에 필요한 것들 제공
    - 모델 허브(Model Hub)
        - 사전 훈련된 100만개 이상의 AI 모델이 있는 라이브러리
        - 앱스토어와 유사하게, 특정 작업을 위해 이미 훈련된 AI ‘두뇌‘를 찾아 바로 사용 가능
        - NLP, 컴퓨터비전(이미지처리), 오디오 및 멀티모달AI 모델까지 다양함.
        - 사용자는 원하는 작업(텍스트 분류, 이미지 생성 등), 언어, 사용 라이브러리(파이토치, 텐서플로우 등)등 다양한 기준으로 검색가능
        - 각 모델에는 ’모델 카드(Model Card)’라는 상세설명서가 있음
            - 모델 용도, 사용법, 한계점 및 버전관리(Git 기반), 다운로드 통계 등 제공
    - 데이터셋(Datasets)
        - 용량이 큰 데이터셋의 경우, 필요한 부분만 스트리밍 방식으로 처리 가능 (편리성 높음)
        - 설명서인 ’데이터셋 카드‘와 버전 관리 기능도 제공
    - “트랜스포머 라이브러리 - Transformer Library” = AI 개발 필수 아이템
        - 허깅페이스의 대표 라이브러리
        - 모델 허브(도서관)의 모델을 파이썬 몇줄로 쉽게 사용할 수 있도록 하는 도구로 AI어플리케이션 개발 속도를 획기적으로 높임
            - BERT, GPT-2, T5 같은 모델 등 손쉽게 사용가능
            - 파이토치, 텐서플로우, JAX 등 인기있는 머신러닝 프레임워크와 완벽히 호환
            - 용도: 모델로 예측(추론)하기, 내 모델의 추가 학습시키는 미세조정(Fine tuning), 모델을 처음부터 학습시키는 과정까지 모두 지원(사전학습모델 덕분에 흔치않음)
    - 스페이스 (Spaces)
        - 내 AI 모델, 라이브 데모를 만들어 공유/호스팅하는 서비스
        - 즉, AI 모델을 위한 인터랙티브 전시관 ⇒ 웹에서 직접 데모 실행하면 AI가 수행하는 일 확인
        - 파이썬 기반 웹 UI 라이브러리인 Gradio, Streamlit을 사용시, 코드 몇줄로 빠르게 인터랙티브 데모 생성 가능,정적인 HTML/Javascript 기반 데모도 호스팅 가능.
        - 허깅페이스는 기본적인 CPU, 메모리의 호스팅 자원을 무료로 제공 ⇒ 유료로 GPU 업그레이드 가능
    - 기타
        - 토크나이저 (Tokenizers): 컴퓨터는 글자를 바로 이해하지 못함. 글자를 숫자(토큰)로 변환하는 과정 이 필요. 해당 라이브러리가 변환단계를 위한 효율적인 도구 제공
        - 액셀러레이트 (Accelerate): 복잡한 코드수정없이, 여러개의 GPU,TPU 같은 다양한 하드웨어 환경에서 머신러닝 훈련코드를 쉽게 실행할 수 있도록 지원 (대규모 모델 훈련의 기술장벽 완화 효과)
        - 추론 API / 엔드포인트 (Inference API / Endpoints): 개발위한 서버없이 API 호출만으로 허깅페이스에 호스팅된 모델을 실시간으로 사용할 수 있게 해주는 서비스 (서버리스옵션). 유료이며 실제 서비스 운용에 필요한 전용 인프라 / 안정적인 확장성 제공
            - 오토트레인 (Autotrain): 모델을 자동으로 훈련/평가/배포 가능한 노코드/로우코드 서비스 (코딩부담완화)
