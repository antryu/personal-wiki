# 마음챙김 (Mindful) — 과학 기반 호흡 수련 플랫폼

> 매일 1분, 호흡으로 삶을 바꾸는 경험

## 비전

마음챙김은 **동양의 수련 전통**과 **현대 신경과학**을 결합하여, 누구나 쉽게 시작할 수 있는 호흡 명상 플랫폼입니다. 복잡한 명상 앱이 아닌, 호흡이라는 가장 본질적인 행위에 집중하여 사용자의 일상에 고요함을 가져옵니다.

**핵심 철학**: 시작은 30초, 목표는 평생의 습관.

## 시장 기회

### 글로벌 명상 앱 시장
- **시장 규모**: 글로벌 명상 앱 시장 $6.4B (2025) → $21.2B (2030)
- **연평균 성장률**: 27.1% CAGR
- **한국 시장**: 웰니스 시장 3.2조원, 디지털 명상 침투율 아직 5% 미만 → 거대한 블루오션

### 타겟 사용자
| 세그먼트 | 규모 | 니즈 |
|---------|------|------|
| 직장인 (25-45세) | 1,200만명 | 스트레스 관리, 집중력 |
| 수면 문제 경험자 | 성인 30%+ | 수면 전 루틴 |
| 명상 입문자 | 시장의 70% | 쉬운 시작점 |

## 차별점

### 1. 호흡 중심 UX
기존 명상 앱과 달리, **호흡**이라는 단일 행위에 올인합니다. 8가지 과학 기반 호흡 패턴(478호흡, 박스호흡, 수면호흡, 에너지호흡, 집중호흡 등)을 제공합니다.

### 2. 한국 시장 최적화
- 한국어 네이티브 UX (번역이 아닌 처음부터 한국어 설계)
- 한국적 미학 (서체: Noto Serif KR, 절제된 동양적 디자인)
- 시간대별 테마 (새벽/아침/오후/저녁/밤 — 자연스러운 수련 리듬)

### 3. 게이미피케이션 & 리텐션
- **레벨 시스템**: 첫 발걸음 → 마음의 스승 (10단계)
- **배지 수집**: 연속 수련, 누적 시간 기반 성취
- **스트릭**: 연속 수련 동기부여
- **여정 내러티브**: 수련이 쌓일수록 나만의 이야기가 펼쳐짐

### 4. 소셜 수련
- "지금 N명이 함께 수련 중" — 혼자가 아닌 느낌
- 그룹 수련, 파트너 수련 기능 (함께 탭)

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프론트엔드 | Next.js 16, React 19, TypeScript, Tailwind CSS v4 |
| 애니메이션 | Framer Motion 12 (호흡 원, 화면 전환, 축하 효과) |
| 오디오 | Web Audio API (싱잉볼, 호흡 톤, 종소리 합성) |
| 상태 관리 | React Context + useReducer |
| 배포 | Vercel (서버리스) |

## 데모 실행

```bash
npm install
npm run dev
```

http://localhost:3000 에서 확인. 데스크톱에서는 iPhone 프레임, 모바일에서는 전체 화면으로 표시됩니다.

**데모 팁**: 베젤 영역을 세 번 탭하면 시간대 테마 변경 및 여정 모드 전환이 가능한 숨겨진 컨트롤이 열립니다.

## 프로젝트 구조

```
app/                    # Next.js App Router
components/
  screens/              # 9개 화면 (온보딩, 호흡, 완료, 홈, 여정, 함께, 프로필, 통계, 가격)
  breathing/            # 호흡 원 애니메이션 (3모드: idle/ready/active)
  effects/              # 시각 효과 (빛의 파동)
contexts/               # 앱 상태 (화면 라우팅, 세션, 스트릭, 배지)
hooks/                  # 호흡 타이머, 오디오, 시간 테마, 카운터
lib/                    # 호흡 패턴, 상수, 테마, 목업 데이터
```
