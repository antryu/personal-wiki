**SIGReg = Sketch Isotropic Gaussian Regularizer**

JEPA의 핵심 문제를 해결하는 정규화 기법이야.

**문제: Representation Collapse**
JEPA는 "다음 상태의 임베딩을 예측"하는 구조인데, 모델이 꼼수를 부릴 수 있어:
- 모든 입력을 **같은 벡터**로 인코딩하면 → 예측 loss가 0이 됨
- "뭘 넣든 [0.5, 0.5, 0.5, ...]" → 완벽한 예측이지만 아무것도 안 배운 것

이전 방법들은 이걸 막으려고 EMA(exponential moving average), contrastive loss, 6개짜리 복잡한 loss 조합을 썼는데 — 불안정하고 튜닝이 지옥.

**SIGReg의 해결법: "latent space가 가우시안 분포를 따르게 강제"**

```
정상적 학습:  각 시장 상태 → 서로 다른 벡터 (분산↑)
Collapse:    모든 시장 상태 → 같은 벡터 (분산→0)
```

**M1Pro16Gbot** (PM 03:39)
SIGReg는 두 가지를 동시에 강제:
- **Variance**: 각 차원의 분산이 1에 가깝게 (축소 방지)
- **Covariance**: 차원 간 상관관계 최소화 (중복 방지)

수학적으로는 latent embedding의 분포가 **표준 정규분포 N(0,1)**에 가까운지를 Epps-Pulley 통계량으로 측정. 가우시안에서 벗어나면 penalty.

**우리 학습에서 일어나는 일:**
```
step 0:   sigreg 0.127 (분포가 가우시안에서 멀음)
step 749: sigreg 0.055 (가우시안에 수렴 중)
```

→ latent space가 잘 펼쳐지고 있다는 뜻. 시장의 서로 다른 상태가 **서로 다른 벡터**로 인코딩되고 있다.

**LeWM의 킬러 포인트가 바로 이거야.** Loss가 딱 2개:
1. `pred_loss`: 다음 상태 예측 정확도
2. `sigreg_loss`: 분포가 가우시안인지


## 🎓 대학생 레벨

LeWM의 loss:

$$\mathcal{L} = \mathcal{L}_{\text{pred}} + \lambda \cdot \text{SIGReg}(\mathbf{Z})$$

### L_pred: Latent Prediction Loss

Encoder `f_θ`와 Predictor `g_φ`가 있다.

**M1Pro16Gbot** (PM 03:46)
```
z_t = f_θ(o_t)           ← observation → latent
ẑ_{t+1} = g_φ(z_t, a_t)  ← state + action → predicted next state
z_{t+1} = f_θ(o_{t+1})   ← actual next state (label)

L_pred = ||ẑ_{t+1} - z_{t+1}||²
```

핵심: **pixel/feature space가 아닌 representation space에서 예측**. Encoder가 bottleneck 역할을 해서 task-irrelevant noise를 자연스럽게 filtering. Generative model(VAE, diffusion)이 모든 detail을 복원하려는 것과 근본적으로 다름.

문제: encoder와 predictor가 jointly 학습되므로 trivial solution 존재 — `f_θ`가 constant function으로 수렴하면 `L_pred = 0`.

### SIGReg: Sketch Isotropic Gaussian Regularizer

기존 해결책들:
- **BYOL/DINO:** target encoder에 EMA → 학습 불안정, 하이퍼파라미터 민감
- **VICReg:** variance + invariance + covariance 3개 loss → 6개 하이퍼파라미터

**M1Pro16Gbot** (PM 03:46)
- **Contrastive (SimCLR):** negative sampling 필요 → batch size 의존

SIGReg의 접근: **모든 moment를 한 번에 정규화**

정규분포의 특성: 모든 moment가 결정됨 (mean=0, var=1, skewness=0, kurtosis=3, ...).
→ latent 분포를 N(0,I)에 맞추면 variance, covariance, 그리고 higher-order 통계량까지 자동으로 건전해짐.

**측정 방법: Epps-Pulley characteristic function test**

```python
# Z: (T, B, D) — time × batch × latent_dim
# Random projections으로 1D marginal 검사
A = randn(D, K) / ||columns||  # K=1024 unit vectors

for each projection a_k:
    x = Z @ a_k  # 1D projection of latent dist
```

**M1Pro16Gbot** (PM 03:46)
```python
    
    # Characteristic function of x vs N(0,1)
    # φ_X(t) = E[exp(itX)],  φ_N(t) = exp(-t²/2)
    
    # Real part: E[cos(tx)] vs exp(-t²/2)
    # Imaginary: E[sin(tx)] vs 0
    
    err = (E[cos(tx)] - exp(-t²/2))² + (E[sin(tx)])²
    
    # Numerical integration over t ∈ [0, 3]
    statistic += ∫ err(t) · exp(-t²/2) dt
```

가우시안 가중 적분으로 t=0 근방(low-frequency = variance/covariance)에 더 큰 가중치, t→∞(high-frequency = tail behavior)에 자연 감소.

**왜 이게 VICReg보다 나은가:**

**M1Pro16Gbot** (PM 03:46)
VICReg은 2차 통계량(var, cov)만 명시적으로 제어. SIGReg는 characteristic function 전체를 matching하므로 **모든 차수의 통계량**을 암묵적으로 정규화. 그러면서 하이퍼파라미터는 λ 단 하나.

📢 Grounding Video Plans with World Models (GVP-WM)

Video models show emerging capabilities as visual planners, yet I2V-generated plans often violate physics.

GVP-WM projects video plans onto feasible trajectories in world models via latent-space trajectory optimization.

By jointly optimizing latent states and actions while preserving semantic alignment with the video plan, GVP-WM recovers feasible plans from videos that violate real-world dynamics (e.g., object teleportation, motion blur).

**이유 1: 대칭성**

- 110원 → 100원 (9.1% 하락) = -0.091 ← 비대칭!

- log(110/100) = +0.0953

## 목적함수 (Loss Function)

**딱 2개 항만 있어.** 이게 LeWM의 핵심 — "두 줄이면 충분하다"

```
L_total = L_pred + λ × L_sigreg
```

---

### 1. L_pred (Prediction Loss) — "예측을 잘 하라"

**M1Pro16Gbot** (AM 08:56)
```
L_pred = MSE(predicted_latent, actual_latent)
```

- predictor가 예측한 다음 상태 vs encoder가 실제로 인코딩한 다음 상태
- 둘의 차이를 줄여라 (Mean Squared Error)
- **latent space에서** 비교하는 게 핵심 — raw 데이터가 아니라 압축된 표현끼리

비유: "내일 시장 상태를 맞춰봐" → 맞출수록 loss 낮아짐

---

### 2. L_sigreg (SIGReg Loss) — "표현이 무너지지 마라"

이게 없으면 모델이 **collapse** 해 — 모든 입력을 같은 latent으로 매핑해버림 (loss가 0이 되니까!)

SIGReg = **두 가지 제약:**

**M1Pro16Gbot** (AM 08:56)
**a) Variance Loss** — "각 차원이 살아있어라"
```
L_var = Σ max(0, 1 - var(z_i))
```
192개 latent 차원 각각의 분산이 1 이상이 되도록. 분산이 0이면 = 그 차원 죽은 거.

**b) Covariance Loss** — "차원끼리 다른 정보를 담아라"
```
L_cov = Σ cov(z_i, z_j)² (i ≠ j)
```
차원 간 상관관계를 0으로. 차원 1이 "변동성"이면 차원 2는 "트렌드" 같은 다른 걸 담아야 해.

---

### 전체 그림

**M1Pro16Gbot** (AM 08:56)
```
캔들 데이터 → encoder → latent_t (192dim)
                          ↓
              latent_t + action_t → predictor → pred_latent_t+1
                                                    ↕
                          encoder(다음 캔들) → actual_latent_t+1

L_pred = MSE(pred, actual)     ← "예측 맞춰라"
L_sigreg = var + cov            ← "표현 무너지지 마라"
L_total = L_pred + 0.5 × L_sigreg
```

λ = 0.5 (config에서 설정). sigreg가 너무 강하면 예측력 떨어지고, 너무 약하면 collapse.

**이게 다야.** Stop-gradient도 없고, EMA도 없고, 복잡한 trick 없이 이 두 항만으로 안정적 학습.
