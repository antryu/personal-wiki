```
|     | 타임랩스            | 하이퍼랩스               |
| --- | --------------- | ------------------- |
| 카메라 | 고정              | 이동하면서 촬영            |
| 느낌  | 한 자리에서 시간만 빨리감기 | 도시를 날아다니며 시간 빨리감기   |
| 예시  | 빌딩에서 해 뜨는 거     | 거리를 걸으면서 24시간이 10초로 |
```

**한 줄:** Linear 이슈 보드를 감시 → 이슈마다 격리된 Codex 에이전트를 자동 배정 → PR까지 자동 완료. **팀은 에이전트를 감독하는 대신 "일"을 관리한다.**

```
        │

        ├── 이슈 B → Workspace B → Codex 에이전트
        └── 이슈 C → Workspace C → Codex 에이전트
                                        │
```

```
    → 군집 행동에서 패턴 추출
```
