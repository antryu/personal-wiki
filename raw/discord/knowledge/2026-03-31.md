MyBidWise 적용 시나리오

**Case 1: 입찰 공고 법적 리스크 체크**

```
# 공고에 "개인정보" 키워드 감지
grep -r "개인정보" legalize-kr/kr/ | head -5

# 결과:
# kr/개인정보보호법/법률.md:제38조...
# kr/정보통신망법/법률.md:제49조...
```

**M1Pro16Gbot** (AM 08:48)
**Case 2: 법령 개정 이력 추적**

```
# 개인정보보호법 최근 개정
git log --oneline -- kr/개인정보보호법/법률.md

# 특정 조항 변경 추적
git log -p -- kr/개인정보보호법/법률.md | grep "제38조"
```
