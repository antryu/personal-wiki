6가지 Agent Team Patterns

1. **Pipeline** — Sequential dependent tasks
2. **Fan-out/Fan-in** — Parallel independent tasks
3. **Expert Pool** — Context-dependent selective invocation
4. **Producer-Reviewer** — Generation + QA review
5. **Supervisor** — Central agent with dynamic task distribution
6. **Hierarchical Delegation** — Top-down recursive delegation
