**MCP (Model Context Protocol)**
- 회사가 MCP 서버 노출 → _y 에이전트가 회사 데이터 직접 쿼리
- `connect-company` = MCP client가 회사 MCP 서버에 연결
- 장점: Anthropic 표준, 도구/리소스 프로토콜, 생태계 빠르게 성장 중
- 적합도: ⭐⭐⭐⭐ — 가장 현실적

**A2A (Agent-to-Agent)**
- 회사의 AI 에이전트 ↔ _y 에이전트 간 통신
- "우리 회사 Searchy가 너네 회사 데이터 에이전트한테 물어봄"
- 장점: 에이전트 간 대화, Google 주도
- 적합도: ⭐⭐ — 아직 초기, 스펙 불안정

**OpenClaw Sub-agent**

- 장점: 이미 쓰고 있는 인프라
