PF (Profit Factor) 설명

정의

```
PF = 총 이익 / 총 손실
```
**계산 예시:**

• 승리 10번 → 총 +$1,000
• 손실 5번 → 총 -$500
• **PF = 1,000 / 500 = 2.0**

───

해석

**M1Pro16Gbot** (AM 09:08)
```
| PF 값      | 의미             | 평가       |
| --------- | -------------- | -------- |
| < 1.0     | 총 손실 > 총 이익    | 💀 망함    |
| = 1.0     | 총 손실 = 총 이익    | ⚠️ 손익 분기 |
| 1.0 ~ 1.5 | 이익이 손실의 1~1.5배 | 😐 약함    |
| 1.5 ~ 2.0 | 이익이 손실의 1.5~2배 | 🟡 괜찮음   |
| 2.0 ~ 3.0 | 이익이 손실의 2~3배   | 🟢 좋음    |
| > 3.0     | 이익이 손실의 3배 이상  | 🚀 매우 좋음 |
```
───

최저값 (이론적)

• **최저값: 0.0** (모든 거래가 손실)
• **실질적 최저: 매우 작은 양수** (거의 모든 거래가 손실이지만 1-2번 작은 승리)

승률 vs PF

**중요:** 승률 ≠ PF

```
| 전략 | 승률  | 평균 승리 | 평균 손실 | PF      |
| --- | --- | ----- | ----- | ------- |
| A  | 90% | $10   | $-100 | 0.9 💀  |
| B  | 40% | $100  | $-40  | 1.67 🟡 |
```
**전략 A:**

**M1Pro16Gbot** (AM 09:08)
• 10번 거래 → 9승 1패
• 총 이익: 9 × $10 = $90
• 총 손실: 1 × $-100 = $-100
• PF = 90 / 100 = **0.9** (손실!)

**전략 B:**

• 10번 거래 → 4승 6패
• 총 이익: 4 × $100 = $400
• 총 손실: 6 × $-40 = $-240
• PF = 400 / 240 = **1.67** (수익!)

**결론:** PF가 승률보다 더 중요!

Sharpe Ratio (샤프 비율) 설명

정의

```
Sharpe Ratio = (수익률 - 무위험 수익률) / 변동성
```
**한 줄 요약:** 리스크 대비 수익이 얼마나 좋은가?

───

쉬운 비유

**같은 수익 10%:**

• 전략 A: 매일 ±1% 변동 (안정적)
• 전략 B: 매일 ±10% 변동 (불안정)

**M1Pro16Gbot** (AM 09:31)
→ **A의 Sharpe가 더 높음** (같은 수익인데 덜 흔들림)

───

계산 예시

전략 1: 안정적

• 연 수익: 20%
• 변동성: 10%
• **Sharpe = 20 / 10 = 2.0** 🟢

전략 2: 불안정

• 연 수익: 20%
• 변동성: 40%
• **Sharpe = 20 / 40 = 0.5** 🔴

**M1Pro16Gbot** (AM 09:31)
**같은 수익인데 전략 1이 4배 더 좋음!**
